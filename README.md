# MissingEggDoc

公开的游戏设计文档库：**抖音小火人世界 · 3D 轻量侦探游戏** 整体方案，Markdown 分区文档。

> **一句话项目定位**：基于抖音小火人原生系统的 3D Lowpoly 探案小世界——玩家扮演小火人侦探，在微缩农场中破解动物们的「跨服对话」，寻找母鸡淑芬丢失的、画有爱心标记的宝贝蛋；全流程真相与玩家认知存在刻意错位，终局以人类温柔介入完成情绪反转。
