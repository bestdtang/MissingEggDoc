# 11 美术方向

## 11.1 视觉策略（与 `08` 对齐）

采用 **「3D 场景雕塑 + 2D 对话立绘 + 漫画分镜」** 三层表现策略。核心原则：**3D 负责空间与角色存在感，2D 负责情绪表演，漫画负责主线收束。**

- **3D Lowpoly 场景**：构建高度集约、色彩明快的微缩农场，强调「盒子盆景」精致感；承载一周目调查阶段与二周目释怀盘。
- **3D NPC 静态雕塑**：场景中的 NPC 不承担复杂动画表演，只提供不同剧情阶段 / 状态下的**静态 pose**（如大黄瘫倒 / 清醒站起、黑猫树上 / 树下、淑芬焦急守窝）。可做轻微呼吸、晃动、挤压回弹等低成本形变，让角色"活着"，但不做完整走路、表演或口型动画。
- **2D 对话立绘**：NPC 的主要情绪表达放在对话中完成，用动态立绘 / 表情差分表现夸张反应、吐槽脸、心虚、傲慢、释怀等表演；3D pose 只负责让玩家在场景中认出"是谁、处于什么状态"。
- **漫画分镜（主线收束专用）**：**主线终点的卧室揭晓 + 小鸡回巢段不在 3D 场景中实时呈现**，改由手帐展开为漫画载体的**分镜插画**承担（详见 `07` **步骤 二-9 · 手帐漫画**、`09` §9.3）。分镜风格与 2D 立绘的"夸张漫画感"保持质感连续，但**情绪基调更克制温柔**，承接温度落点；主人在分镜中仅以**剪影 / 局部动作**出场，不正面亮相（与 `characters/主人.md` "缺席的温柔" 定位一致）。

## 11.2 NPC 形象设计概念（概念美术首轮）

> 当前初稿问题：可爱度够，但偏"动物贴纸 / 吉祥物"；拟人表演不足。我们的目标是"能演戏的动物角色"。

- **淑芬（劳模母鸡）**：形体要丰满、有重量，但站姿必须硬。胸口前挺、脖子高抬、脚爪抓地，像农场大姐大。焦急时羽毛炸开、眼眶发红；温柔时轮廓变圆，翅膀像围裙包住身体。不要只画成哭哭母鸡，要有"嘴硬、能扛事、护崽"的气场。
- **小鸡侦探团（4 只半大幼鸡 / 熊孩子群像）**：重点是"一组中二小孩"，不是四只普通小鸡。领头更高、更挺、墨镜最大；跟班用墨镜破损、落叶帽、站姿高低区分。群体站位要鬼祟：排队、围圈、抱团、互相推锅。纸板墨镜和落叶帽要有自制破烂感。参考：《小黄人》的群体差异。
- **悲伤蛙**：采用**细长垂头版本**。身体像被水泡软，头低、肩塌、长手长腿松垮下垂，坐在绿油油薄荷鱼"垫子"上像一条折弯湿布。眼睛大但无光，嘴角下压，喉囊长叹时轻微鼓起。要可怜、荒诞、丧。参考：《马男波杰克》丧气姿态。
- **老鼠兄弟（这个不用3D）**：必须一眼分出兄弟分工。鼠哥宽、稳、压低帽檐，抱算盘 / 账本，是墙缝黑市老板；鼠弟尖、快、身体前倾、嘴巴大，是随时说漏嘴的捧哏。构图固定为两只从墙缝探出半身，爪子扒边，像洞口双人相声。不要只画成两只可爱胖鼠。
- **大黄（老实人）**：画成一只大体型黄狗，身体像厚重的长方形软垫，四肢短粗，脸宽、鼻头大、眉毛下垂。开局不是坐着，是整只狗横着瘫在谷仓墙根：半个肚子压住侧倒木梯，后腿摊开，舌头歪出，头顶湿树叶。眼神要像做错事的大狗，委屈、宿醉、完全不会撒谎。清醒后仍是同一只笨重大狗，但胸口挺起来，项圈摆正。参考：《飞屋环游记》Dug 的忠诚感。
- **乌鸦**：形体瘦、尖、黑，像一个贪婪又自大的小老头。脖子前伸，肩背微驼，眼睛很贼，爪子永远抓着东西；站姿不是普通鸟站立，而是"占山为王"地往前压。可以有破烂皇冠、瓶盖项链、亮片披风一类低级王权感，但重点在本人：炫耀、护食、嘴硬、神经质。参考：经典卡通乌鸦贼相、夸张反派小丑感、破落国王的姿态。
- **黑猫（坏叔叔外壳 / 宅猫傲娇）**：不要做灵敏小猫，也不要圆萌。外表要有"坏叔叔"压迫感：瘦长但不轻盈，脸可以更长、更阴，眼神像《狮子王》Scar 那类坏狮子，嘴角带冷笑，坐姿像窝在家里发号施令的宅猫。它看起来刻薄、阴沉、难接近，实际内核是傲娇和被抢软垫后的自尊受伤。毛发可略乱，黑眼圈明显，像长期失眠的室内猫。参考：Scar 的坏叔叔气质、宅在沙发上的大猫、经典反派猫的懒散压迫感。
- **闪电蜗牛 Flash（慢语速喜剧角色）**：要更傻、更好笑，不只是安静小蜗牛。它很想跟你说话，眼神努力、身体努力前倾，但每个动作都慢半拍；说话前像憋了很久，眼柄慢慢转过来，观众已经等急了。喜剧点接近《疯狂动物城》树懒 Flash：热情、认真、慢到离谱；也可以有一点《冰河世纪》松鼠那种执着又笨拙的劲。不要画成普通宠物蜗牛，要像一个反应很慢但很有表达欲的小喜剧演员。

## 11.3 UI 风格

- 轻量、手绘感笔记本 UI 与 Lowpoly 场景形成材质对比；避免厚重拟物边框抢戏。
- **笔记本采用手帐拼贴风**（内容清单见 `09`）：横版 16:9 跨页；版面与摆放由美术定稿；视觉元素含**钢笔字 / 便签 / 插图 / 大头针 + 红线 / 透明胶带 / 涂改痕**。新条目以**自动书写 / 钉入 / 贴入**等演出写入；过程行笔记的**划掉与旁注**保留可见，作为推理过程的可视化质感。
- **主线两页 vs 老鼠情报第三页**：前者纸张整洁、字迹工整；后者纸张更脏旧、便签带老鼠咬痕、字迹更潦草（"老鼠口述、侦探转录"风格），与主线形成对照。
- **第二页"漫画演绎"形态**：玩家跳入天窗后，第二页**翻面 / 展开为漫画载体**——分镜插画陆续铺满手帐幅面，类似翻开一本剧情漫画；漫画完播后回到第二页常态显示，定格为已完成的拼贴页（详见 `07` **步骤 二-9 · 手帐漫画**）。

### 11.4.0 NPC 概念氛围图（首轮 · 每角色 1 pose）

> **目的**：锁定剪影、比例、性格气质，供 moodboard 与后续 2D 立绘 / 3D 雕塑共同参照。**本轮每 NPC 只画 1 个状态、1 个 pose**——选最能代表性格的那一张，不画表情差分、不画多剧情态。不含小火人、主人（本轮不做）。

- **淑芬 · 守窝大姐大**：
  - 鸡舍前站稳，体块丰满、胸口前挺、脖子高抬、脚爪抓地；
  - 羽毛微炸、眼眶发红，默默流泪，是"能扛事的母亲在硬撑"。
- **小鸡侦探团 · 中二拦路**：
  - 4 只首尾相连排成一列，领头最高、纸板墨镜最大、胸口挺；
  - 跟班 A/B/C 用破损墨镜、落叶帽、高低站姿区分；整体鬼祟装酷，像拦路开会的熊孩子侦探。
- **悲伤蛙 · 池塘 emo**：
  - 细长垂头版，坐在绿油油薄荷鱼"垫子"上，身体像折弯湿布；
  - 头低、肩塌、长手长腿松垮，眼睛空洞盯水面，嘴角下压，像随时长叹。
- **老鼠兄弟 · 墙缝兜售**：
  - 两只从红顶屋墙缝探出半身，爪子扒边；
  - 鼠哥宽稳、压低帽檐、抱算盘；
  - 鼠弟尖快、身体前倾、嘴巴大张。仅 2D，不做 3D。
- **大黄 · 宿醉失职保安**：
  - 谷仓墙根横瘫，体块像厚重软垫；
  - 半个肚子压住侧倒木梯，后腿摊开，头顶湿树叶；
  - 委屈宿醉眼，完全不会撒谎的大笨狗。
- **乌鸦 · 破落国王**：
  - 瘦尖黑鸟，肩背微驼、脖子前伸，站姿往前压像"占山为王"；
  - 贼眼发亮，爪子夸张抓紧瓶盖 / 亮片；可带破烂皇冠，重点是炫耀、护食、嘴硬的本人气质。
- **黑猫 · 宅猫坏叔叔**：
  - 瘦长、脸更长更阴，Scar 式冷笑；像窝在家里发号施令的宅猫——半坐或蜷坐，尾巴懒垂，黑眼圈明显，毛发略乱；
  - 外表刻薄阴沉，内核留一点"被冒犯后的傲娇"余地，不要圆萌小猫。
- **蜗牛 · 憋不住想说**：
  - 宽叶上极小一只；身体努力前倾、眼柄慢半拍转向玩家，表情认真又傻，像树懒 Flash 那样热情但慢到离谱——观众已经等急了它还没开口。

### 11.4.3 NPC 对话立绘（对话框上方 · 立绘即表情）

> **位置与规格**：对话框**上方**，约 **3/4 身上半身**（可略带头肩），不做全身。
>
> **制作口径**：**立绘 = 表情**。每张是完整上半身表演，**不做**底图换脸。每个 NPC：**常态轮播 ≥3 张** + **事件立绘**。
>
> **动作写法**：每张只定格 **一个 moment 的一个动作**（像分镜关键帧），用姿势承载情绪；**不写**「从 A 到 B」的连续变化。优先复用常态；事件只在常态撑不住时出现；时机 / 原因相近、姿势可共用的**合并为一张**。
>
> 形体对齐 11.4.0 概念图。小火人、主人本轮不产。
