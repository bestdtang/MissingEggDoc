## T-3（三天前）

### T-3 上午

平日基线：淑芬正常孵蛋（最后一颗画爱心蛋）、小鸡酝酿"带弟弟踢球"念头、大黄晨巡正常、黑猫窗台睡回笼觉、乌鸦屋顶晨巡寻反光、老鼠墙缝数钱、悲伤蛙池塘哀悼"昨晚月亮"、蜗牛 Flash 决定从叶根爬向叶尖（约十厘米史诗旅程）、主人院中干活已暗暗留意淑芬。

### T-3 下午（核心案发段）

| 角色 | 事件 |
|------|------|
| 小鸡 | 趁淑芬离窝，找天然白色圆石，用短木炭画爱心和鬼脸 → 塞鸡窝伪装、把**真蛋**偷出 |
| 淑芬 | 离窝短暂活动；回窝坐到"蛋"上顿感冰冷彻骨——孵化焦虑积压数日心弦绷至极限，冰冷刺激击穿理性，没细辨真假 → 院内**狂奔大叫深蹲**，拼命用体温救"孩子"；随后被主人引到水盆边缓气，**人在院中、窝中无人** |
| 大黄 | 见淑芬发疯不知所措，院中跟着焦虑打转 |
| 黑猫 | 趴主人窗台冷眼旁观淑芬失控，觉得幼稚可笑，朝院里瞥一眼"你们都好吵" |
| 蜗牛 Flash | 爬至叶 1/3 处，目睹**小鸡抱蛋从身旁鬼鬼祟祟跑过**朝院外 |
| 主人 | 院中干活目睹淑芬失控 → 先把淑芬引到水盆边缓气，避免她继续折腾 → 走近鸡舍判断"最后一颗蛋焦虑失常" → 回屋翻《母鸡产后护理与孵化指南》；**第一反应不是立即取蛋**——原则"先等等看她能否稳下来"，悄悄准备孵化设备（保温箱、黄灯、水桶）**备而不用** |
| 老鼠 | 偷听农场动静、对淑芬骚动好奇，暗中记录"反常事件" |
| 乌鸦 | 正午补觉 |
| 悲伤蛙 | 池边发呆"平静终将被打破" |

### T-3 傍晚

| 角色 | 事件 |
|------|------|
| 小鸡 | 推沾泥真蛋路过谷仓被老鼠吓唬 → 池塘洗泥 → 吵醒悲伤蛙被"深渊预言"吓魂飞魄散 → 趁主人回屋翻书、淑芬仍在水盆边缓气，从鸡舍侧口把真蛋推回窝 → 慌乱中把**石头假蛋扔进鸡舍外草丛** |
| 老鼠 | 见小鸡推沾泥蛋路过，**随口编**"沼泽水怪"恐吓（纯乐子，非蓄谋） |
| 悲伤蛙 | 盯水中月影 → 被小鸡打扰幽怨自语"玉盘终将沉入深渊……" → 小鸡误读为"蛋被吃" |
| 淑芬 | 院中狂奔深蹲精疲力竭回窝——此时真蛋已被推回；感到微弱心跳与温度，以为自己"把孩儿暖回来了"安心继续孵 |
| 蜗牛 Flash | 至叶中段，目睹小鸡推真蛋滚回鸡舍 + 画着爱心和鬼脸的白石头被扔进就在身旁的草丛 |
| 大黄 | 见淑芬恢复正常松口气继续巡逻，对草丛里的画爱心和鬼脸白石头一无所知 |
| 黑猫 | 出门巡视前在院里走一圈，注意到草丛边白石头但不感兴趣（不是闪亮的、只是普通石头） |
| 主人 | 室内翻资料、准备孵化设备 |

### T-3 深夜

淑芬安心孵蛋入睡；大黄夜巡正常；黑猫主人卧室窗台天鹅绒软垫睡觉心满意足；小鸡回鸡舍睡觉对今天的惊吓犹有余悸"弟弟回来了就好"；主人见淑芬已自行回稳，设备靠墙备好但按兵不动"她能自己来就别惊扰"；其余各自休眠。

| 大黄 | 院中见乌鸦掠走草丛里的画爱心和鬼脸白石头，狂吠扑咬却**跳不高只咬空气** → **认定**那便是淑芬的蛋被叼走 → **心理崩溃**、自认失职；**同日稍后**起借发酵苹果渣烂醉 |
| 黑猫 | 趴窗台旁观乌鸦俯冲 vs 大黄跳不高的闹剧；听乌鸦叫大黄「地毯」，冷冷对大黄重复一遍：**「地毯。」**，走开 |
| 蜗牛 Flash | 微转触角目睹**乌鸦俯冲叼石头 + 大黄追逐战**就在叶旁，乌鸦绕大黄头顶那一圈离叶子不超过一米 |
| 淑芬 | 正常孵蛋对外界骚动不太在意 |

### T-2 下午

| 角色 | 事件 |
|------|------|
| 大黄 | 无法接受失职现实，**偷吃地窖旁发酵苹果渣**把自己灌倒（宿醉第一天） |
| 乌鸦 | **正午前后**谷仓顶摆弄玻璃珠、碎玻璃等闪亮「财宝」，反射光扫到窗台下/谷仓边午睡的黑猫，**毛尖险燎**（可美术微焦痕） |
| 黑猫 | 谷仓附近午觉，薄荷鱼留在角落；被屋顶玻璃珠聚光 / 碎玻璃反光激怒，**一次**上谷仓顶找乌鸦理论；对峙中**顺带认清**巢里假蛋，仍**选择沉默**，下顶——**偷鱼窗口**；发现薄荷鱼失窃 → 断定鼠 → **红顶屋墙缝**外堵约半小时进不去、记仇 |
| 老鼠 | **趁黑猫上屋顶**偷走谷仓角落**【极品薄荷鱼玩具】**；逃跑路上被大黄酒疯狂吠惊到，慌乱中把薄荷鱼**掉在池塘边荷叶旁**，仓皇逃回墙缝；从墙缝抬头瞥见**黑猫上谷仓屋顶没多久又下来、爪上啥也没带**——三年宿仇的猫鸦居然没打起来，记进一档八卦池 **#10** |
| 悲伤蛙 | 发现身旁多了散发刺鼻气味的绿色物体（薄荷鱼），觉得"腐朽气息契合千疮百孔心境"——**拽来垫在胖臀下当浮木** |
| 主人 | 偶尔从窗口观察淑芬看起来稳定（不知小鸡已还真蛋），判断无需立即介入；继续完善孵化设备细节（温控调试、黄灯测试、铺垫材料） |

| 黑猫 | 下午又去墙缝外堵一会儿进不去，憋着火上床睡觉，**打算晚上去找主人撒娇消气**——昨晚悲剧的前因 |

| 主人 | **孵化设备准备完成**只差"必须今夜动手"的信号；持续观察淑芬：连续数日高度紧张孵蛋疲惫加剧，蛋已有临破壳前细微动静，心里预判"如果今夜气温再降就必须动手" |

| 悲伤蛙 | 池塘边继续发呆精神状态低落 |

### T-1 深夜（核心事件段）

> 整个故事的**真相发生时段**，所有 NPC 状态在此汇聚。

| 角色 | 事件 | 备注 |
|------|------|------|
| 大黄 | 深度宿醉；酒精作用发出**极沉闷带哨音巨大鼾声**（"呼噜噜——咕呱"）。声音传到鸡舍方向，也传到主人卧室窗外 | 催化剂 |
| 主人 | **被大黄酒嗝鼾声惊醒**借此确认时机 → 出门感受到**深夜寒意骤降**（倒春寒最冷一夜）→ **先到鸡舍探看**见淑芬**死守巢穴但精疲力竭** → 当机立断今夜必须介入 → 回卧室取水桶穿雨靴 → ①**第一趟（池塘）**舀一桶水（孵化箱加湿）雨靴踩泥来回 → 回卧室水桶搁桌边、温柔捧起黑猫、**抽走天鹅绒软垫**垫保温箱底 → ②**第二趟（鸡舍）**雨靴踩泥再来回、趁淑芬熟睡温柔抱走真蛋 → 回卧室真蛋放入保温箱、加湿、开**黄色保温灯**、人工孵化启动 | 寒夜骤降 + 淑芬疲态触发两天观望终点；两次外出对应老鼠"雨靴来回两趟" |
| 悲伤蛙 | 见主人提桶**舀走池水**水位肉眼可见下降 → "生命之源正在枯竭" | 第一重打击 |
| 黑猫 | 卧室窗台软垫入睡 → 主人温柔把黑猫捧起 → 猫以为按摩 → 结果被放冰冷木地板、**抽走软垫** → 史诗级感情诈骗 → 从猫洞挤出带满腔怒火路过池塘 → 见蛙盯重新凝聚的月影叹气怒火上头 → 伸爪**对水中月影狂拍水花四溅** → "闭嘴！那是虚无的光斑！" → 爬大橡树整夜自闭 | 第二重打击（蛙角度）；猫**完全没有"蛋"的直接目击**，也不知道桌上装置与黄灯用途。它只知道软垫被主人拿走；"软垫可能卷入失踪蛋去向"必须等第二章玩家事实清单首次激活。蛋具体位置、软垫去向、黄灯用途全部保留至红顶屋物证一次性揭晓 |
| 悲伤蛙 | 被黑猫拍碎月影 → **身体严丝合缝压住薄荷鱼**黑猫未闻到 → 蛙精神彻底崩溃陷入最深抑郁 | 薄荷鱼"盲区保全" |
| 淑芬 | 睡梦中蛋被主人取走**浑然不觉**；次日清晨才发现空窝 | 睡着时发生 |
| 小鸡 | 鸡舍内深夜被大黄恐怖鼾声惊到 → 集体蜷缩抱团"水怪来了！水怪就在外面低吼！" → 更加确信水怪理论，为明天成立侦探团埋动机 | 误导音效 |
| 蜗牛 Flash | 至叶约 2/3 处，目睹**主人从鸡舍方向温柔抱真蛋沿院中小路从身旁经过走向红顶屋**。院中小路正经过蜗牛所在叶子下方——这片农场最核心的一次目击 | 见证核心真相 |
| 老鼠 | 墙缝见红顶屋方向**亮起刺眼黄灯** → 听到**极重人类雨靴脚步**（主人两趟外出"咔咔咔——来回走了两趟"）→ 数小时后隐约听见极清脆**「咔嚓」碎裂声**（**蛋壳破裂声**孵化进展；与木材卡榫无关、与主人雨靴动作不在同一时段） | 局限视角：只见光、听声，不见内部 |
| 乌鸦 | 屋顶睡觉不知外面动静 | |
