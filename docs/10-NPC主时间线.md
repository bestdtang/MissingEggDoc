| 悲伤蛙 | 午后池边发呆，看着水面涟漪感叹「平静终将被打破」。 |

### T-1 深夜（核心事件段）

> 本段是整个故事的**真相发生时段**，所有 NPC 状态在此汇聚。

| 角色 | 状态 / 事件 | 备注 |
|------|-------------|------|
| 大黄 | 深度宿醉；睡梦中因酒精作用发出**极沉闷带哨音的巨大鼾声**（「呼噜噜——咕呱」类）。声音传到鸡舍方向，也传到主人卧室窗外。 | 此声是催化剂 |
| 主人 | **被大黄的酒嗝鼾声惊醒**，也借此确认了时机→出门→感受到**深夜寒意骤降**（倒春寒最冷一夜，比预报还低）→走到鸡舍，见淑芬**死守巢穴但精疲力竭**→当机立断：**今夜必须介入**（两天观望至此终止）→**去池塘舀走一桶水**（孵化箱加湿用）→进卧室，温柔捧起黑猫、**抽走天鹅绒软垫**垫保温箱底→去鸡舍，趁淑芬熟睡**温柔抱走真蛋**→开**黄色保温灯**，启动人工孵化。 | 寒夜骤降 + 淑芬疲态：两重信号触发两天的观望终点 |
| 悲伤蛙 | 见主人提桶**舀走池水**，水位肉眼可见下降→「生命之源正在枯竭。」绝望加深。 | 第一重打击 |
| 黑猫 | 夜间找主人撒娇→卧室窗台天鹅绒软垫入睡→主人回来抱蛋时，**温柔把黑猫捧进怀里**→猫以为深夜按摩，准备发出享受呼噜→结果：猫被放到**冰冷木地板**，主人**抽走软垫**垫保温箱，打开**刺眼黄灯**→猫怒目睥睨间，主人从床边小心取出一个**白色圆滚滚的东西，极轻柔地放在软垫上**——黑猫怒火熏心，根本没细看那是什么，脑子里只有「我的软垫被占了」→史诗级感情诈骗，自尊重创→**从猫洞挤出，带满腔怒火路过池塘**→见蛙仍盯重新凝聚的月影叹气，怒火上头：伸爪**对水中白圆球狂拍，水花四溅**→咬牙：「闭嘴！那是虚无的光斑！」→发泄完，**爬上大橡树，整夜自闭**。 | 第二重打击（蛙角度）；猫当时亲眼目击了取蛋安置全过程，只是怒火中烧未意识到那是蛋——此为第二章记忆触发的埋点 |
| 悲伤蛙 | 被黑猫拍碎月影→**身体严丝合缝压住薄荷鱼（绿垫）**，黑猫未闻到→蛙精神彻底崩溃，陷入最深抑郁。 | 薄荷鱼的「盲区保全」 |
| 淑芬 | 睡梦中蛋被主人取走，**浑然不觉**；次日清晨才会发现空窝。 | 睡着时发生 |
| 小鸡侦探团 | 鸡舍内，深夜被大黄的恐怖鼾声惊到→集体蜷缩抱团：「水怪来了！水怪就在外面低吼！」→更加确信水怪理论，为明天成立侦探团埋下动机。 | 误导音效 |
| 蜗牛 Flash | 爬至叶约 2/3 处，目睹**主人从鸡舍方向温柔抱着真蛋，沿院中小路从身旁经过，走向红顶屋**。院中小路正经过蜗牛所在叶子下方，是这片农场最核心的一次目击。 | 见证核心真相 |
| 老鼠兄弟 | 墙缝里，见红顶屋方向**亮起刺眼黄灯**→听到一串**极重的人类雨靴脚步声**（主人去池塘再折回）→后来隐约听见极清脆的**「咔嚓」碎裂声**（**蛋壳破裂声**，孵化进展，已定稿；非木材卡榫声）。 | 局限视角：只见光、听声，不见内部 |
| 乌鸦 | 屋顶睡觉，不知外面任何动静。 |  |

## 交叉关系热图（速查）

> 「→」表示单向影响；「↔」表示双向互动。
