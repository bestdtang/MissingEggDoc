# 15 玩家线索与交互点总表

> 本表只收录玩家在 3D 场景中可点击的环境线索、可拾取物、空间交互点与非线索收集点。NPC 对话 / 质询 / 交易不在本表展开；终局漫画物证属于结尾演绎，不列入本表。

---

## 15.1 统一交互规则

- **触发方式**：玩家走近交互点，出现交互键，点击后触发交互。
- **观察 / 拾取类线索**：点击后出现线索大图 + 描述框，由文本描述现场情况。
- **空间交互点**：第一次点击可给出线索大图 + 描述框或空间反馈；满足道具条件后，第二次点击执行放置 / 通路动作。
- **NPC 对话拆分**：NPC 对话 / 质询 / 交易不再混入本表；由对话自然给出的信息，不作为额外可捡拾线索点列入。
- **信息收录反馈**：交互结束后，笔记本 UI 按键出现红点，表示有新信息收录；玩家打开笔记本查看。笔记本具体条目与页面演出后续另行细化。

## 15.2 字段说明

| 字段 | 说明 |
|------|------|
| 阶段 | 所属章节或结尾阶段 |
| 位置 / 交互点 | 玩家在 3D 场景中看到并点击的环境点、道具点或空间点 |
| 交互反馈 | 点击后的表现形式：线索大图 + 描述框、拾取、空间动作等 |
| 关联角色 | 该交互点对应的 NPC / 真相链 |
| 解锁 / 卡点用途 | 解锁后续对话、空间、道具、章节或终局演出 |
