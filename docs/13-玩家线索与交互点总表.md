## 13.1 统一交互规则

- **触发**：走近 → 交互键 → 点击。
- **观察 / 拾取类**：线索大图 + 描述框。
- **空间交互点**：首次点击给反馈；满足条件后再次点击执行放置 / 通路 / 跳跃。
- **NPC 信息**：由 `07` 对应步骤承载，不另列环境点。
