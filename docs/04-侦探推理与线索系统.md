# 04 侦探推理与线索系统

## 4.1 推理目标与胜利条件

- **表目标**：满足淑芬委托，找回「画爱心的宝贝蛋」的下落并给玩家可理解的结案叙述。
- **里目标**：拆穿多层动物误会链，**跳入红顶屋天窗触发漫画演绎**——在漫画分镜中完成 **物证总览闭环**（保温箱、软垫、黄灯、池塘水等要素一一对应）+ **健康小鸡回巢、淑芬母子团聚** 的情感闭环（见 `05`/`07-7.2` 步骤 14–15、`09-9.2` 第二页页面收束段）。
- **二周目（释怀盘）目标**：补完老鼠未购入的八卦盲盒、收齐每个可交互 NPC 的一次性结尾对话、完成奶酪碎全收集；二周目无硬卡关，全收集为可选满足项（详见 `07-7.2` 步骤 16 与 `03-3.5` 机制 7）。

## 4.3 推理链与防剧透

- **结构**：与 `05` 的 **30/50/20** 网状叙事比例配套——真相碎片埋在农场各处，避免「世界中心主义」单线灌输。
- **防剧透**：蜗牛全知但极慢；老鼠可卖真情报但分段解锁；笔记本采用**手帐式固定槽位 + 过程行笔记**（详见 `09-9.2`），槽位按发现进度逐项被画入 / 钉入 / 贴入，假设错误时**当场划掉、旁注修正**，避免 UI 一次性剧透列表，也让推理过程本身成为可视化叙事。

## 4.4 提示、校验与卡关兜底

- **空间提示**：屋顶、猫门等仍为硬能力闸；黄灯误读由悲伤蛙谜语或卧室物证独占。
- **防跳关**：**第一章假蛋真相 + 大黄清醒** 后解锁黑猫；黑猫索要玩具 → 老鼠情报 → 青蛙处取回（链式依赖明确）。
- **软提示**：老鼠可花钱买到「根本没有水怪」等反误导信息。

## 4.5 错误反馈与重试

- **质询失败**：NPC 给喜剧化嘴硬或跑题，不惩罚死亡；可重复尝试出示物品。
- **玩家认知错位**：允许阶段性「错误推理」存在（喜剧来源），但每一章需有**硬物证**回收错误方向（假蛋、卧室物证）。
