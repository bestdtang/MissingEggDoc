# 04 侦探推理与线索系统

## 4.1 推理目标与胜利条件

- **表目标**：满足淑芬委托，找回「画爱心的宝贝蛋」的下落并给玩家可理解的结案叙述。
- **里目标**：拆穿多层动物误会链，最终在红顶屋完成**物证总览闭环**（保温箱、软垫、黄灯、池塘水等要素一一对应，见 `05`/`07`）。

## 4.2 线索类型与获取方式

| 类型 | 示例 | 获取方式 |
|------|------|----------|
| 强相关真相碎片 | 主人舀水加湿、黄灯与红顶屋动线 | 进入卧室物证阅读、黑猫控诉与窗外可读信息 |
| 弱相关生活抓马 | 大黄宿醉自责、淑芬离窝狂奔 | 对话、环境叙事 |
| 红鲱鱼 | 石头假蛋、乌鸦囤积、青蛙谜语 | 场景拾取、NPC 夸饰发言 |
| 物证实体 | 假蛋石头、薄荷鱼等 | 探索拾取、质询交换、跑腿产出 |

## 4.3 推理链与防剧透

- **结构**：与 `05` 的 **30/50/20** 网状叙事比例配套——真相碎片埋在农场各处，避免「世界中心主义」单线灌输。
- **防剧透**：蜗牛全知但极慢；老鼠可卖真情报但分段解锁；避免 UI 一次性剧透列表（笔记本条目建议随发现渐进解锁）。

## 4.4 提示、校验与卡关兜底

- **空间提示**：屋顶、猫门等仍为硬能力闸；黄灯误读由悲伤蛙谜语或卧室物证独占。
- **防跳关**：**第一章假蛋真相 + 大黄清醒** 后解锁黑猫；黑猫索要玩具 → 老鼠情报 → 青蛙处取回（链式依赖明确）。
- **软提示**：老鼠可花钱买到「根本没有水怪」等反误导信息。

## 4.5 错误反馈与重试

- **质询失败**：NPC 给喜剧化嘴硬或跑题，不惩罚死亡；可重复尝试出示物品。
- **玩家认知错位**：允许阶段性「错误推理」存在（喜剧来源），但每一章需有**硬物证**回收错误方向（假蛋、卧室物证）。
