# 01 项目概述与目标

## 1.1 一句话定位

基于**抖音小火人原生系统**的 **3D Lowpoly 探案小世界**：玩家扮演**小火人侦探**，在一座充满戏剧张力的**微缩农场**中，通过收集线索、破解动物们的「跨服对话」，寻找母鸡**淑芬**丢失的、画有爱心标记的**宝贝蛋**（全流程真相与玩家认知存在刻意错位，终局收束为温柔反转）。

## 1.2 品类与参考

- **品类融合**：治愈探索 + 荒诞推理 + 情感反转；轻量 3D 场景叙事 + 推理收集。
- **体验关键词**：「跨频道聊天」、冷幽默、官方吐槽、红鲱鱼高能误导、短视频友好切片。

## 1.3 设计目标与非目标

**目标**

- **核心体验**：治愈探索 + 荒诞推理 + 情感反转（与对外一句话一致）。
- **目标时长**：**约 1–1.7 小时**全流程（与 `02`/`07` 对齐；甲方拍板后填死数）。
- **平台基因**：适配短视频节奏；内置多个**高能二创传播点**（如闪电蜗牛慢语速、悲伤蛙谜语、大黄醉狗翻身等——具体演出以 `07`/`09` 为准）。

**非目标（初版可明确写「不做」以免范围膨胀）**

- 不做重操作战斗主循环；不以数值养成为核心卖点。
- 不以全语音演出为硬性交付（听觉策略见 `11`）。

## 1.4 范围边界

- **MVP 叙事主骨架**：**两章**结构（天降大锅 → 红顶屋温柔反转）+ 尾声蜗牛收束；动物群像与物证链以 `05`/`06`/`07` 为准。
- **后续版本**：更多案件单元、皮肤与拍照分享深度、抖音能力扩展等，待 `13` 中 TBD 决策后单独立项。
