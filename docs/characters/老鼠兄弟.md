# 黑市老鼠兄弟

## 人设

- **视觉**：两只胖灰鼠；缩在谷仓墙 **缝隙情报窗口**；橡子壳破帽、极小微算盘或记账本。
- **性格**：极其市侩、唯利是图、胆大包天（敢偷猫物）、纯正 **乐子人**。
- **关系**：与 **黑猫** 长期汤姆杰瑞式战争——猫守缝外掏鼠，鼠趁不在偷猫私物报复。

## 时间线（事实链）

1. **猫鼠战利品与遗失（T-2）**  
   - 趁黑猫巡视，溜进院子偷 **【极品薄荷鱼玩具】**，拟带回洞当靠枕。  
   - 逃跑途中 **大黄** 突发酒疯狂吠；鼠惊慌过池塘边，把薄荷鱼掉在荷叶旁（后被 **悲伤蛙** 当「腐朽浮木」垫臀）。

2. **水怪谣言（T-3 傍晚，与小鸡线衔接）**  
   - 见小鸡推泥巴蛋好玩，**随口编造**「沼泽水怪」吓唬小鸡（农场本无水怪）。

3. **墙缝听觉（T-1 深夜）**  
   - 主人亮黄灯的卧室方向，兄弟在墙缝听见极清脆 **「咔嚓」** 碎裂声（与真相对照用，勿与假蛋线混淆；具体声学对象以 `05` 定稿为准）。

## 交互与台词

### 资本家的情报售卖（讨回猫玩具）

- **位置**：谷仓墙缝。  
- **玩家**气势汹汹要薄荷鱼。  
- **老鼠 A**（敲算盘、贱笑）：「要玩具？大侦探尴尬了。我们确实偷了那蠢猫的宝贝，哈哈！但逃跑时**弄丢了**。」  
- **老鼠 B**（搓手）：「不过我们知道掉哪儿了。重量级情报，一口价 **三块高级奶酪碎**（农场解谜/收集代币），少一块都不行！」

### 「盲盒情报箱」（从开局可持续）

- 上交奶酪碎后，老鼠倒卖「农场秘密」。  
- **前期**：多为胡扯（例：大黄项圈其实是镀银、乌鸦窝里藏了主人假牙等——具体清单可关卡填表）。  
- **上交增多后**吐露可核对事实，例如：  
  - **老鼠 A**：「水怪？哈哈哈哈！那群戴假墨镜的小鸡还真信！我们三天前看它们推泥巴蛋好玩**随口编的**！这农场根本没有水怪！」  
  - **老鼠 B**：「就刚刚，主人亮黄灯那屋里，我们墙缝听得一清二楚——**咔嚓！** 极度清脆的碎裂声！」

### 与黑猫支线的接口

- 黑猫大橡树台词要求玩家向老鼠夺回玩具、且「别客气」——与本节付费情报、遗失下落衔接（玩具 → 池塘青蛙，见悲伤蛙详档）。
