# 看门狗 · 大黄

## 人设

- **视觉**：体型庞大但憨态的金毛或中华田园黄狗；颈挂金属牌「保安（Guard）」。  
- **开局状态**：极度宿醉与悲痛；头顶敷额湿树叶，瘫狗窝旁；周围散落主人酿酒用 **发酵苹果渣**；不时极响酒嗝。
- **性格**：极度忠诚、责任心爆棚、**不会撒谎**、老实巴交；一伤心就借酒消愁，**酒品极差**。
- **动机**：农场保安，誓言保护财产；自认遭遇职业生涯奇耻大辱，深陷自责。

## 时间线（事实链）

1. **耻辱的清晨（T-2）**  
   - 院中尽职巡逻时，屋顶 **乌鸦** 俯冲从**草丛**掠走 **画爱心的白色圆球（假蛋石）**。  
   - 大黄狂吠扑咬，笨重跳不高，只咬到空气；眼看「淑芬的孩子」进乌鸦屋顶。  
   - 心理崩溃：自认不可饶恕失职，甚至**没脸告诉淑芬**。

2. **借酒消愁（T-2～T0 前连续约两天）**  
   - 极度自责下罢工，偷吃地窖旁发酵苹果渣，烂醉如泥约两天。  
   - **后果**：院门警戒心全面下降。

3. **昨晚的恐怖呼噜（T-1 深夜）**  
   - 主人取走真蛋时大黄仍深度宿醉。  
   - **误导音效**：睡梦中酒精作用发出沉闷、带奇怪哨音的巨大鼾声（类似「呼噜噜——咕呱」）。  
   - 声音传到鸡舍，加剧本已心虚的 **小鸡侦探团** 恐惧——被脑补为「吃蛋水怪」恐怖低吼。

## 交互与台词

### 悲痛的醉汉（强化锁定乌鸦）

- **条件**：玩家在池塘/小鸡线后仍存疑，或需要「老实人背书」时来找大黄。  
- **大黄**（酒嗝、捂脸）：「嗝……大侦探……你别问了，我不配戴这根保安项圈，把它拿走吧呜呜呜……」  
- **玩家问**是否见乌鸦偷蛋。  
- **大黄**（时间线错乱，痛哭）：「是的！嗝……就在不久前……天黑黑的还是亮亮的？我头好痛记不清了！但我发誓！我亲眼看到那只**黑恶魔**把**白色的宝贝**掠走了！我跳得不够高……我没保护好淑芬的希望！所以我只能用酒精麻醉自己……嗝！」  
- **玩家体验**：最老实 NPC 的「强力证词」打消怀疑，**彻底锁定第一章 Boss 位 = 乌鸦**。

> **设计注意**：大黄此处「亲眼目击」的证词，其实有一半是被 **黑猫在 T-2 清晨** 植入的：黑猫当时漫不经心说了一句「那乌鸦把草丛里白色圆球叼到屋顶去了」，精准落进大黄已崩溃的内疚里，让「白色圆球 = 淑芬的蛋」的错误等式从此牢不可破。大黄并非在撒谎——他真的相信自己看见了偷蛋，只是宿醉断片 + 黑猫的一句话共同构建了这份「记忆」。

### 真相大白后的救赎

- **条件**：玩家从乌鸦巢取得【鬼脸/爱心石头假蛋】并出示。  
- **大黄**（揉醉眼）：「这……这是一块画着鬼脸的石头？！等等！嗝……所以我拼死想救的、害我内疚到连喝两天酒的……是个假玩意儿？！」  
- **大黄**（站起、树叶掉落、眼神清澈）：「汪！太好了！我没弄丢淑芬的孩子！我不是废柴保安！」  
- **恢复保安姿态**：「报告大侦探！既然那是石头，真蛋一定还在危险中！我醉着时……好像听见**很重的人类雨靴**往**红顶屋**走，门廊里还有**刺眼的黄灯**一闪一闪。真蛋会不会被两脚兽拿走了？**大橡树上那只黑猫**能进屋子——你去找它！」
