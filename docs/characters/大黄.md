# 看门狗 · 大黄

## 人设

- **视觉**：体型庞大但憨态的金毛或中华田园黄狗；颈挂"保安（Guard）"金属牌。
- **开局**：极度宿醉与悲痛；头顶湿树叶，瘫狗窝旁；周围散落主人酿酒用**发酵苹果渣**；不时极响酒嗝。
- **性格**：极度忠诚、责任心爆棚、**不会撒谎**；一伤心借酒消愁，酒品极差。

## 时间线（事实链）

1. **T-2 清晨**：院中巡逻时**乌鸦从草丛掠走画爱心和鬼脸白石头**（石头假蛋），大黄狂吠扑咬但跳不高、只咬到空气；**认定**那便是**淑芬的蛋**被乌鸦衔走，自责自此扎根。
2. **T-2～T0 前**：失职之下连日偷吃地窖旁发酵苹果渣，烂醉两天，警戒全面下降。
3. **T-1 深夜**：深度宿醉，发出**沉闷带哨音的鼾声**（"呼噜噜——咕呱"）——**主人借此声惊醒并确认时机外出**；同声又被鸡舍内小鸡误读为"吃蛋水怪低吼"。

> **大黄「亲眼目击」**：T-2 清晨，乌鸦从草丛衔走**画爱心和鬼脸白石头**飞向谷仓方向，大黄追击不及，**认定那便是淑芬的蛋**（实为石头假蛋，属视觉误判）。**同日稍后**起他因自责灌醉自己；至玩家于狗窝晤谈②阶段时，口内仍咬定「乌鸦叼蛋」，唯昼夜、次序等细节含糊。黑猫当时只讥一句地毯，**没有和大黄聊起衔走之物**。

## 第一章 · 四阶段状态机

每阶段进入有明确触发条件，下一阶段台词在前一阶段菜单中不出现（关键质询锁）。

| 阶段 | 状态 | 玩家可拿到 | 进入下一阶段 |
|------|------|-----------|--------------|
| ① 完全睡 | 鼾声沉闷哨音 | 笔记本自动入【沉闷带哨音的鼾声】 | **轻交互**（呼喊/拍/拨水盆，无锁） |
| ② 半睡半醒 | 宿醉复述清晨目击 + 干草垛环境观察 | **【大黄目击：乌鸦叼蛋上屋顶】**（不锁，自动入）+ **【大黄身下压着干草垛】** | **解酒小任务**（交付【谷物泡水】） |
| ③ 清醒 · 未知假蛋 | 清醒后补述清晨链，仍坚信乌鸦偷蛋 | 应玩家请求交出**【干草垛】** | 玩家**出示【石头假蛋】** |
| ④ 清醒 · 已知假蛋 | 保安归位 + 嗅觉追踪 offer | 「**真蛋一定还在！我用鼻子帮你追！**」任务点迁移红顶屋外 | 进入第二章 |

### 阶段 ② 关键台词要点

- "嗝……清清楚楚……**乌鸦把淑芬的蛋叼上屋顶了**……我跳不够高……没拦住……头疼……昨晚还是今早我分不清了……" → **【大黄目击：乌鸦叼蛋上屋顶】**自动入笔记本（不锁）。越早带偏越好（玩家仍不知那是石头）。
- **环境观察**：大黄屁股下压着一堆干草垛（"硌得他直翻身"，宿醉无力起身挪），玩家观察后入笔记本**【大黄身下压着干草垛】**——阶段 ③ 关键质询的锁。
- **拼合效果**：与悲伤蛙关键质询里**蛙式修辞**「黑色恶魔拍碎玉盘」拼合（**「黑色恶魔」仅蛙口，大黄不说**）→ 玩家把「黑鸟 / 昨夜 / 池塘异象」**误缝成一条线**，高度确信乌鸦 = 偷蛋元凶，攀谷仓顶动机服务。

### 阶段 ②→③ 解酒小任务

**提示来源（三源任一即可、无锁）**：
- 小鸡阶段一·解酒小帮忙：「鸡舍水槽边老谷物泡水，大黄叔叔上次就喝那个好的」
- 淑芬阶段 1（母鸡常识口径，更心疼大黄）
- 老鼠一档 B 组 #7（1 块奶酪，醉因 + 解酒双信息打包）

**辅助暗示**：狗窝旁**旧木桶**（桶底干谷物渣 + 谷糠气味 + 旧咬痕）保留作为加深"鸡舍那边的味儿"印象的环境物件，**不再是必经线索**。

**执行**：鸡舍水槽拾【谷物泡水】 → 交大黄。大黄醒后湿树叶掉落、神智回，**仍坚信乌鸦偷蛋**。

### 阶段 ③ 关键质询：索要干草垛

- **锁**：持【大黄身下压着干草垛】+ 大黄已 ③。
- **未满足**：选项不出现，玩家可回狗窝补观察（无硬卡关）。
- **满足**：「**能不能把那堆干草给我？**」→ 大黄："这破草垫子？拿去吧——本汪鼻子回来了，也该站起来巡逻了。" → 获【干草垛】 → 谷仓底部交互点放置 → 平台跳跃路径打通 → 攀谷仓顶。

**阶段 ③ 语气**：仍以"蛋在乌鸦屋顶"为前提，给草垫时是"你想上屋顶？正好我也想巡逻"，**而非已知乌龙后的歉疚**。

### 阶段 ③ 可选质询：谷仓午睡点碎片记忆

- **触发**：持【谷仓午睡点】+ 大黄已 ③。
- **满足**：「谷仓角落那片被压扁的草窝是哪个动物的？」 → 大黄回忆："我嚎叫时脑子糊着、好像看见**两个灰乎乎的东西抱着什么往红顶屋那边跑**！太快了没看清……"——这只是可问对话补充，不作为线索点，也不锁后续。

### 阶段 ③→④：出示【石头假蛋】

- **锁**：持【石头假蛋】+ 大黄已 ③。
- **满足**：「[出示假蛋] 你昨天看到的真不是这玩意儿？」 → 进入 ④。

### 阶段 ④：保安归位 + 嗅觉追踪 offer

- 揉眼瞪着假蛋：「这……是块画着鬼脸的石头？！所以我害我内疚到连喝两天酒的是个假玩意儿？！」
- 项圈擦亮、尾巴疯狂摇摆：「太好了！我不是废柴保安！」
- 鼻子转向农场嗅一下，神情郑重指向**红顶屋**：「**淑芬的蛋——气味从那栋红顶屋飘出来的，就是那里。我跟你过去守着。**」
- **任务点迁移**：从狗窝迁至**红顶屋外**（不是大橡树）；与玩家共同进入第二章（沿途**必经淑芬**，强制情感节拍）。
- 大黄证词便签贴入第二页中上槽位（雨靴两趟 / 黄灯 / 红顶屋指向）。
- **可选扩写（非必播）**：自嘲那带哨音的呼噜先把院子吵得发闷，迷糊里像听见雨靴在泥地来回——与老鼠"两趟雨靴"、小鸡"恐怖低吼"形成噪音时间轴回响；叙事因果（鼾声惊醒主人）仍留在主人时间线内侧、不对玩家直说。

## 第二章 · 红顶屋外任务点

### 二次嗅觉确认

抬头嗅 + 二次确认：「没错——蛋的气味就在屋子里，我敢用保安项圈担保。但这门我撞不开，你绕屋子转一圈看看能不能找到别的路子。」 → 玩家自由探查红顶屋外。

### 玩家质询"这门给谁的" → 摇树召唤黑猫

- **触发**：玩家持【下树线索】（精美小门 + 不明兽毛 / 食碗等 ≥ 2 条），回大黄处。
- **未满足**：大黄只说「快进去看看蛋是不是还在！」摇树选项不出现，笔记本提示"再看看红顶屋外"。
- **满足**：「**这里有一扇精美的小门，还有动物的毛——这门是给谁的？**」 → 大黄拍脑门："黑猫！那只傲慢的蠢猫！他有专属猫门！" → 走到大橡树下撩拨树干 / 摇树 → 「**猫大爷！下来！这事得你出马！**」 → 黑猫现身（详见 `characters/黑猫.md`）。

**第二章后续位置**：大黄全程蹲守大橡树下，配合两路谈判段的撩猫戏（"嘴硬底软兄弟"反差萌）；尤其在两路均满足后补一脚撩拨"你昨晚可是被两脚兽当面骗了一手、自尊扫地"给黑猫台阶，让它有体面理由起身去开窗。

## 二周目结尾对话（待补）

风味：保安归位的踏实 + 自责彻底翻篇——湿树叶摘下、项圈擦亮、尾巴摇起来；可点对黑猫"嘴硬底软兄弟"的吐槽（"那只破猫终于把垫子要回去了"），或对主人温柔介入的迟来感激。
