# 看门狗 · 大黄

## 人设

- **视觉**：体型庞大但憨态的金毛或中华田园黄狗；颈挂"保安（Guard）"金属牌。
- **开局**：极度宿醉与悲痛；头顶湿树叶，**瘫在谷仓墙根旁**（根本没回狗窝）；身下压着侧倒的短木梯，周围散落**发酵苹果渣**；不时极响酒嗝。
- **性格**：极度忠诚、责任心爆棚、**不会撒谎**；一伤心借酒消愁，酒品极差。

## 时间线（事实链）

1. **T-2 清晨**：院中巡逻时**乌鸦从草丛掠走画爱心和鬼脸白石头**（石头假蛋），大黄狂吠扑咬但跳不高、只咬到空气；**认定**那便是**淑芬的蛋**被乌鸦衔走，自责自此扎根。
2. **T-2 上午～T0**：清晨追乌鸦跳不够高后，**踩上谷仓墙根短木梯**想翻过入口小围墙继续追鸟，梯脚打滑侧倒；挫败之余偷吃发酵苹果渣；酒劲上头，**执念回谷仓旁**趴在倒下的梯边等乌鸦，**醉死在梯上**，一压两天一夜，**未回狗窝**（详见下表「木梯 · 为何压在身下」）。
3. **T-1 深夜**：深度宿醉，发出**沉闷带哨音的鼾声**（"呼噜噜——咕呱"）——**主人借此声惊醒并确认时机外出**；同声又被鸡舍内小鸡误读为"吃蛋水怪低吼"。

### 木梯 · 为何压在身下（事实链）

| 问题 | 答案 |
|------|------|
| 梯子在哪？ | **谷仓墙根**短木梯；作用是翻过谷仓入口小围墙，进入后续平台跳跃路线。抓鸟时踩梯，**梯脚打滑侧倒**，横在墙根地面。 |
| 大黄在哪？ | **同一地点**——谷仓旁，**没有回狗窝**；酒醉后趴在倒梯边等乌鸦，半个身子压在踏板上。 |
| 玩家先发现什么？ | 路过**狗窝**（E22 · 氛围）见空窝——只作为"保安不在岗"的即时提示，不入笔记本；循鼾声或查谷仓 → 才在**谷仓旁**找到他（E04）。 |
| 为什么压着？ | 不是守梯子，是**踩梯失败后喝酒 → 执念趴等 → 醉死**，一压至 T0。 |
| 清醒后可问 | 「梯子都踩倒了，鸟还在头顶叫……本汪就在这儿等，等睡着了。」 |

> **大黄「亲眼目击」**：T-2 清晨，乌鸦从草丛衔走**画爱心和鬼脸白石头**飞向谷仓方向，大黄追击不及，**认定那便是淑芬的蛋**（实为石头假蛋，属视觉误判）。**同日**踩梯再败、灌醉，至 T0 仍在**谷仓旁**与玩家晤谈②阶段；口内仍咬定「乌鸦叼蛋」。黑猫当时只冷冷重复了一遍乌鸦叫大黄的「地毯」，**没有和大黄聊起衔走之物**。

## 第一章 · 四阶段状态机

> 谷仓段节点对照：**1-A** 初见 → **1-A′** hub → **1-B**/**1-C** 叫醒 → **1-D** hub → **1-E**/**1-F**/**1-G**。详见 [大黄-对话脚本-树状样章](./大黄-对话脚本-树状样章.md)。

每阶段进入有明确触发条件，下一阶段台词在前一阶段菜单中不出现（关键质询锁）。

| 阶段 | 状态 | 玩家可拿到 | 进入下一阶段 |
|------|------|-----------|--------------|
| ① 完全睡 | 鼾声沉闷哨音 | 笔记本自动入【沉闷带哨音的鼾声】 | **轻交互**（呼喊/拍/拨水盆，无锁） |
| ② 半睡半醒 | 宿醉复述清晨目击 + 木梯环境观察（E04） | **【大黄目击：乌鸦叼蛋上屋顶】**（不锁，自动入）+ **【大黄身下像压着一架木梯】**（E04，拼合锁 1/2） | **木梯拼合锁** + **叫醒大黄**（交付【谷物泡水】） |
| ③ 清醒 · 未知假蛋 | 清醒后补述清晨链，仍坚信乌鸦偷蛋 | 应玩家请求交出**【木梯】** | 玩家**展示【屋顶白石头插图】** |
| ④ 清醒 · 已知假蛋 | 保安归位 + 嗅觉追踪 offer | 「**真蛋一定还在！我用鼻子帮你追！**」任务点迁移红顶屋外 | 进入第二章 |

### 阶段 ② 关键台词要点

- "嗝……清清楚楚……**乌鸦把淑芬的蛋叼上屋顶了**……我跳不够高……没拦住……头疼……昨晚还是今早我分不清了……" → **【大黄目击：乌鸦叼蛋上屋顶】**自动入笔记本（不锁）。越早带偏越好（玩家仍不知那是石头）。
- **环境观察（E04）**：**谷仓墙根**侧倒的短木梯，大黄半个身子压在踏板上；玩家视角记为**「像木梯」**，入笔记本**【大黄身下像压着一架木梯】**——**木梯拼合锁 1/2**。若玩家尚未发现 E06 小围墙缺梯，只记录本条，不提出挪梯需求；若已持 E06 条目，半睡半醒时玩家可开口让他挪开，大黄迷糊拒绝。进入阶段 ③ 后不再另设「已观察木梯」索要锁。
- **空窝（E22）**：玩家调查**狗窝**时给出"保安不在岗"的即时空间反馈，不入笔记本；与谷仓旁鼾声/E04 共同把玩家导向「去谷仓边找他」。
- **拼合效果**：与悲伤蛙关键质询里**蛙式修辞**「黑色恶魔拍碎玉盘」拼合（**「黑色恶魔」仅蛙口，大黄不说**）→ 玩家把「黑鸟 / 昨夜 / 池塘异象」**误缝成一条线**，高度确信乌鸦 = 偷蛋元凶，攀谷仓顶动机服务。
- **木梯拼合锁（一-7 前置，顺序不限）**：

| 线索 | 来源 | 笔记本条目 |
|------|------|------------|
| 缺梯 | 谷仓入口 E06 首次调查（未持【木梯】状态） | 【谷仓入口有小围墙，没有木梯翻不过去】 |
| 像梯 | 谷仓旁 E04 环境观察 | 【大黄身下像压着一架木梯】 |

**二者齐备** → 笔记本写入任务指针「得叫醒大黄，向他借木梯」→ 方可持【谷物泡水】进入叫醒流程。仅一条时：递泡水不叫醒；醉话**不**提前说破「梯子在我这儿」（避免单线剧透）。

### 阶段 ②→③ 叫醒大黄

**执行锁**：大黄处于阶段②半睡半醒 + 持【谷物泡水】。E06 缺垫脚物不是醒酒前置。

**提示来源（三源任一即可；泡水位置无拼合要求）**：
- 小鸡阶段一·叫醒小帮忙：「鸡舍水槽边老谷物泡水，大黄叔叔上次就喝那个好的」
- 淑芬阶段 1（母鸡常识口径，更心疼大黄）
- 老鼠一档 #7（1 块奶酪，醉因 + 叫醒法双信息打包）

**辅助暗示**：**狗窝**旧木桶 E11（桶底干谷物渣 + 谷糠气味）保留作为泡水方向的环境暗示，与 E22 空窝同区；**不再是必经线索**。

**执行**：鸡舍水槽拾【谷物泡水】 → 交大黄。大黄醒后湿树叶掉落、神智回，**仍坚信乌鸦偷蛋**；从梯上爬起来，木梯完整露出来。

### 阶段 ③ 关键质询：索要木梯

- **锁**：大黄已 ③ + 持【谷仓入口有小围墙，没有木梯翻不过去】。
- **未满足**：选项不出现；若大黄已醒但玩家未发现 E06，先去确认谷仓入口缺少垫脚物。
- **满足**：「**能把梯子借我吗？我要去谷仓顶。**」→ 大黄："哦……刚才压在我身下那架。拿去吧。" → 获【木梯】 → 谷仓入口小围墙架设 → 平台跳跃入口打通 → 攀谷仓顶。

**阶段 ③ 语气**：仍以"蛋在乌鸦屋顶"为前提；借梯只解决通路，不恢复嗅觉自信，**而非已知乌龙后的歉疚**。

### 阶段 ③ 可选质询：谷仓午睡点碎片记忆

- **触发**：持【谷仓午睡点】+ 大黄已 ③。
- **满足**：「谷仓角落那片被压扁的草窝是哪个动物的？」 → 大黄回忆："我嚎叫时脑子糊着、好像看见**两个灰乎乎的东西抱着什么往红顶屋那边跑**！太快了没看清……"——这只是可问对话补充，不作为线索点，也不锁后续。

### 阶段 ③→④：展示【屋顶白石头插图】

- **锁**：笔记本持【屋顶白石头插图】+ 大黄已 ③。
- **满足**：「[展示笔记本] 你昨天看到的真不是这玩意儿？」 → 进入 ④。

### 阶段 ④：保安归位 + 嗅觉追踪 offer

- 揉眼瞪着笔记本插图：「这……是块画着鬼脸的石头？！所以我害我内疚到连喝两天酒的是个假玩意儿？！」
- 项圈擦亮、尾巴疯狂摇摆：「太好了！我不是废柴保安！」
- 鼻子转向农场嗅一下，神情郑重指向**红顶屋**：「**淑芬的蛋——气味从那栋红顶屋飘出来的，就是那里。我跟你过去守着。**」
- **任务点迁移**：从**谷仓旁**迁至**红顶屋外**（不是大橡树）；与玩家共同进入第二章（沿途**必经淑芬**，强制情感节拍）。
- 大黄证词便签贴入第二页中上槽位（雨靴两趟 / 黄灯 / 红顶屋指向）。
- **可选扩写（非必播）**：自嘲那带哨音的呼噜先把院子吵得发闷，迷糊里像听见雨靴在泥地来回——与老鼠"两趟雨靴"、小鸡"恐怖低吼"形成噪音时间轴回响；叙事因果（鼾声惊醒主人）仍留在主人时间线内侧、不对玩家直说。

### 首次抵达（**2-A**）

`!RedRoof_IntroShown` 时第一次点大黄：「就是这儿。」→ 写 `RedRoof_IntroShown`。大黄抬头嗅 + 二次确认：「没错——蛋的气味就在屋子里，我敢用保安项圈担保。先看看门。」

玩家调查紧闭大门（**E13**）后确认正门进不去；**2-hub** 未查 E13 时只播催查「你先去看看那扇门。」

### 红顶屋外 hub（**2-hub** · 黑猫未召唤）

`!Dog_BlackCatSummoned` 期间的主交互 hub：

| 菜单 | 条件概要 | 节点 |
|------|----------|------|
| 「正门进不去。」 | `E13` 已查，未播过碰壁 | **2-A′** 一次性 |
| 「红屋顶里还住着别的动物？」 | `TreeClueCount >= 2` | **2-B** 摇树 |

**下树线索**：**E14–E16 / E28** 各首次 +1，`TreeClueCount >= 2`（**不含 E13**）。计数齐备且已查 E13 后，摇树项才出现。

### 摇树召唤黑猫（**2-B** → 黑猫 **2-A**）

`E13_ViewDoorBlocked && TreeClueCount >= 2` 时，**2-hub** 选「红屋顶里还住着别的动物？」→ 大黄拍脑门想起黑猫，摇树：「**猫大爷！下来！这事得你出马！**」→ 写 `Dog_BlackCatSummoned` → 黑猫 **2-A** 三人同场。

**2-A** 播毕后**主交互切至黑猫**（黑猫 **2-hub** 谈判菜单）。再点大黄走 **2-C** 闲聊轮播。

### 黑猫现身后（**2-C** / **2-E**）

- **2-C**：黑猫已下树、未进屋（或攻顶前加油已播）——等权重闲聊轮播，无推进。
- **2-E**：黑猫 **2-E** 汇合开窗写入 `BlackCat_Entered` 后，攻顶前点大黄一次性「加油。我等着真相。」→ 写 `RedRoof_RoofWaitShown`。

激将正文（「你昨晚可是被那两脚兽当面骗了一手……」）写在黑猫 **2-E** 节点内；大黄样章 **2-D** 仅为跳转占位，不重复台词。

**第二章后续位置**：大黄全程蹲守大橡树下；两路谈判期间配合撩猫戏（"嘴硬底软兄弟"反差萌），汇合开窗时给黑猫台阶让它有体面理由起身。
