# 乌鸦 · 对话脚本（树状）

> **状态**：乌鸦对话**实施准稿**（以本树状脚本为准）。  
> **变量**：见 [17-全局游戏状态变量](../17-全局游戏状态变量.md)；本脚本只引用该表，不另造变量。  
> **描述行**：text 树块内一律 `描述：（……）`，见 [18 §18.2](../18-树状对话脚本生成方法.md)。  
> **方法**：[18](../18-树状对话脚本生成方法.md) · [16](../16-NPC对话脚本书写守则.md)。

---

## 流程总览

**一、登顶前 · 路径喊话（点击交互）**

1. **1-A** 谷仓底仰望（**点击** E06 架梯后交互点）
2. **1-B** / **1-C** 攀爬 **E35 / E36** 路径喊话点（**点击**地点特效）

**二、屋顶巅峰对峙**

1. **2-A** 对峙（**登顶强制播**）→ 玩家点 **E10** → **2-B** → **2-hub【回访】+【菜单】**
2. **2-hub** → **2-C** 玻璃珠追问 / **2-D** 离场
3. **2-E** 屋顶回访【轮播】；`E08&&!Crow_GlassBeadAsked` 首轮 → **2-C** → 对话结束

**二周目**：`NGPlus && !Crow_NGPlusShown` → NGPlus 一次性

变量写入见各节点【变量】；全局对照 [17 §17.12](../17-全局游戏状态变量.md#1712-乌鸦树状脚本速查)。

**对话结束**：氛围段、告辞、回访轮播、NGPlus 后结束。第二章乌鸦不参与主交互链。

## 一、登顶前 · 路径喊话

> 〔系统注〕第一章乌鸦常驻屋顶，**不可点乌鸦 NPC 菜单**。**1-A**～**1-C** 均为进入范围后**玩家点击交互键**（非强制播）。

---

### 1-A · 谷仓底部仰望

> 〔系统注〕`E06_LadderPlaced` 后，玩家**点击**谷仓底交互点播本段。

```text
1-A
│
└─ 描述：（谷仓顶传来一阵得意的嘎嘎声）
   乌鸦：本王的垃圾王国——今日也光辉灿烂！
   描述：（一声鸦叫拖长尾音，又归于安静）
   玩家：我得上去看看。

→ 对话结束
```

---

### 1-B · 攀爬途中 B-1（E35）

> 〔系统注〕`!E35_CrowTauntShown`。玩家进入 **E35** 范围后**点击交互键**播本段。

```text
1-B
│
└─ 乌鸦：谁？！谁在本王阵法边缘乱踩！
   乌鸦：滚远点！凡人免进！

→ 对话结束

【变量】
· E35_CrowTauntShown = true
```

---

### 1-C · 攀爬途中 B-2（E36）

> 〔系统注〕`!E36_CrowTauntShown`。玩家进入 **E36** 范围后**点击交互键**播本段。

```text
1-C
│
└─ 乌鸦：往上爬啊！往上爬！
   乌鸦：本王倒要看看你怎么上来！
   描述：（一阵嘎嘎大笑从屋顶传下来）

→ 对话结束

【变量】
· E36_CrowTauntShown = true
```

## 二、屋顶巅峰对峙

> 〔系统注〕**首登屋顶**：`!Crow_RoofIntroShown` 时 **2-A** **登顶强制播**（**不点乌鸦**）→ 玩家点 **E10** → **2-B** → **2-hub**。之后再**点乌鸦**走入口判定：
>
> 1. `NGPlus && !Crow_NGPlusShown` → **NGPlus**
> 2. `Crow_RoofIntroShown` → **2-E**；`E08_ViewBurnMark && !Crow_GlassBeadAsked` 时 **2-E**【条件】→ **2-C**；否则 **2-E**【轮播】

---

### 2-A · 对峙

> 〔系统注〕**首次登顶到达屋顶强制播**（不需点乌鸦 NPC）。须交代**黑猫曾上顶**等；`Crow_RoofIntroShown` 写在 **2-A**【变量】。播毕同链引导玩家点 **E10** 进 **2-B**。

```text
2-A
│
└─ 乌鸦：站住。
   乌鸦：踏入垃圾王国领地的凡人——报上名来。
   玩家：我是侦探。我来找淑芬的蛋。
   乌鸦：蛋？
   描述：（乌鸦歪头，眼珠子贼亮地转了一圈）
   乌鸦：本王这儿只有神灵宝石。
   乌鸦：部落图腾，绝版手办。你找错门了。
   玩家：大黄说，你把淑芬的蛋叼走了。
   乌鸦：那条跳跃能力为零的地毯？
   乌鸦：它隔那么远乱吠，懂什么。
   乌鸦：本王捡的是白白亮亮的大宝贝。
   乌鸦：就连那位也爬上来，瞪了本王的神灵宝石一眼——
   描述：（乌鸦得意地扬起下巴）
   乌鸦：然后他就怂了，一声不吭下去了。嫉妒，肯定是嫉妒。
   玩家：让我看看你那个神灵宝石行吗？
   乌鸦：看可以。
   描述：（乌鸦翅尖张开，挡住白石头）
   乌鸦：拿？想都别想。
   玩家：……我就看看。
   乌鸦：凡人，你最好只是看看。

→ 2-B

【变量】
· Crow_RoofIntroShown = true
```

---

### 2-B · 近距离观察白石头（E10）

> 〔系统注〕玩家点击巢中白石头；`E10` 环境写入 `E10_ViewWhiteStone`（见 `13`）。

```text
2-B
│
└─ 玩家：……
   玩家：这就是块石头。
   乌鸦：放肆！
   乌鸦：你懂什么图腾学？！
   乌鸦：爱心是远古封印！
   乌鸦：鬼脸是驱邪结界！
   描述：（乌鸦翅膀死死盖住白石头，只露出爱心那一面）
   乌鸦：充满神秘力量。

→ 2-hub【回访】+【菜单】
```

---

### 2-hub · 对峙后 hub

> 〔系统注〕仅 **2-B** 同链进入；`Crow_RoofIntroShown` 后再次点乌鸦走入口判定进 **2-E**，不回到本 hub。

```text
2-hub
│
├─ 【回访】
│  乌鸦：凡人，记清楚本王的辉煌。
│
└─ 【菜单】
   「谷仓下面那些烧焦的痕迹，是你干的？」（E08_ViewBurnMark && !Crow_GlassBeadAsked）→ 2-C
   「我先走了。」→ 2-D
```

---

### 2-C · 玻璃珠阵法

> 〔系统注〕**2-hub**【菜单】进入时 `Crow_GlassBeadRemedial` 保持 false；**2-E**【条件】分支进入前写 `Crow_GlassBeadRemedial=true`（见 **2-E**【变量】）。

```text
2-C
│
└─ 玩家：谷仓下面那些烧焦的痕迹，是你干的？
   乌鸦：本王的玻璃法阵。
   乌鸦：防贼防盗防黑毛。
   玩家：把稻草烧着了也叫防盗？
   乌鸦：那是阵法的余波。
   乌鸦：说明本王威力过猛。

→ 2-hub【回访】+【菜单】（!Crow_GlassBeadRemedial）
→ 对话结束（Crow_GlassBeadRemedial）

【变量】
· Crow_GlassBeadAsked = true
· Crow_GlassBeadRemedial = false
```

---

### 2-D · 离场

```text
2-D
│
└─ 玩家：我先走了。
   乌鸦：下去吧。
   乌鸦：别踩本王的宝贝。
   描述：（一声又长又得意的嘎嘎）

→ 对话结束
```

---

### 2-E · 屋顶回访

> 〔系统注〕`Crow_RoofIntroShown` 后任意回访。`E08_ViewBurnMark && !Crow_GlassBeadAsked` 时【条件】分支赴 **2-C**（写 `Crow_GlassBeadRemedial=true`）；否则【轮播】。

```text
2-E
│
├─ 【条件】（E08_ViewBurnMark && !Crow_GlassBeadAsked）
│  → 2-C
│
└─ 【轮播】
   ├─ 乌鸦：又来了？
   │  玩家：随便走走。
   │  乌鸦：参观收费。
   │
   ├─ 乌鸦：宝石还在。
   │  乌鸦：不用问。
   │  描述：（乌鸦用翅尖敲了敲白石头）
   │
   └─ 玩家：案子还没破。
      乌鸦：和本王有什么关系。
      乌鸦：本王的王国一切正常。

→ 对话结束

【变量】（仅【条件】分支赴 **2-C** 时）
· Crow_GlassBeadRemedial = true
```

## 二周目

```text
NGPlus
│
└─ 描述：（乌鸦蹲在巢边，比平时安静）
   乌鸦：……你来了。
   玩家：案子破了。
   乌鸦：本王知道。那就是块石头。
   玩家：……你现在承认了？
   乌鸦：本王从来都知道！爱心封印、鬼脸结界——都是本王亲自设计的考题！
   玩家：鬼脸是小鸡画的。
   乌鸦：……考题的一部分。
   描述：（乌鸦别过脸，翅尖把白石头往身下按了按）
   玩家：黑猫呢？你们和好了吗？
   乌鸦：那只黑毛？！昨天凌晨四点又来本王阵法边缘晃。
   乌鸦：本王用碎玻璃反光晃了他一眼——他就龇了一声「别照本喵」，甩着尾巴下去了。
   玩家：……还是没和好。
   乌鸦：本王不需要和好。全农场最高的王，不需要。
   乌鸦：下去吧，戴帽子的。
   描述：（乌鸦用翅膀盖住白石头，低下头）

→ 对话结束

【变量】
· Crow_NGPlusShown = true
```

## 条件覆盖自检

### 入口判定

**远距离**：`E06_LadderPlaced`→**1-A** · E35→**1-B** · E36→**1-C**

**点乌鸦**（`Crow_RoofIntroShown` 后）：`NGPlus&&!Crow_NGPlusShown`→**NGPlus** · `Crow_RoofIntroShown`→**2-E**（`E08&&!Crow_GlassBeadAsked` 首轮 → **2-C**）/【轮播】

**首登屋顶**（`!Crow_RoofIntroShown`）：登顶 → **2-A** 强制播 → 点 **E10** → **2-B** → **2-hub**（不经「点乌鸦」入口判定）

> **hub 子树**：**2-hub** 仅 **2-B** 同链可达；**2-C** 返链 `!Crow_GlassBeadRemedial` 为 **2-hub** / **2-E** 双入口分叉，保留。

### 节点 / 菜单 / 返链


| 节点         | 写入                                                     | 出口                                                                 |
| ---------- | ------------------------------------------------------ | ------------------------------------------------------------------ |
| **1-B**    | `E35_CrowTauntShown`                                   | 结束                                                                 |
| **1-C**    | `E36_CrowTauntShown`                                   | 结束                                                                 |
| **2-A**    | `Crow_RoofIntroShown`                                  | **2-B**                                                            |
| **2-B**    | —（`E10` 环境）                                            | **2-hub**                                                          |
| **2-hub**  | —                                                      | **2-C**/**2-D**                                                    |
| **2-C**    | `Crow_GlassBeadAsked` · `Crow_GlassBeadRemedial=false` | **2-hub**（`!Crow_GlassBeadRemedial`）/ 结束（`Crow_GlassBeadRemedial`） |
