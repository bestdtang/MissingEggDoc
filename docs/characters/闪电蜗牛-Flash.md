# 「闪电」蜗牛 Flash

## 人设

- **视觉**：极小蜗牛；位于农场中央（如鸡舍与红顶屋之间）一片 **巨大宽叶** 上；从开局到结束活动半径约 **不超过三厘米** 级（策划可按镜头填具体数）。
- **性格**：全知视角、情绪极稳、**慢到与凡人不在同一时钟维度**。
- **动机**：按自己的节奏客观陈述所见「真理」——听者是否听完不在其模型内。

## 时间线（事实链）

- **起点**：约 T-3，Flash 决定从该叶 **根部爬向叶尖**（策划定为约十厘米级「史诗旅程」）。
- **T-3 下午**：爬至叶约 1/3 处，**清楚看见**小鸡侦探团推走真蛋、塞石头全过程。
- **T-3 傍晚**：至叶中段，看见小鸡慌张把真蛋推回、爱心石丢进草丛。
- **T-2 清晨**：微转触角，目睹 **乌鸦** 从草丛叼走石头。
- **T-1 深夜**：至叶约 2/3 处，见证 **主人** 温柔抱真蛋回红顶屋。
- **T0（当前）**：近叶尖，准备把三天所见 **浓缩成极慢长句** 告诉奔波中的侦探。

## 交互与台词（分阶段）

### 1. 游戏刚开局（迷茫期）

- **情境**：淑芬大哭，玩家将四处问线索。  
- **玩家点击蜗牛。**  
- **Flash**：「我——」  
- **玩家 OS**（可 UI 字幕）：「这有个结巴，算了别浪费时间，先去问别的家伙吧。」

### 2. 解开乌鸦假蛋之谜后（震惊期）

- **情境**：玩家爬上屋顶，发现乌鸦守着的是小鸡画的假石蛋，怀疑人生。  
- **玩家路过再点蜗牛。**  
- **Flash**：「我——看——见——假——蛋——被——乌——鸦——叼——走——了——，真——蛋——被」  
- **玩家 OS**：「卧槽你早知道是假蛋？！等等真蛋怎么了？！接着说啊！！」

### 3. 章间焦灼期（多线搅局）

- **情境**：大黄断片、青蛙谜语、小鸡口径等搅在一起，玩家刚倾向「主人与昨夜」介入。  
- **玩家点蜗牛。**  
- **Flash**：「……真——蛋——昨——晚——被——主——人——」  
- **玩家 OS**：「被主人吃了？拿走了？你卡在这存心急死我吗！不指望你了，去找黑猫开门！」

### 4. 大结局后（卧室真相已见）

- **情境**：玩家已贿赂黑猫进卧室，见保温箱与破壳等，合上侦探笔记本。  
- **玩家最后一次点蜗牛。**  
- **Flash**：「……主——人——抱——进——了——温——暖——的——屋——子——。大——侦——探——，案——子——破——了——吗——？」  
- **玩家 OS**：「腿跑断被全农场忽悠一圈，剧本一直在你嘴里——语速快一点这游戏三分钟就通关了！」
