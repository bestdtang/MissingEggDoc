# 劳模母鸡 · 淑芬

## 人设

- **视觉**：体态丰满、羽毛蓬松；平时昂首挺胸，今日神情绷着。
- **性格**：**嘴硬心软，雷厉风行**。遇事不等不靠，自己先上——蛋一摸到冰冷就冲出去疯狂深蹲，正是这个性子。嘴上不饶人，心里最软；不轻易示弱，一旦真求助就是真到极限了。
- **动机**：当年那批小鸡（即"小鸡侦探团"）已破壳十天半个月在院子撒欢；**最后一颗蛋**迟迟无动静。淑芬画上**爱心标记**视若珍宝，连续数日自己守着——直到今早蛋不见了、满农场找不着，才不得不开口求助。

## 时间线（事实链）

1. **T-3 下午初 · 偷龙转凤**：淑芬暂时离窝（沙浴/觅食），小鸡趁机偷出真蛋，塞入画了爱心和鬼脸的石头假蛋（窝中羽毛碎屑略作包裹）。
2. **T-3 下午 · 踢球洗蛋**：小鸡带"弟弟"出院踢球，蛋滚进烂泥弄脏；路遇老鼠被随口编的"沼泽水怪"吓唬，仓皇推着脏蛋跑去池塘洗泥；悲伤蛙见状幽怨"宝珠终将沉入深渊"，小鸡魂飞魄散、确信水怪就在此处。
3. **T-3 下午末 · 淑芬回窝 · 趁乱换回**：
   - 淑芬回窝摸到石头，冰冷彻骨——孵化焦虑积压数日，这一刻直接击穿理性，**没识破是石头**，本能崩溃，院内**狂奔大叫、疯狂深蹲**，拼命用体温"救孩儿"。
   - 小鸡此时手里恰好拿着洗净真蛋，**趁淑芬跑出去尖叫的空档**，从鸡舍侧口将真蛋换回窝中，**石头假蛋慌乱扔进鸡舍外草丛**（后被乌鸦叼回屋顶巢 → E10 物证）。
   - 淑芬很快跑回——此时已是真蛋，感到微弱心跳与温度，以为自己折腾把孩儿暖回来了，松一口气安心继续孵化。
4. **T-1 深夜 · 真正的偷蛋贼**：主人备好设备从鸡舍取走爱心真蛋（人工孵化）。

## 交互阶段（全程可选回访）

> 淑芬全程在鸡舍内或门口。每阶段都可回来搭话，回答随进度更新。**对话不提供失蛋案硬线索**，但每句话折射她对孩子与主人的情感，是游戏情感线的底色。第二章大黄嗅觉追踪回鸡舍时玩家**必然路过淑芬**——自然的强制情感节拍。

### 阶段 0：委托（起因，强制触发）

树状节点 **1-A**。撑着尊严开口——"我的蛋不见了，画了爱心的那颗，今早没了。其他孩子出壳好几天了，就这最小的还没出来，我一直守着它。昨晚前它还像是轻轻动过一下，我不可能认错。"挺背："劳烦你帮我找找，好吗？"

补一句"主人这两天也老来窝边转悠，我赶他他也不走……"——既是嘴硬性格体现，也自然带出主人长期关注这颗蛋的底色。

### 阶段 1：第一章调查途中（可随时回访）

树状 **1-hub-intro** → **1-hub**（单句【回访】「……还有什么我能做的吗？」+ 话题 **1-B**～**1-G**）。点淑芬先播回访【轮播】，再进真 hub；子项返回直进 hub，不重播 intro。
- **关于主人**（无锁）：「主人那两天老在我窝边转悠……我赶他，他也不走，就站在旁边不说话。说不上什么感觉，就是……挺讨厌的。」→ 让玩家留意主人对这颗蛋的特殊关注，为第二章主人取蛋埋下温柔底色。
- **关于乌鸦**（**锁**：大黄 **1-A** 已播 · `DogStatus>=2`）：旁观者视角觉得"乌鸦最近怪，总往这边瞅"——有质感不硬指引。
- **关于悲伤蛙**：提醒"它整天发呆，别让它那些话把你绕进去"。
- **关于老鼠**（**锁**：玩家先与红顶屋墙缝老鼠交互过 ≥ 1 次）：「红顶屋墙缝那俩破毛球？嘴里没一句正经的——要是真知道什么早自己跑出来邀功换奶酪了。」→ **红鲱鱼回收旁证**，给"老鼠不可信"贴母鸡视角标签，为小鸡阶段二揭穿水怪铺垫。
- **询问谷仓午睡点**（持【谷仓午睡点】主动询问）：「不是我，我不出鸡舍那么远，犯不着跑去那边趴着。」→ 仅作为可问对话，不作为线索点，也不锁后续。
- **关于叫醒大黄**（与小鸡阶段一、老鼠一档 #7 **三源任选、无先后**。**锁**：玩家尚未持【谷物泡水】，避免重复提示）：「大黄那条蠢狗又喝多了？**狗窝都空着——准又醉在谷仓那边了**，你喊不醒的。**鸡舍水槽边那桶老谷物泡水**管用，他爱喝那个。上次也是我给他端去的。」（语气藏不住心疼）

### 阶段 2：小鸡认罪后（可选告知）

树状 **2-A**。锁：`E10_ViewWhiteStone` && `!Shufen_StoneRevealShown`。与 **3-A** 无先后。按序复述 **T-3 下午**：离窝沙浴 → 回来一摸冰凉 → 满院子跑、深蹲 → 回窝蛋又热、以为是自己焐热的 → 骂阿满/小崽子 → **边界**：大前天下午是这回事，今早发现没了是另一回事。

> 注：淑芬**不知**小鸡趁空换真蛋。真蛋 T-1 深夜被主人取走——边界句不剧透取蛋者。

### 阶段 3：第二章 · 主人方向（可选告知）

树状 **3-A**。锁：`DogStatus==4` && `!Shufen_MasterTrustShown`（与 **2-A** 无先后）。先否定「坏人」，补 **T-3 下午深蹲时主人在旁看着、他也很担心**，再「不会有事的……」

> 不是线索，是情感确认——玩家此时接近真相、尚未漫画揭晓，淑芬的信任暗示方向对、结局会是好的。与 **2-A** 无先后；未播 **2-A** 时，本句亦补全「主人当时在看着」这一格。

### 阶段 4：揭示后（进屋确认真相后告知）

"……那就好。""谢谢你。"
