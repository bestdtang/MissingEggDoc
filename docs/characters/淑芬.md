# 劳模母鸡 · 淑芬

## 人设

- **视觉**：体态丰满、羽毛蓬松；眼神充满母性。
- **性格**：温柔、勤恳，为孩子操碎心。
- **动机**：当年一批小鸡（即「小鸡侦探团」）已破壳十天半个月并在院子撒欢；**最后一颗蛋**迟迟无动静。淑芬画上 **爱心标记**，视若珍宝，耐心等最小的孩子降生。

## 时间线（事实链）

1. **冰冷的触感（T-3 下午）**  
   - 趁淑芬**暂时离窝**（院内觅食、饮水或沙浴等——具体镜头可美术定），**小鸡侦探团**偷出真蛋、塞入画爱心的石头假蛋。  
   - 淑芬回窝一孵立刻察觉异样：这颗「蛋」太冷、太硬，没有生命温度。  
   - 她跳出草窝，在院内**狂奔、大叫、疯狂深蹲**试图给窝「升温」（无其他母鸡可求助时的自救丑态）。

2. **主人视角与小鸡换蛋空档**  
   - **农场主**：在院中干活，看见平时稳重的淑芬失控狂奔深蹲；走到鸡舍理解为：最后一颗蛋迟迟不破壳，母鸡焦虑到「自己跟自己较劲」，打算翻《母鸡产后护理与孵蛋指南》准备帮忙。  
   - **小鸡窗口**：正因淑芬离窝折腾、场面混乱，被老鼠与青蛙吓破胆的小鸡侦探团获得空档，把**洗净的真蛋**推回窝，**石头假蛋**扔出鸡舍外草丛。

3. **温度回归（T-3 傍晚）**  
   - 淑芬回到窝再次趴蛋；此时窝内已是**真蛋**，她感到微弱心跳与温度，以为自己的折腾「终于把孩儿暖回来了」，松一口气继续安心孵化。

4. **真正的「偷蛋贼」（T-1 深夜）**  
   - 主人备好设备，从鸡舍取走爱心真蛋（人工孵化）。
