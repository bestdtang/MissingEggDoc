# 悲伤蛙

## 人设

- **视觉**：微胖青蛙；坐一块绿油油「垫子」（实为 **猫薄荷鱼玩具**）；眼神空洞，像随时叹出长气。
- **性格**：重度抑郁、存在主义口吻、永远在跨频道聊天；农场终极 **谜语人**。
- **动机**：哀悼池塘水与「美好事物（如月亮倒影）」的消逝、破碎；**对丢蛋案本身不关心**，甚至缺乏「蛋」的常识框架。

## 时间线（事实链）

1. **恐吓的源头：预言与月亮（T-3 傍晚）**  
   - 小鸡侦探团推沾泥真蛋来池边清洗；悲伤蛙盯水面 **洁白月亮倒影** 发呆。  
   - 小鸡问蛋会不会被水怪吃。  
   - 悲伤蛙头也没回，幽怨叹息：「看吧……那颗完美的、白色的圆……最终都会沉入冰冷幽暗的深渊，被这片死水吞噬……无一幸免……」  
   - 小鸡魂飞魄散，带真蛋逃跑，**深信池塘有吃白球的水怪**。

2. **「腐朽浮木」坐垫（T-2 下午）**  
   - 老鼠被黄狗吓到，薄荷鱼掉池塘边。  
   - 悲伤蛙见这散发刺鼻气味的绿物，当作「散发腐朽气息的烂木头」，拽来垫在胖臀下——契合千疮百孔心境。

3. **精神连环重创（T-1 深夜）**  
   - **第一重（枯竭）**：农场主来池塘 **舀走一桶水**（给孵化箱加湿等）；蛙见水位下降，绝望：生命之源枯竭。  
   - **第二重（破碎）**：刚经历被夺软垫的 **黑猫** 带怒路过池塘；见蛙仍对重新凝聚的月亮倒影叹气，火起。  
   - **黑猫**冲水边对水中「白色圆球（月影）」利爪狂拍，水花四溅，咬牙：「闭嘴吧蠢货！虚无的光斑！人类的爱一样全是假的！碎了就碎了！」  
   - **盲区**：悲伤蛙吓缩成一团，发福身体 **严丝合缝压住** 绿色垫子；狂怒黑猫 **未闻** 蛙臀下薄荷味。  
   - 蛙眼见精神寄托被击碎，抑郁加深。

## 交互与台词

### 史诗级跨服聊天（前期高能误导）

- **触发**：开局被小鸡骗来池塘查「水怪」；玩家问昨晚是否见「圆圆白球遇害」。  
- **悲伤蛙**（悲痛长叹）：「是的，我亲眼目睹了那场连环的悲剧。先是那个高大的两脚兽（主人），他带着冰冷的容器，**抽干了我们的生命之源**……」  
- **悲伤蛙**（发抖）：「但这还不是最可怕的！就在生命枯竭之时，那个**邪恶的黑色恶魔**降临了！它伸出残忍的利爪，就在我面前，把那个完美的、白色的圆，**生生拍成碎片**……彻底消失了。一切都是虚无。」  
- **玩家体验**：易拼成「主人抽水 + 黑色动物（乌鸦/黑猫）拍碎蛋」的惊悚假推理，加深对黑色动物怀疑。

### 终局跑腿（交出薄荷鱼）

- **触发**：终盘从老鼠重金情报得知猫玩具在青蛙处；回池塘索要绿垫。  
- **悲伤蛙**（缓缓挪臀，推薄荷鱼，叹气）：「你想要我身下这块长满青苔的死寂浮木？拿去吧。它散发着令人迷幻的腐朽气味，就像生命流逝一样让人头晕目眩……既然你喜欢收集虚无，那就给你吧。」  
- **系统**：获得 **【沾着青蛙味儿的极品猫薄荷鱼】**（命名以 UI 为准）。
