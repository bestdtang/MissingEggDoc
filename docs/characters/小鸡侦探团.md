# 小鸡侦探团

## 人设

- **视觉**：3～4 只毛茸茸半大幼鸡。为彰显「侦探」身份：头上可能顶落叶假装侦探帽，脸上戴废纸壳挖洞的粗糙「墨镜」。走路喜欢首尾相连、鬼鬼祟祟排成一列。
- **性格**：中二病、菜又爱玩、极其自信的「撒谎精」；内心慌得一批。
- **动机**：以为是自己害了还没破壳的弟弟/妹妹，极度愧疚。想靠「假情报」忽悠玩家去打败「怪物」，掩盖自己的「犯罪记录」并顺便救回弟弟。

## 时间线（事实链）

1. **偷龙转凤（T-3 下午）**  
   - 小鸡觉得窝里未破壳的弟弟/妹妹太无聊，想带出去踢足球。  
   - 趁母鸡 **淑芬** 去做午休沙浴空档，找来圆石涂白漆，认真画上与真蛋一致的 **爱心标志**，塞进鸡窝伪装；**真蛋**被偷出。

2. **连环惊吓与跨服聊天（T-3 傍晚）**  
   - **老鼠**（恐吓乐子）：推沾泥真蛋路过谷仓，被老鼠看见；老鼠为看乐子吓唬：沾泥的圆东西、泥土芬芳，今晚会被池塘「沼泽水怪」抓走吃掉等。  
   - **悲伤蛙**（暴击误读）：小鸡吓到池塘边洗泥；吵醒荷叶上叹气的青蛙。青蛙盯着水中 **月亮倒影**，幽怨指水：完美的白圆终将沉入冰冷幽暗深渊……小鸡把水中倒影误当成「一颗圆圆的蛋」，深信水里有吃白球的怪物。

3. **毁尸灭迹与真假分离**  
   - 小鸡连滚带爬把**洗净的真蛋**推回鸡窝。  
   - **画爱心的石头假蛋**慌乱中扔在鸡舍外草丛（后因子午线反光被 **乌鸦** 叼回屋顶巢）。

4. **昨晚与今早（侦探团成立）**  
   - **昨晚**：真蛋被主人拿走（人工孵化，卧室保温箱）；小鸡不知。  
   - **今早**：淑芬发现蛋没了崩溃大哭；小鸡醒来以为天塌——老鼠与青蛙说的成真，水怪按味道抓走弟弟，全因自己带蛋出去玩。  
   - 为少挨骂、将功补过，立刻戴纸板墨镜，化身「小鸡侦探团」，**主动带偏**玩家调查节奏。

## 交互与台词

### 阶段一：满嘴跑火车的目击证人（误导池塘）

- **位置/时机**：开局，鸡舍附近，围成一圈鬼祟开会。  
- **玩家体验**：高概率被骗，直奔池塘 → 遭遇满嘴「深渊」的悲伤蛙，一头雾水。  
- **领头小鸡**（推纸板墨镜）：「咳咳，同行你好。关于这起失踪案，我们**暗影侦探团**已经掌握了决定性的线索！」  
- **跟班小鸡 A**：「没错！我发誓，我看到一只**长着三个脑袋、浑身绿色的水怪**，把蛋偷去了池塘那边！」

### 阶段二：谎言被戳破的嫌疑犯

- **位置/时机**：玩家持【假蛋】回鸡舍附近对质。  
- **玩家体验**：恍然大悟——追半天的「水怪」与「被乌鸦偷走的蛋」大段为乌龙链。  
- **领头小鸡**（大哭）：「呜呜呜我招了！那是我们**前天画的假蛋**！我们只是想带弟弟去踢球，但我们发誓，**那天晚上我们就把真蛋放回妈妈的窝里了**！」  
- **跟班小鸡 A**：「这个石头假蛋我们前天就扔在草丛里了，谁知道怎么会到屋顶！一定是那只**黑心乌鸦**捡走的！」
- **跟班小鸡 B**（突然不哭了，气鼓鼓）：「还有水怪！那俩破毛球根本就是瞎编！前天我们推球路过谷仓的时候，他们躲在墙缝里偷笑——现在想想他们说话从头到尾都在憋笑！」
- **领头小鸡**（小声，气短）：「呜呜……还说什么"沼泽水怪"……我们这农场根本就没有沼泽……」

> **设计意图**：此处为**水怪红鲱鱼的免费回收主路径**。小鸡作为水怪谣言的直接受害者道破"是老鼠瞎编"，既完成红鲱鱼回收（不依赖老鼠付费二档情报），又给小鸡添一份"我们也被骗了"的可怜，柔化罪犯感。配合淑芬阶段 1"关于老鼠"的旁证，玩家在不花奶酪的情况下也能完整识破水怪谎言。

### 阶段三：第二章开场——鸡舍门口（愧疚中的待命）

- **位置**：鸡舍门口，第二章全程可交互。
- **外观变化**：纸板墨镜歪在喙尖上，不再成列排队，三两只蹲在门口各自发呆，偶尔瞄一眼鸡舍里还在哭的淑芬。平日嚣张的气势彻底没了。
- **玩家路过/靠近时**：
  - **跟班小鸡 B**（低声）：「大侦探来了……」
  - **跟班小鸡 C**（立刻缩脖子，像要消失）：「别被看见别被看见——」
  - **领头小鸡**（扯了 C 一把）：「你给我收敛一点。」（对玩家，勉强抬头，用翅膀推了推已经歪到鼻尖的墨镜）「……有事？」

- **玩家询问大黄解酒的事**：
  - **跟班小鸡 B**（一下跳起来）：「大黄叔叔？！他鼻子还没好？！」
  - **领头小鸡**（压着声音，语气里有一丝藏不住的振奋）：「……上次大黄叔叔宿醉，是我们帮他弄好的。鸡舍里水槽边上有一桶谷物泡水——他就好这口，喝了就能恢复。」
  - **跟班小鸡 C**（悄悄扯领头小鸡的羽毛，小声但有点期待）：「……我们这次能帮上忙？」
  - **领头小鸡**（轻咳，假装淡定）：「拿去吧。算是……把欠大黄叔叔的还回来一点。」

- **道具获取**：玩家与鸡舍内**水槽旁的谷物泡水桶**交互，获得道具【谷物泡水】。

- **设计意图**：小鸡的信息在玩家主动询问时才给出，不主动出击。领头小鸡维持着可怜的小小尊严（「我们没主动凑上去」），但本质上是愧疚驱动的自我救赎；**跟班小鸡 C 的期待感**是人物弧光的点睛——他们需要一次机会证明自己不是只会惹麻烦的混蛋。
