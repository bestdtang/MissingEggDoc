# 小鸡侦探团

## 人设

- **视觉**：**4 只**毛茸茸半大幼鸡（**阿满**、**米粒**、**豆豆**、**瓜子**）；头顶落叶假装侦探帽，脸戴废纸壳挖洞的粗糙"墨镜"；走路首尾相连、鬼鬼祟祟排队。
- **性格**：中二、菜又爱玩、极其自信的撒谎精；内心慌得一批。
- **动机**：以为是自己害了还没破壳的弟弟/妹妹，极度愧疚——想靠假情报忽悠玩家去打败"怪物"，掩盖犯罪记录顺便救回弟弟。

## 时间线（事实链）

1. **T-3 下午 · 偷龙转凤**：嫌窝里弟弟太无聊想带出去踢球；趁淑芬午休沙浴，找天然白色圆石，用短木炭画上爱心和鬼脸塞进鸡窝，把**真蛋**偷出。
2. **T-3 下午 · 连环惊吓**：
   - **老鼠恐吓**：推沾泥真蛋路过谷仓，被老鼠瞎编"沼泽水怪"吓唬。
   - **悲伤蛙暴击**：吓到池塘边洗泥时吵醒青蛙；青蛙对**水中月亮倒影**幽怨叹"完美的宝珠终将沉入冰冷深渊" → 小鸡误读为"水里有怪物吃蛋"。
3. **T-3 下午末 · 真假分离**：淑芬回窝摸到石头失控、院内狂奔尖叫；小鸡趁淑芬跑出去的空档，把洗净真蛋从鸡舍侧口推回鸡窝；**画爱心和鬼脸的白石头假蛋慌乱扔在鸡舍外草丛**（后被乌鸦叼回屋顶巢）。淑芬很快跑回，蛋已换回，以为自己的体温救了孩儿。
4. **T-1 ~ T0**：昨晚真蛋被主人取走（小鸡不知）→ 今早淑芬大哭，小鸡以为"老鼠青蛙的话成真，水怪按味道抓走弟弟" → 戴墨镜化身侦探团，主动带偏调查。

## 第一章 · 阶段一 · 默认：嚣张的暗影侦探团

**位置**：鸡舍附近，围圈鬼祟开会，不让玩家轻易插进圈子。

**默认口径（无线索锁）**：
- 阿满："咳咳，同行你好。这起失踪案，**暗影侦探团**已经掌握了决定性线索——但对外保密。专业人士的规矩，懂吧？"
- 米粒、豆豆接茬张望，紧张但嘴硬。

**设计要点**：水怪误导**不再是默认台词**——避免开局直接被推往池塘。玩家须先在鸡舍周边收集"他们在搞事"的证据才能戳破伪装。

## 第一章 · 阶段一 · 关键质询：戳穿暗影侦探团

树状 **2-B**（`ChickTraceCount>=2 && ChickStatus<3`；碎片来源：**E01** 短木炭 / **E02** 散落羽毛 / **E03** 偷听 **1-A**）。**2-E** 招供后菜单不再出现。

- **满足**：菜单新增 **「你们到底在搞什么鬼？」**
- 阿满被点名后心虚，慌乱甩出水怪说：「**长着三个脑袋、浑身绿色的水怪**！把蛋偷去池塘那边！」
- 豆豆补刀："池塘那只**抑郁的青蛙**肯定见过！"
- 阿满重新挺直找回侦探团尊严："专业团队的内部情报，破格分享给你！"
- **系统**：【水怪说】入第一页中上槽位 → 池塘悲伤蛙关键质询解锁。

**玩家体验**：戳穿伪装拿到水怪说 = 第一次主动推理；玩家已隐约觉得"水分很大"，但水怪说本身仍是去池塘的必要钥匙——荒诞侦探的标志段落。

## 第一章 · 阶段一 · 叫醒大黄小帮忙（无锁，案件外不撒谎）

> 把第二章原"鸡舍门口愧疚待命"段的叫醒帮忙前移到第一章——让小鸡在仍处于撒谎团伪装的同时，于**案件外的小事**上第一次露出"其实想帮忙"的真心。玩家若发现狗窝空着、又在谷仓边搬不动梯子，会自然来问怎么弄醒大黄。这是小鸡两段弧光中的**第一段（小赎罪）**，为阶段二的**大赎罪**（招供 + 水怪揭穿）铺垫。

- **触发**：玩家询问"大黄宿醉怎么办"或"谷仓上不去 / 梯子搬不动"。
- 豆豆 一下跳起："大黄叔叔？！他鼻子还没好？！"
- 阿满压声、专业架子下藏振奋："上次大黄叔叔宿醉是我们帮他弄好的——鸡舍水槽边那桶**老谷物泡水**他爱喝。"
- 瓜子 小声有点期待："……我们这次能帮上忙？"
- 阿满轻咳："拿去吧。算是侦探团对村里前辈的关照。"
- **笔记本记录**：玩家与鸡舍水槽旁谷物泡水桶交互，记录**【谷物泡水】**。

## 第一章 · 阶段二 · 关键质询：E10 质询 → 招供

树状 **2-E**（`E10_ViewWhiteStone`）。

- **满足**：菜单新增 **「这玩意儿，你们怎么解释？」**（锁 `E10_ViewWhiteStone`）
- 阿满墨镜啪嗒掉地大哭："呜呜呜我招了！那是我们**大前天画的假蛋**！只是想带弟弟去踢球——弟弟滚泥里弄脏了，我们正好跑去池塘洗，回来的时候妈妈已经满院子乱跑了，**我们就趁乱从侧口把真蛋放回窝里了**！"
- 米粒 甩锅："这石头我们大前天扔在草丛里了，一定是那只**黑心乌鸦**捡走的！"
- 豆豆 突然不哭、气鼓鼓："还有水怪！**那俩破毛球根本就是瞎编**！大前天我们推球路过谷仓，他们躲墙缝里偷笑——现在想想他们说话从头到尾都在憋笑！"
- 阿满气短小声："呜呜……还说什么沼泽水怪……**我们这农场根本就没有沼泽**……"
- 瓜子 小声坚定："……我们再也不偷弟弟出去踢球了。"

**关键作用**：
- **水怪红鲱鱼免费回收主路径**——不依赖老鼠付费二档情报，由小鸡（受害者视角）直接道破"是老鼠瞎编"；配合淑芬阶段 1"关于老鼠"的旁证形成双线兜底。
- **招供台词"那天下午我们就趁乱把真蛋放回妈妈的窝里了"是第一章末关键认知转折**——真蛋是**昨夜**从鸡舍消失的，主案件焦点从"乌鸦"移向"昨夜"，承接第二章。

## 阶段三：第二章 · 愧疚待命

- **外观**：纸板墨镜歪在喙尖，不再列队，阿满带米粒、豆豆、瓜子挤在门口发呆，偶尔瞄一眼还在哭的淑芬。
- **交互**：玩家**点小鸡**走入口判定进 **3-A**【轮播】（`DogStatus==4 && ChickStatus==3`；一次性）。无路过 trigger。
- **作用**：阶段三**只剩情绪线**——推进任务已在前两阶段完成；第二章承担玩家主动找小鸡时的温度细节。
