# 失眠暴躁黑猫

## 人设

- **视觉**：乌黑优雅；严重黑眼圈、毛发凌乱；T0 开局躲大橡树树冠生闷气。**可选美术彩蛋**：耳尖/背线一处**微焦毛尖**（T-2 谷仓玻璃珠聚光 / 眩光燎痕），特写镜头下可见。
- **性格**：极度傲慢、领地意识极强；自诩农场"巡视官"；洁癖与起床气重；**记仇且不手软**。
- **关系**：
  - 视农场主为"专属铲屎官"；
  - 与红顶屋墙缝**老鼠兄弟**长期汤姆杰瑞式敌对；
  - 嘲笑**大黄**跳不高、蠢忠诚；
  - 与**乌鸦**有多年积怨：巢在谷仓顶、凌晨嘎嘎炫富收藏，黑猫窗台正对谷仓，**三年来常被吵醒**（树上独白仍可利用这份烦）。
  - **本次上谷仓顶的直接导火索**：T-2 **正午前后**乌鸦在屋顶摆弄玻璃珠、碎玻璃等闪亮「财宝」，玻璃珠聚焦热点 + 碎玻璃刺眼反光扫到谷仓边/窗台下打盹的黑猫，**毛尖险燎**（美术可做**微焦痕**彩蛋）。猫怒上顶找乌鸦算账，**顺带**瞥见巢里假蛋。

## 时间线（事实链）

1. **T-2 清晨 · 旁观 + 嘲地毯**：窗台旁观乌鸦俯冲叼草丛石头 vs 大黄狂吠跳不高的闹剧；乌鸦绕大黄头顶盘旋时嚣张叫大黄「**地毯**」——黑猫旁观冷冷对大黄重复了一遍：「**地毯。**」，走开。
2. **T-2 下午 · 一次上顶：眩光纠纷 → 瞥见假蛋 → 沉默 + 失去薄荷鱼，记仇**：谷仓附近午觉，【极品薄荷鱼玩具】留在角落；被屋顶**玻璃珠聚光 / 碎玻璃反光**燎到后**只这一次**上谷仓顶找乌鸦。对峙中**顺带看清**巢里石头假蛋，认出天然白石上用木炭画的鬼脸和爱心，仍出于傲慢 + 宿怨 + 看戏心态**选择沉默**，下顶。薄荷鱼已被老鼠趁其在顶偷走（搬运中被大黄酒疯狂吠惊到、掉池塘边荷叶旁、悲伤蛙当浮木垫臀）。黑猫堵墙缝未果记仇。T0 全场误把乌鸦推上嫌疑人位时，**这步沉默**仍压在乌鸦头上。
3. **T-1 白天 · 带仇憋着**：主人窗台睡觉，偶尔去墙缝口骚扰老鼠，始终进不去，憋着。
4. **T-1 深夜 · 史诗级感情诈骗 → 上树自闭**：夜间找主人撒娇 → 卧室窗台天鹅绒软垫入睡 → 主人温柔把黑猫捧起，猫以为深夜按摩、准备发出享受呼噜 → 结果被放在冰冷木地板上、**软垫被夺**。黑猫只确认自己的垫子被拿走，没看见桌上装置、黄灯用途或任何蛋 → 史诗级感情诈骗，自尊重创 → 从猫洞挤出带满腔怒火路过池塘 → 见悲伤蛙仍盯水中重新凝聚的月影叹气，火起伸爪**对水中月影狂拍水花四溅**（被悲伤蛙误读为"黑色恶魔拍碎玉盘"）→ 咬牙"闭嘴！那是虚无的光斑！" → 爬上大橡树整夜自闭。

## 第一章：树上独白（隐藏状态 · 不可交互）

第一章全程黑猫不可交互；玩家靠近大橡树时从树冠上方传来独白音效（声音来源不明）。台词分四层、按规则播放，**伏笔句首次硬触发必播**，保证所有玩家都听过。


| 层级        | 内容                                          | 触发                                    | 数量  |
| --------- | ------------------------------------------- | ------------------------------------- | --- |
| **首次硬触发** | 伏笔句：「**傻子在炫耀，瞎子在到处问。**」                     | 第一次靠近大橡树必播                            | 1   |
| **条件解锁**  | 诱导屋顶（"屋顶那只蠢东西……守着他的宝贝……" / "鸡舍上面那个，今天又叫……"） | 玩家**已知乌鸦存在**（与大黄对话至少一次、大黄证词提及乌鸦）后入常规池 | 2   |
| **常驻常规**  | 性格展示（"你踩到我的根了" / "吵" / "凡人……无趣"）            | 始终在常规池内、等权重轮播                         | 3   |
| **彩蛋池**   | 近乎明示："最好笑的鸟……总是守着最假的宝。"                     | 每次靠近 5% 独立掷骰                          | 1   |


**伏笔句双层含义**：

- 首次听 = 傲慢黑色幽默吐槽（傻子=乌鸦炫耀；瞎子=玩家被到处误导）；
- 第二章揭穿后回放 = 他**早就看清乌鸦守的是石头**——"傻子在炫耀着假宝贝，瞎子在到处问着假线索"，他在树上看戏看得很满意。

**为什么诱导句要锁在"已知乌鸦"之后**：避免玩家在不知道乌鸦存在的情况下被树上声音莫名其妙地推向屋顶，保证剧情顺序的信息释放。

**口碑铺垫**：玩家可从老鼠闲话、大黄对话中听到"难搞黑猫"的侧写，进入第二章前对其有一定预期。

## 第二章：摇树召唤 → 树下谈判

### 现身：被晃下树

第一章末玩家与大黄共同抵达红顶屋外，先确认**正门进不去**；大黄提示另找入口。随后玩家在屋外勘察到**精美猫门 / 不明兽毛 / 食碗 / 大橡树根抓痕**等【下树线索】≥ 2 条后，主动向大黄追问「这门是给谁的？」——大黄一拍脑门想起黑猫，走到大橡树下撩拨树干、用力摇树：「**猫大爷！下来！这事得你出马！**」

黑猫被晃下树落地，毛发更乱、白眼一翻、低沉警告：「滚开！没看到本喵正因**感情欺骗**疗伤吗？那该死的两脚兽用温柔手法骗我、**抢走本喵的御用软垫**！再说，"摇树"是什么不文明手段？！」

大黄厚脸皮："不摇你不下来。"
黑猫瞥大黄一眼转向玩家："……你又是谁？"

**此刻起黑猫一直在大橡树下，第二章所有对话都在树下进行**；它蹲在那儿一副"本喵不想理人"的架子，但也并不离开——好奇心被这一摇晃出来了一点，只是绝不会先开口。

### 玩家面对的局面

黑猫掌握着唯一一条进屋路径（它的专属猫门 + 屋顶天窗），但情绪上断然拒绝带路。要让它愿意起身、带玩家去开窗，玩家必须同时撬开**两条独立的谈判入口**——一条诉诸**案情**（让黑猫意识到玩家有理由怀疑蛋在屋里，且自己的软垫可能卷入其中），一条诉诸**物权**（玩家替它把丢失的薄荷鱼找回来）。两路没有先后依赖，可以并行推进，但**两路都必须达成**：黑猫的**好奇**（"那家伙到底拿本喵的垫子做了什么？"）+ **自尊**（"被两脚兽当面骗了一手不能就这么算了"）+ **物权**（"软垫去哪了？薄荷鱼必须回来"）三重驱动力凑齐，它才肯起身。

### 案情汇报线 · 玩家汇报案情（激活顿悟）

玩家**只列事实清单、不下结论**——尤其不能说出"孵化""保温箱""软垫垫底"等揭晓前不该出现的词。黑猫昨夜只确认**软垫被主人拿走**，完全没看见过蛋，也不知道屋里发生了什么——它只把那一夜理解为"自尊被狠狠羞辱"。要让它愿意进屋看个清楚，必须靠玩家把事实一条条摆出来。

**汇报清单**：

- **【硬必 1】淑芬今早醒来发现蛋空了**——窝里没有任何打斗、外来气味或拖拽痕迹（开场强制获得）。
- **【硬必 2】大黄的鼻子刚恢复，确认蛋的气味现在还从红顶屋里飘出来。不是过去式**（大黄 ④ + 红顶屋外二次嗅觉确认必有）。
- **【已知事实】悲伤蛙曾用谜语口吻说，昨晚有高大的两脚兽带冰冷容器抽走“生命之源”**（第一章持【水怪说】质询悲伤蛙时强制获得，本质是主人提桶来池塘舀水）。

**汇报卡点**：除上述硬必两条之外，还**必须同时持有**红顶屋外勘察拿到的两条【案情汇报物证】：

- **【门旁空水桶：桶底是池塘泥沙】**——证明主人昨夜取了水后又把桶带回来；
- **【门前雨靴脚印：朝鸡窝方向、夜里踩出】**——证明主人昨夜出门去了鸡窝。

两条【案情汇报物证】**任缺其一**时，汇报推进到「你说主人昨晚出去过——有什么能证明？」处，黑猫追问后**菜单不再出现继续选项**，玩家需返回红顶屋外补全后再回来。

**两条【案情汇报物证】齐全 → 汇报完整 → 黑猫顿悟必触发**。玩家把空窝、红顶屋内仍有蛋味、门旁水桶、夜间雨靴脚印与"主人拿走软垫"并排摆上桌；黑猫第一次意识到，自己的软垫很可能不是普通失窃，而是被卷进了这颗失踪蛋的去向里：「**那家伙昨晚拿走本喵的垫子，可能不是为了气我。**」「**蛋在屋里，垫子也在屋里。很好，本喵现在要亲眼看看。**」

但黑猫**仍不知道**蛋现在在屋里什么位置、自己的软垫被拿去做了什么、那盏黄灯到底烧来干嘛——这些悬念全部保留到红顶屋揭晓。它此刻和玩家一样好奇，但还没起身。

### 薄荷鱼线 · 发现链

要撬开「**你是不是丢了什么？跟那群老鼠有关？**」这个质询选项，玩家须先拿到两个谷仓环境依据，证明自己确实注意到黑猫曾在谷仓附近停留并突发离开：

- **【谷仓午睡点】**第一章攀谷仓途中发现角落被压扁的草窝。
- **【烧焦稻草 / 皮毛】**第一章攀谷仓途中发现，被乌鸦屋顶玻璃珠聚光 / 碎玻璃眩光燎到的残留。

两条环境依据均满足 → 玩家撬开质询「谷仓角落那里……那个草窝，是你的吧」→ 黑猫愣一下尾巴尖弹动，视线短暂飘向红顶屋墙缝；玩家只顺势问「跟老鼠有关系吗？」并提出"我帮你——你帮我开屋子"的交易 → 黑猫沉默片刻："……去找老鼠。找回来再说。"

→ 薄荷鱼线开启后有两条取回路径：**自主路径**为玩家凭【悲伤蛙身下绿垫有奇异草本甜气】自行拼合并到池塘索要；**老鼠路径**为玩家前往红顶屋墙缝，老鼠先谎称东西在手里，收 **8 块奶酪碎**后供出"最后掉在池塘边一带"，并看在大单份上开放新一批情报。两条路径最终都到悲伤蛙处触发三轮对暗号；若付费路径失败，可回老鼠处免费获得【青蛙弱点：排第七】作为兜底。玩家从悲伤蛙处取回**【沾蛙味的极品猫薄荷鱼】** → 返回大橡树下交黑猫 → 黑猫人设微崩、猛吸、呼噜半秒、迅速收住："喵……就是这个——不对！一条臭鱼买不走本喵！"——薄荷鱼标志着【薄荷鱼线】完成，黑猫**仍蹲在树下不动**，要等【案情汇报线】也完成才肯起身。

> 老鼠收下 8 块薄荷鱼情报费时**解锁二档后组 6 条深夜情报**（3 真 + 3 bullshit · 盲盒抽取 · 50% 命中）。后组情报作为回溯与 replay 补充，不承担【案情汇报线】推进。

### 汇合 · 起身去开窗（好奇 + 自尊 + 物权三重驱动）

两路均满足后，大黄从旁补一脚撩拨："猫大爷，你昨晚可是被那两脚兽**当面骗了一手**啊！整个农场都看你睡得呼呼的、还以为是要按摩——结果一觉醒来软垫没了、自尊扫地！**这要是传出去，今年冬天你睡稻草堆里我可不让你蹭我狗窝。**"

黑猫被这句撩到尾巴尖弹动一下，找到下台阶的体面理由——它从树下蹲位**站起身**，语气冷淡但少了怒气：

- 不是为发疯母鸡，也不是听蠢狗使唤；
- **就要亲眼进屋看个清楚**——那家伙昨晚到底拿本喵的软垫做了什么、垫子去了哪儿；
- 「至于你说的那颗破蛋——既然它现在多半就在屋里，**顺便看一眼也不亏**。」（傲慢人设的标准签名——不会承认是为蛋起身，但好奇心已被点燃。）
- 「**本喵有专属钥匙。你要进去——天窗。本喵从里面推开。你不许走本喵的猫门。**」

→ 黑猫用专属钥匙打开**专为它凿刻的猫门**（紧闭大门下方，主人手工边缘），熟练钻进屋，从内**啪嗒推开屋顶天窗**——这声响是玩家攻顶的目标音效。

### 进屋方式与玩家入口

- **黑猫**：用专属钥匙开猫门进屋，从内推开天窗。
- **玩家**：小火人侦探**钻不过猫门** → 通过红顶屋外的**平台跳跃挑战**登上屋顶面（屋檐 / 雨水桶 / 旁侧木箱堆叠等组合，具体卡点待美术与关卡定稿） → 从被推开的天窗**下降进入卧室**。
- **节奏意义**：这次跳跃挑战与第一章谷仓屋顶形成**首尾呼应**（两次屋顶 = 两次推进节点）。详见 `07-7.2` 第二章流程与 `08-3D场景` 红顶屋地标说明。

## 揭穿黑幕（可选高光 · 双触发）

黑猫"明知石头却装作不知 → 让乌鸦背锅"是全场最有意思的反派式人设高光，不应只能通过黑猫主动自爆获得。本段以**双触发**让推理流玩家"自己问出来"、休闲玩家也能兜底触达。

- **早触发（推理奖励路径）**：
  - **前置条件**（同时满足）：
    1. 玩家已在第一章谷仓顶确认"乌鸦守的是画了鬼脸和爱心的白石头"
    2. 玩家已知"黑猫 T-2 下午上过谷仓顶"——以下任一满足即可：
       - 乌鸦对峙台词（第一章谷仓顶必播，等同全员获得）
       - 老鼠一档 #10
  - **触发时机**：薄荷鱼归还后的案情汇报阶段（黑猫已开口倾听、记忆触发段落之后、起身去开窗之前），对话菜单新增可选项「**你上谷仓屋顶那次……看到什么了？**」
  - **设计意图**：乌鸦对峙台词已保证几乎全部玩家持有条件 2，早触发实际对大多数玩家开放；推理钩子由玩家主动拼接，追问由玩家主动发起。
- **晚触发（兜底路径）**：
  - **触发时机**：若玩家没买老鼠一档 #10、或在汇报阶段错过早触发选项，可在**进卧室见保温箱与软垫之后**回大橡树下，对话菜单出现同款追问选项。
  - 这条路径保留"被动告知"的喜剧反讽味道，照顾不抽老鼠情报或快速通关的玩家。
- **追问对白要点（双触发共用）**：
  - 玩家："你之前上屋顶的时候，是不是已经知道那是块石头？"
  - 黑猫（停顿，舔爪）："嗯。白石头，木炭鬼脸，旁边还画了颗歪爱心。一眼就看出来了。"
  - 玩家："你为什么不说？！"
  - 黑猫（极其淡漠）："那只乌鸦每天早上四点开始嘎嘎嘎已经三年了。让侦探多转几圈……本喵觉得相当公平。"
- **早触发专属呼应（可选加分）**：黑猫多接一句呼应第一章树上的伏笔独白——"我在树上不是说过吗？傻子在炫耀，瞎子在到处问。你以为我是在骂你？"
  - 玩家体验：拍腿——"原来第一章那句吐槽就是答案。"

## 二周目结尾对话（释怀盘 · 一次性 · 待补）

漫画演绎合本后玩家被放回红顶屋大门前进入二周目；**首次靠近黑猫**（大橡树下或红顶屋窗台上，常驻位置策划定稿）时一次性触发。

**风味定位**：傲慢式接管 + 嘴硬底软——软垫归位、形成"这颗破蛋以后归本喵守着"的接管态度，口头仍冷淡（"本喵勉强同意把那间屋子继续租给那家伙住"）；可保留对乌鸦三年宿仇的最终结案吐槽，呼应"傻子在炫耀，瞎子在到处问"伏笔句的最后回响。具体台词待编剧定稿。
