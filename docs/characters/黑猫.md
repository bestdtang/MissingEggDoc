# 失眠暴躁黑猫

## 人设

- **视觉**：乌黑优雅；严重黑眼圈、毛发凌乱；T0 开局躲大橡树树冠生闷气。**可选美术彩蛋**：耳尖/背线一处**微焦毛尖**（T-2 谷仓玻璃珠聚光 / 眩光燎痕），特写镜头下可见。
- **性格**：极度傲慢、领地意识极强；自诩农场"巡视官"；洁癖与起床气重；**记仇且不手软**。
- **关系**：
  - 视农场主为"专属铲屎官"；
  - 与红顶屋墙缝**老鼠兄弟**长期汤姆杰瑞式敌对；
  - 嘲笑**大黄**跳不高、蠢忠诚；
  - 与**乌鸦**有多年积怨：巢在谷仓顶、凌晨嘎嘎炫富收藏，黑猫窗台正对谷仓，**三年来常被吵醒**（树上独白仍可利用这份烦）。
  - **本次上谷仓顶的直接导火索**：T-2 **正午前后**乌鸦在屋顶摆弄玻璃珠、碎玻璃等闪亮「财宝」，玻璃珠聚焦热点 + 碎玻璃刺眼反光扫到谷仓边/窗台下打盹的黑猫，**毛尖险燎**（美术可做**微焦痕**彩蛋）。猫怒上顶找乌鸦算账，**顺带**瞥见巢里假蛋。

## 时间线（事实链）

1. **T-2 清晨 · 旁观 + 嘲地毯**：窗台旁观乌鸦俯冲叼草丛石头 vs 大黄狂吠跳不高的闹剧；乌鸦绕大黄头顶盘旋时嚣张叫大黄「**地毯**」——黑猫旁观冷冷对大黄重复了一遍：「**地毯。**」，走开。
2. **T-2 下午 · 一次上顶：眩光纠纷 → 瞥见假蛋 → 沉默 + 失去薄荷鱼，记仇**：谷仓附近午觉，【极品薄荷鱼玩具】留在角落；被屋顶**玻璃珠聚光 / 碎玻璃反光**燎到后**只这一次**上谷仓顶找乌鸦。对峙中**顺带看清**巢里石头假蛋，认出天然白石上用木炭画的鬼脸和爱心，仍出于傲慢 + 宿怨 + 看戏心态**选择沉默**，下顶。薄荷鱼已被老鼠趁其在顶偷走（搬运中被大黄酒疯狂吠惊到、掉池塘边荷叶旁、悲伤蛙当浮木垫臀）。黑猫堵墙缝未果记仇。T0 全场误把乌鸦推上嫌疑人位时，**这步沉默**仍压在乌鸦头上。
3. **T-1 白天 · 带仇憋着**：主人窗台睡觉，偶尔去墙缝口骚扰老鼠，始终进不去，憋着。
4. **T-1 深夜 · 史诗级感情诈骗 → 上树自闭**：夜间找主人撒娇 → 卧室窗台天鹅绒软垫入睡 → 主人温柔把黑猫捧起，猫以为深夜按摩、准备发出享受呼噜 → 结果被放在冰冷木地板上、**软垫被夺**。黑猫只确认自己的垫子被拿走，没看见桌上装置、黄灯用途或任何蛋 → 史诗级感情诈骗，自尊重创 → 从猫洞挤出带满腔怒火路过池塘 → 见悲伤蛙仍盯水中重新凝聚的月影叹气，火起伸爪**对水中月影狂拍水花四溅**（被悲伤蛙误读为"黑色恶魔拍碎玉盘"）→ 咬牙"闭嘴！那是虚无的光斑！" → 爬上大橡树整夜自闭。

## 第一章：树上独白（隐藏状态 · 不可交互）

第一章全程黑猫不可交互；玩家靠近大橡树时从树冠上方传来独白音效（声音来源不明）。台词分四层、按规则播放，**伏笔句首次硬触发必播**，保证所有玩家都听过。


| 层级        | 内容                                          | 触发                                    | 数量  |
| --------- | ------------------------------------------- | ------------------------------------- | --- |
| **首次硬触发** | 伏笔句：「**傻子在炫耀，瞎子在到处问。**」                     | 第一次靠近大橡树必播                            | 1   |
| **条件解锁**  | 诱导屋顶（"屋顶那只蠢东西……守着他的宝贝……" / "鸡舍上面那个，今天又叫……"） | 玩家**已知乌鸦存在**（与大黄对话至少一次、大黄证词提及乌鸦）后入常规池 | 2   |
| **常驻常规**  | 性格展示（"你踩到我的根了" / "吵" / "凡人……无趣"）            | 始终在常规池内、等权重轮播                         | 3   |


**伏笔句双层含义**：

- 首次听 = 傲慢黑色幽默吐槽（傻子=乌鸦炫耀；瞎子=玩家被到处误导）；
- 第二章揭穿后回放 = 他**早就看清乌鸦守的是石头**——"傻子在炫耀着假宝贝，瞎子在到处问着假线索"，他在树上看戏看得很满意。

**为什么诱导句要锁在"已知乌鸦"之后**：避免玩家在不知道乌鸦存在的情况下被树上声音莫名其妙地推向屋顶，保证剧情顺序的信息释放。

**口碑铺垫**：玩家可从老鼠闲话、大黄对话中听到"难搞黑猫"的侧写，进入第二章前对其有一定预期。

## 第二章：摇树召唤 → 树下谈判

> **对话实施**：节点编号、入口路由、菜单锁与变量写入以 [黑猫-对话脚本-树状](./黑猫-对话脚本-树状.md) 为准；本节写世界事实与谈判逻辑，不重复逐句台词。

### 点黑猫 · 入口路由（`Dog_BlackCatSummoned` 后）

| 优先级 | 条件 | 进入 |
|--------|------|------|
| 1 | `NGPlus` | **NGPlus 回访**【轮播】 |
| 2 | `Comic_Revealed && !BlackCat_StoneRevealShown` | **3-B** 晚触发揭穿 |
| 3 | `CaseLineDone && MintFishLineDone && !BlackCat_Entered` | **2-E** 汇合开窗 |
| 4 | `Dog_BlackCatSummoned && !BlackCat_Entered` | **2-hub** 谈判菜单 |
| 5 | `BlackCat_Entered` | 不可点（已进屋） |

**2-A** 只在大黄 **2-B** 摇树演出里播，玩家不能单独点进。

### 现身：被晃下树（**2-A**）

第一章末玩家与大黄共同抵达红顶屋外，先确认**正门进不去**（**E13**）；大黄 **2-hub** 催查后，玩家须持【下树线索】≥ 2 条（**E14–E16 / E28**，不含 E13），回大黄 **2-hub** 选「红屋顶里还住着别的动物？」→ 大黄 **2-B** 摇树。

黑猫被晃下树落地，毛发更乱、白眼一翻、低沉警告：「滚开！没看到本喵正因**感情欺骗**疗伤吗？那该死的两脚兽用温柔手法骗我、**抢走本喵的御用软垫**！再说，"摇树"是什么不文明手段？！」

大黄厚脸皮："不摇你不下来。"
黑猫瞥大黄一眼转向玩家："……你又是谁？"

**此刻起黑猫一直在大橡树下，第二章主交互在树下黑猫 NPC**；它蹲在那儿一副"本喵不想理人"的架子，但也并不离开——好奇心被这一摇晃出来了一点，只是绝不会先开口。播毕进 **2-hub**【回访】+【菜单】。

### 树下谈判 hub（**2-hub**）

**2-hub**【菜单】项（实施锁见树状）：

| 菜单 | 条件概要 | 节点 |
|------|----------|------|
| 「蛋的线索指向红顶屋里面。」 | `!CaseLineStarted` | **2-B** 开场必播（只一次） |
| 「我还有新的线索要汇报。」 | `Started && !Done` | **2-B-hub** 物证并列菜单 |
| 「谷仓角落那里……那个草窝，是你的吧。」 | `E07`+`E08`，未开薄荷鱼线 | **2-C** |
| 「找回来了。」 | `MintFishPending`+`MintFish_Obtained` | **2-D** 交还 |
| 「你上谷仓屋顶那次——」 | `E10`+`MintFishLineDone`，未揭穿 | **3-A** 早触发 |
| 「先不打扰你了。」 | — | 对话结束 |

双路齐后再点黑猫**不进 hub**，路由直进 **2-E**。

### 玩家面对的局面

黑猫掌握着唯一一条进屋路径（它的专属猫门 + 红顶屋二层窗），但情绪上断然拒绝带路。要让它愿意起身、带玩家去开窗，玩家必须同时撬开**两条独立的谈判入口**——一条诉诸**案情**（让黑猫意识到玩家有理由怀疑蛋在屋里，且自己的软垫可能卷入其中），一条诉诸**物权**（玩家替它把丢失的薄荷鱼找回来）。两路没有先后依赖，可以并行推进，但**两路都必须达成**：黑猫的**好奇**（"那家伙到底拿本喵的垫子做了什么？"）+ **自尊**（"被两脚兽当面骗了一手不能就这么算了"）+ **物权**（"软垫去哪了？薄荷鱼必须回来"）三重驱动力凑齐，它才肯起身。

### 案情汇报线 · 玩家汇报案情（**2-B** → **2-B-hub** → **2-B-C**）

玩家**只列事实清单、不下结论**——尤其不能说出"孵化""保温箱""软垫垫底"等揭晓前不该出现的词。黑猫昨夜只确认**软垫被主人拿走**，完全没看见过蛋，也不知道屋里发生了什么——它只把那一夜理解为"自尊被狠狠羞辱"。要让它愿意进屋看个清楚，必须靠玩家把事实一条条摆出来。

**2-B 开场必播**（`!BlackCat_CaseLineStarted`，只一次）合并陈述三条必经事实——玩家走到这里时主线已给齐，不必拆菜单：

- **淑芬今早醒来发现蛋空了**——窝里没有任何打斗、外来气味或拖拽痕迹。
- **大黄的鼻子刚恢复，确认蛋的气味现在还从红顶屋里飘出来**（不是过去式）。
- **悲伤蛙曾用谜语口吻说，昨晚有高大的两脚兽带冰冷容器抽走“生命之源”**（第一章持【水怪说】质询悲伤蛙时强制获得）。

播毕写 `BlackCat_CaseLineStarted`。若此时**两项物证皆无**（`!E17 && !E18`）→ 卡点对话结束；若持 **E17** 或 **E18** → 进 **2-B-hub**。

**2-B-hub** 并列菜单（已播项隐藏）：

- **【门旁空水桶：桶底是池塘泥沙】**（**E17** → **2-B-A**）
- **【门前雨靴脚印：朝鸡窝方向、夜里踩出】**（**E18** → **2-B-B**）
- 「先说到这儿。」（尚有未播物证时 → **2-B-D** 卡点离场）

两条物证**各在菜单里播过一次** → **2-B-C** 顿悟 → 写 `BlackCat_CaseLineDone`。补证**不再进 2-B**，只从 **2-hub**「我还有新的线索要汇报。」进 **2-B-hub**。

但黑猫**仍不知道**蛋现在在屋里什么位置、自己的软垫被拿去做了什么、那盏黄灯到底烧来干嘛——这些悬念全部保留到红顶屋揭晓。它此刻和玩家一样好奇，但还没起身。

### 薄荷鱼线 · 发现链（**2-C** → **2-D**）

要撬开 **2-hub** 菜单「谷仓角落那里……那个草窝，是你的吧」，玩家须先拿到两个谷仓环境依据（**E07** 午睡点 + **E08** 烧焦稻草 / 皮毛），证明自己确实注意到黑猫曾在谷仓附近停留并突发离开。

→ **2-C** 打开薄荷鱼线，写 `BlackCat_MintFishPending`；黑猫派活时**不说物件名称**，玩家从老鼠 / 蛙处才知道是薄荷鱼。

取回路径见老鼠 / 蛙线脚本；取得后写 `MintFish_Obtained`。**2-hub**「找回来了。」→ **2-D** 交还 → 写 `BlackCat_MintFishLineDone`。黑猫人设微崩、猛吸、呼噜半秒、迅速收住——**仍蹲在树下不动**，要等案情线也完成才肯起身。

> 老鼠收下 8 块薄荷鱼情报费时**解锁二档后组 6 条深夜情报**（3 真 + 3 bullshit · 盲盒抽取 · 50% 命中）。后组情报作为回溯与 replay 补充，不承担【案情汇报线】推进。

### 汇合 · 起身去开窗（**2-E** · 好奇 + 自尊 + 物权三重驱动）

`BlackCat_CaseLineDone && BlackCat_MintFishLineDone && !BlackCat_Entered` 时，再点黑猫走路由直进 **2-E**（或 **2-B-C** / **2-D** 出口连播）。大黄激将正文写在黑猫 **2-E** 节点内（大黄样章 **2-D** 仅作跳转占位，不重复台词）。

两路均满足后，大黄从旁补一脚撩拨："猫大爷，你昨晚可是被那两脚兽**当面骗了一手**啊！……这要是传出去，今年冬天你睡稻草堆里我可不让你蹭我狗窝。"

黑猫被撩到尾巴尖弹动，找到下台阶的体面理由——它从树下蹲位**站起身**，语气冷淡但少了怒气：

- 不是为发疯母鸡，也不是听蠢狗使唤；
- **就要亲眼进屋看个清楚**——那家伙昨晚到底拿本喵的软垫做了什么、垫子去了哪儿；
- 「至于你说的那颗破蛋——既然它现在多半就在屋里，**顺便看一眼也不亏**。」
- 「**本喵有专属钥匙。你要进去——爬二层窗。本喵从里面推开。你不许走本喵的猫门。**」

→ 黑猫用专属钥匙打开**专为它凿刻的猫门**，熟练钻进屋，从内推开**红顶屋二层窗**——啪嗒一声是玩家攻顶的目标音效。写 `BlackCat_Entered=true`；再点黑猫不可交互。再点大黄走 **2-E** 攻顶前加油（一次性）。

### 攻顶途中屋内放话（**2-F** · 氛围）

黑猫开窗后，玩家沿红顶屋外平台跳跃路线攻顶。**E37** 起跳后第一段、**E38** 接近二层窗之前各远程触发 **2-F** 一句放话；玩家跳跃中无台词、无菜单，不入笔记本。逐句见树状 **2-F**。

### 进屋方式与玩家入口

- **黑猫**：用专属钥匙开猫门进屋，从内推开二层窗。
- **玩家**：小火人侦探**钻不过猫门** → 通过红顶屋外的**平台跳跃挑战**登上屋顶面（屋檐 / 雨水桶 / 旁侧木箱堆叠等组合，具体卡点待美术与关卡定稿） → 从被推开的二层窗**下降进入卧室**。
- **节奏意义**：这次跳跃挑战与第一章谷仓屋顶形成**首尾呼应**（两次屋顶 = 两次推进节点）。详见 `07-7.2` 第二章流程与 `08-3D场景` 红顶屋地标说明。

## 揭穿黑幕（可选高光 · **3-A** / **3-B** 双触发）

黑猫"明知石头却装作不知 → 让乌鸦背锅"是全场最有意思的反派式人设高光，不应只能通过黑猫主动自爆获得。

- **早触发 · 3-A**（推理奖励路径）：
  - **菜单入口**：**2-hub**「你上谷仓屋顶那次——」；`!BlackCat_Entered && !BlackCat_StoneRevealShown`。
  - **读取条件**：`E10_ViewWhiteStone`（第一章谷仓顶确认白石头假蛋）+ `BlackCat_MintFishLineDone`。
  - **设计意图**：薄荷鱼归还后、起身开窗前，推理流玩家主动拼接「黑猫 T-2 下午上过谷仓顶」与假蛋事实，自己问出来。
- **晚触发 · 3-B**（兜底路径）：
  - **路由入口**：漫画收束后点黑猫；`Comic_Revealed && !BlackCat_StoneRevealShown`。**不进 2-hub**；播毕对话结束。
  - 照顾未在 hub 点开 **3-A**、或未抽老鼠一档 #10 的玩家。
- **追问对白要点（双触发共用）**：
  - 玩家："你之前上屋顶的时候，是不是已经知道那是块石头？"
  - 黑猫（停顿，舔爪）："嗯。白石头，木炭鬼脸，旁边还画了颗歪爱心。一眼就看出来了。"
  - 玩家："你为什么不说？！"
  - 黑猫（极其淡漠）："那只乌鸦每天早上四点开始嘎嘎嘎已经三年了。让侦探多转几圈……本喵觉得相当公平。"
- **早触发专属呼应（3-A 可选加分）**：黑猫多接一句呼应第一章树上的伏笔独白——"我在树上不是说过吗？傻子在炫耀，瞎子在到处问。你以为我是在骂你？"

二者共用 `BlackCat_StoneRevealShown`；播过其一后不再出现。

## 二周目结尾对话（释怀盘 · **NGPlus** · 一次性）

漫画演绎合本后玩家被放回红顶屋大门前进入二周目；**首次点树下黑猫 NPC**（非常驻「点树」）时一次性触发 **NGPlus 回访**【轮播】。

**风味定位**：傲慢式接管 + 嘴硬底软——软垫归位、形成"这颗破蛋以后归本喵守着"的接管态度，口头仍冷淡（"本喵勉强同意把那间屋子继续租给那家伙住"）；可保留对乌鸦三年宿仇的最终结案吐槽，呼应"傻子在炫耀，瞎子在到处问"伏笔句的最后回响。具体台词见树状 **NGPlus 回访**。
