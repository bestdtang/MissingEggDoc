# 农场主（主人）

## 人设

- **可见性**：全程不可直接交互；通过**动物侧叙、环境痕迹、室内物证**构成形象——是本案真相的核心行动者，以「缺席的温柔」推动整条故事。
- **性格**：心思细腻的老式农人；不爱打扰动物的自然规律，倾向在旁观察、确认「必须介入」后才出手；对黑猫有远超普通的特殊感情。
- **核心矛盾**：心疼淑芬连日焦虑，却不愿打断母鸡自然孵化的本能——两难之下一直观望，直到寒夜骤降与大黄的鼾声同时击中他，才当机立断。

---

## 时间线（事实链，与 `10-NPC主时间线` 对照）

### T-3 下午：目击失控，启动「备而不用」

- 目击淑芬院中狂奔、深蹲、大叫，判断「母鸡孵化焦虑严重失常」。
- **第一反应并非立即取蛋**：翻出《母鸡产后护理与孵化指南》细读，心里的原则是「先等等——她要是能自己稳下来，就不要插手」。
- 同步悄悄开始准备孵化设备（保温箱框架、黄色加温灯、水桶）作为应急储备，**备而不用**。

### T-3 深夜：淑芬自然回稳，主人暂缓行动

- 见淑芬傍晚已恢复正常孵化，主人松口气，将设备靠墙备着，今夜按兵不动。
- 内心逻辑：「她能自己来，就别惊扰。」

### T-2：观望为主，推进准备

- 偶尔从窗口看淑芬状态——表面稳定（他不知道小鸡已还了真蛋），判断无需立即介入。
- 悄悄对孵化设备做细节完善（温控调试、黄灯测试、箱内铺垫材料）。

### T-1 白天：万事俱备，等待信号

- 孵化设备全部就绪，只差一个「必须今夜动手」的明确信号。
- 持续观察淑芬：连续数日高度紧张孵蛋，疲惫加剧，体力明显透支。
- 心里预判：「如果今夜气温再降，就必须动手了。」

### T-1 深夜：倒春寒 + 大黄的鼾声——行动

- **大黄的酒嗝鼾声**（极沉闷带哨音）惊醒主人，也成为推他迈出房门的契机。
- 出门即感到骤降寒意——这一夜是倒春寒最冷的夜，比预报还低三度。
- 先到鸡舍探看，见淑芬死守巢穴却精疲力竭，当机立断：**今夜必须介入**——返回卧室取上水桶与雨靴。
- 行动四节拍：
  - ① **第一趟外出（池塘）**：提空水桶去池塘舀一桶水（孵化箱加湿用），雨靴踩泥来回一次。
  - ② **回卧室处理保温箱底**：把水桶搁到桌边，温柔捧起黑猫（黑猫的噩梦开始），抽走天鹅绒软垫铺进保温箱底。
  - ③ **第二趟外出（鸡舍）**：身上仍沾着池塘水气与稻草屑，雨靴踩泥再来回一次；趁淑芬熟睡温柔抱走真蛋。
  - ④ **回卧室启动孵化**：把真蛋放入软垫上的保温箱，开黄色保温灯，加湿水，人工孵化正式开始。
- **听觉锚点**：老鼠墙缝听到的"人类雨靴来回走了两趟"对应 ① ③ 两次外出；"亮黄灯"对应 ④；"咔嚓"是数小时后蛋壳破裂声，与本夜主人动作无关。

## 游戏中的存在方式

### 不可见 / 不可交互

主人在 T0 清晨后可能在院里干活，但不进入对话流程，是永远缺席的「谜底之手」。**主线漫画演绎段（红顶屋天窗收束后 · 详见 `07-7.2` 步骤 15）**亦仅以**剪影 / 局部动作**出场（手、身影、捧着保温箱靠近鸡舍等），不正面亮相、不开口；保持"以缺席的温柔推动故事"的定位。

## 关于黑猫开窗的触发机制（与第二章关联）

说服黑猫开窗的关键**不是外部物证，而是玩家汇报案情后激活的猫的顿悟**：

- 黑猫当晚就在卧室现场，但**目击范围有限**——他**半睡半醒间听见主人深夜出门**（主人身上后来带着湿泥、稻草、池塘水气，黑猫当时没在意）→ 主人把黑猫扔到冷地板 → 抽走软垫 → **在桌上摆弄什么装置** → 开黄灯。**黑猫完全没看见过蛋**——没看见主人手里带蛋回来、没看见桌上或软垫上出现过蛋、不知道蛋是否进过卧室；他怒火熏心、半闭眼，**完全没看清桌上是什么装置（保温箱），更不知道软垫接下来去向**。怒火更让他根本没去推测"主人是不是干了什么大事"——只在意"软垫被占"。
- 玩家将调查所得（淑芬失蛋、老鼠雨靴/黄灯、悲伤蛙舀水、大黄鼻子确认蛋仍在红顶屋）**只列事实、不下结论**地汇报给猫后，黑猫**第一次**把"主人深夜出门 + 鸡舍味道 + 摆弄桌上装置 + 开黄灯"这一串怪行和"主人偷蛋"挂上钩——「那家伙就是凶手。他昨晚出门，**原来是去偷蛋的**。本喵的软垫、桌上那堆奇形怪状的玩意儿、那盏黄灯——全跟这颗破蛋有关。」
- **顿悟点由玩家激活**：这个"原来是去偷蛋的"**不是黑猫记忆库里早已存有的事实**——他从来没看见过蛋——而是被玩家详细的事实汇报触发的**第一次顿悟**。**侦探的工作成果直接体现为"让一个见过现场却没看出门道的关键证人开了窍"**。
- **悬念保留至红顶屋**：黑猫此刻仍**不知道**蛋的具体位置 / 状态、桌上是什么装置、软垫去向、黄灯用途、主人为何要偷蛋——这些必须**亲眼进屋看一遍**才能解开（=红顶屋物证总览的真正功能）；他和玩家此刻一样好奇。
- 猫的行动驱动力是**三重叠加**：**好奇**（"那家伙背着本喵干了什么 / 蛋是不是真的在屋里"——由玩家汇报激活的真心好奇）+ **自尊**（**大黄从树下撩拨**"被当面骗了一手 + 找回脸面"，见 `characters/黑猫.md`"面子台阶与下树"）+ **物权**（软垫去向 / 找回垫子）；三重保证了高自尊角色的完整性。

|------|------|
| 保温箱（临时搭建） | 铁丝固定，非成品，主人手工制作 |
| 箱底天鹅绒软垫 | 黑猫控诉的实物来源 |
| 池水桶 | 悲伤蛙「生命之源枯竭」的对应物 |
| 黄色保温灯 | 动物侧叙「刺眼黄灯」的对应物 |
