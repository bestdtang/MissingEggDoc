# 农场主（主人）

## 人设

- **可见性**：全程不可直接交互；通过**动物侧叙、环境痕迹、漫画收束分镜**构成形象——是本案真相的核心行动者，以**缺席的温柔**推动整条故事。
- **性格**：心思细腻的老式农人；不爱打扰动物自然规律，倾向旁观、确认"必须介入"才出手；对黑猫有特殊感情。
- **核心矛盾**：心疼淑芬连日焦虑，却不愿打断母鸡自然孵化的本能——两难下一直观望，直到寒夜骤降与大黄鼾声同时击中他，才当机立断。

## 时间线（事实链）

| 时段 | 行动 |
|------|------|
| T-3 下午 | 目击淑芬失控，理解为"母鸡孵化焦虑严重失常"；翻《母鸡产后护理与孵化指南》；原则"先等等，看她能否自己稳下来"，悄悄准备孵化设备**备而不用** |
| T-3 深夜 | 见淑芬傍晚已自行回稳，按兵不动；"她能自己来就别惊扰" |
| T-2 | 偶尔窗口观察淑芬（表面稳定，他不知小鸡已还真蛋）；继续完善孵化设备（温控调试、黄灯测试、铺垫材料） |
| T-1 白天 | 设备全部就绪，只差"必须今夜动手"的信号；心里预判"如果今夜气温再降就必须动手"；观察到蛋已有临破壳前的细微动静，判断稳定温湿度很关键 |
| **T-1 深夜** | **被大黄酒嗝鼾声惊醒** + 感受到**倒春寒最冷一夜** → 先到鸡舍探看见淑芬死守巢穴精疲力竭 → 当机立断今夜介入 |

### T-1 深夜行动四节拍

1. **第一趟外出（池塘）**：提空水桶舀一桶水（孵化箱加湿用），雨靴踩泥来回一次。
2. **回卧室处理保温箱底**：水桶搁桌边 → 温柔捧起黑猫（黑猫的噩梦开始）→ 抽走天鹅绒软垫铺保温箱底。
3. **第二趟外出（鸡舍）**：雨靴踩泥再来回一次；趁淑芬熟睡温柔抱走真蛋。
4. **回卧室启动孵化**：真蛋放入软垫上保温箱 → 开**黄色保温灯** → 加湿水 → 人工孵化正式开始；蛋本已临近破壳，保温箱只是在寒夜里补上稳定温湿度。

**听觉锚点对应**：老鼠墙缝听到的"雨靴来回两趟" = ① ③；"亮黄灯" = ④；"咔嚓" = 数小时后蛋壳破裂声，与本夜主人动作无关。

## 游戏中的存在方式

### 不可直接交互 / 仅漫画出场

一周目 3D 主场景中不正面亮相、不可直接交互；玩家只通过动物侧叙与环境痕迹理解主人。终局漫画允许剪影、手部、背影等局部动作出场，用于完成小鸡回巢与温柔收束。

### 外部行为痕迹（进屋前可发现）

| 物证 | 位置 | 指向 |
|------|------|------|
| 空水桶 | 门廊角落 | 主人深夜去池塘舀水后放回 |
| 雨靴泥脚印 | 院中小路 | 从池塘 / 鸡舍延伸至红顶屋（两趟） |
| 暖黄灯光 | 红顶屋窗缝 | 保温灯彻夜亮着 |
| 精美猫门 | 红顶屋墙面紧闭大门下方 | 专为黑猫凿刻，边缘可见手工凿痕与专属钥匙孔；黑猫有钥匙，玩家打不开也钻不过去 |
| 木质猫碗 | 门廊外侧 | 刻爪印纹样，非量产器皿，爱猫外部证据 |

### "红顶屋 = 昨夜事件中心"的多源体感（避免第二章突然空降）

| 事实 | 递给玩家的方式 |
|------|---------------|
| **大黄酒嗝鼾声** 是 T-1 深夜噪音锚点 | 小鸡阶段一·默认期的"水怪低吼"误读；大黄 ④ 可加非必台词"自己鼾声后院里像有雨靴走动"——与老鼠"两趟雨靴"互证。**不对玩家直说"鼾声惊醒了主人"**——叙事内侧因果不外露 |
| **保温灯彻夜亮** | 环境窗缝暖黄 + 老鼠（黄灯+雨靴）+ 红顶屋外物证拼贴 |
| **主人昨夜从鸡舍取走真蛋** | 大黄 ④ 嗅觉追踪 + 红顶屋外【案情汇报物证】 + 淑芬空窝无打斗痕迹形成空间联想；黑猫只知道软垫被拿走，不提供鸡舍味或装置目击 |
| **"红顶屋"作为目标建筑** | 蜗牛 Flash 已见主人抱蛋走向红顶屋 + 脚印汇拢红顶屋门 + 窗缝黄灯 + 猫门形制 + 淑芬旁白 / 老鼠一档 #8「主人连日趴窗看鸡舍」+ 大黄鼻子是**碎片之上的确认键**（不是唯一伏笔） |

### 角色侧叙

- **老鼠**："亮黄灯……雨靴来回两趟……然后是那声咔嚓。"
- **悲伤蛙**："生命之源正在枯竭……"（舀水那幕）
- **黑猫**："感情诈骗！！！那该死的两脚兽用温柔手法骗我！！！"（软垫被夺）
- **蜗牛 Flash**："他……抱着……**那颗蛋**……往那边……走去了……"（核心目击）

### 室内物证（漫画分镜呈现）
