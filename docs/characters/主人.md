# 农场主（主人）

## 人设

- **可见性**：全程不可直接交互；通过**动物侧叙、环境痕迹、室内物证**构成形象——是本案真相的核心行动者，以「缺席的温柔」推动整条故事。
- **性格**：心思细腻的老式农人；不爱打扰动物的自然规律，倾向在旁观察、确认「必须介入」后才出手；对黑猫有远超普通的特殊感情。
- **核心矛盾**：心疼淑芬连日焦虑，却不愿打断母鸡自然孵化的本能——两难之下一直观望，直到寒夜骤降与大黄的鼾声同时击中他，才当机立断。

---

## 时间线（事实链，与 `10-NPC主时间线` 对照）

### T-3 下午：目击失控，启动「备而不用」

- 目击淑芬院中狂奔、深蹲、大叫，判断「母鸡孵化焦虑严重失常」。
- **第一反应并非立即取蛋**：翻出《母鸡产后护理与孵化指南》细读，心里的原则是「先等等——她要是能自己稳下来，就不要插手」。
- 同步悄悄开始准备孵化设备（保温箱框架、黄色加温灯、水桶）作为应急储备，**备而不用**。

### T-3 深夜：淑芬自然回稳，主人暂缓行动

- 见淑芬傍晚已恢复正常孵化，主人松口气，将设备靠墙备着，今夜按兵不动。
- 内心逻辑：「她能自己来，就别惊扰。」

### T-2：观望为主，推进准备

- 偶尔从窗口看淑芬状态——表面稳定（他不知道小鸡已还了真蛋），判断无需立即介入。
- 悄悄对孵化设备做细节完善（温控调试、黄灯测试、箱内铺垫材料）。

### T-1 白天：万事俱备，等待信号

- 孵化设备全部就绪，只差一个「必须今夜动手」的明确信号。
- 持续观察淑芬：连续数日高度紧张孵蛋，疲惫加剧，体力明显透支。
- 心里预判：「如果今夜气温再降，就必须动手了。」

### T-1 深夜：倒春寒 + 大黄的鼾声——行动

- **大黄的酒嗝鼾声**（极沉闷带哨音）惊醒主人，也成为推他迈出房门的契机。
- 出门即感到骤降寒意——这一夜是倒春寒最冷的夜，比预报还低三度。
- 走到鸡舍，见淑芬死守巢穴却精疲力竭，当机立断：**今夜必须介入**。
- 行动三步：①去池塘舀一桶水（孵化箱加湿用）→ ②进卧室，温柔捧起黑猫（黑猫的噩梦开始），抽走天鹅绒软垫铺保温箱底 → ③去鸡舍，趁淑芬熟睡温柔抱走真蛋，开黄灯，启动人工孵化。

## 游戏中的存在方式

### 不可见 / 不可交互

主人在 T0 清晨后可能在院里干活，但不进入对话流程，是永远缺席的「谜底之手」。

### 外部行为痕迹（进屋前可发现）

| 物证 | 位置 | 指向信息 |
|------|------|---------|
| **空水桶** | 门廊角落 | 主人深夜去池塘舀水后放回 |
| **雨靴泥脚印** | 院中小路 | 从池塘方向延伸至红顶屋门口 |
| **暖黄灯光** | 红顶屋窗缝 | 保温灯彻夜亮着 |
| **猫洞** | 红顶屋墙面 | 专门为黑猫凿刻，边缘可见手工痕迹 |
| **木质猫碗** | 门廊外侧 | 刻有爪印纹样，非量产器皿，是主人爱猫的外部证据 |

### 室内物证（进屋后）

| 物证 | 含义 |
|------|------|
| 粗糙保温箱 | 临时搭建，铁丝固定，非成品 |
| 箱底天鹅绒软垫 | 黑猫的御用垫（黑猫控诉的起因） |
| 池水桶 | 加湿用，水迹未干 |
| 黄色保温灯 | 与动物侧叙「黄灯」对应 |
| 《母鸡产后护理与孵化指南》 | 翻开，有折角，主人研究过的痕迹 |
| 黑猫旧照片（裱框） | 桌角，主人与黑猫的合影 |
| 备用猫薄荷球（数颗） | 床头散放，表明主人长期为黑猫备着 |

### 角色侧叙

- **老鼠兄弟**：「亮黄灯……雨靴来回两趟……然后是那声咔嚓。」
- **悲伤蛙**：「生命之源正在枯竭……」（主人舀水那一幕）
- **黑猫**：「感情诈骗！！！那该死的两脚兽用温柔手法骗我！！！」（软垫被夺）
- **蜗牛 Flash**：「他……抱着……那颗白圆球……往那边……走去了……」（核心目击）

## 关于黑猫开窗的触发机制（与第二章关联）

说服黑猫开窗的关键**不是外部物证，而是玩家汇报案情后触发的猫自身记忆**：

- 黑猫当晚就在卧室现场，主人将它放到冷地板后，曾取出白色圆滚滚的东西极小心放到软垫上。猫当时怒火熏心，没有意识到那是蛋。
- 玩家将调查所得（淑芬失蛋、老鼠线索、悲伤蛙目击舀水、红顶屋黄灯）汇报给猫后，猫的记忆碎片自行拼合：「那就是蛋。我亲眼看见他把它放在我的软垫上。」
- 猫的行动驱动力是**自主的「亲眼确认 + 要回垫子」**，而非被玩家说服——这保留了其高自尊的角色完整性。

**室内物证（进屋后，由玩家发现）**

| 物证 | 含义 |
|------|------|
| 保温箱（临时搭建） | 铁丝固定，非成品，主人手工制作 |
| 箱底天鹅绒软垫 | 黑猫控诉的实物来源 |
| 池水桶 | 悲伤蛙「生命之源枯竭」的对应物 |
| 黄色保温灯 | 动物侧叙「刺眼黄灯」的对应物 |
| 《母鸡产后护理与孵化指南》 | 翻开有折角，主人备课的痕迹 |
