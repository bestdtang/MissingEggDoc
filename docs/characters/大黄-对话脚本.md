# 大黄 · 完整对话脚本

> 本文档是 [大黄角色设计](./大黄.md) 的逐句实施稿。  
> 格式说明：每行为一个独立对话框，玩家点击推进。`描述:` 为轻量演出。`〔系统注〕` 为策划机制标注，不出现在游戏内。

---

## 一、阶段①：前置 / 完全睡

> 〔系统注〕触发条件：玩家首次靠近**谷仓墙根旁**醉倒的大黄（非狗窝）。锁点结构：**无锁**（靠近后直接进入初见脚本，无需选择）。  
> **狗窝空窝 E22** 与 **谷仓底缺梯 E06** 为环境 / 空间反馈，具体游戏文本见 [13-玩家线索与交互点总表](../13-玩家线索与交互点总表.md)。  
> E04 不弹出线索大图；本段描述兼作大黄初见演出。2-A 结束后统一刷新笔记，写入【沉闷带哨音的鼾声】与【大黄身下像压着一架木梯】。

描述：谷仓墙根旁，一架短木梯侧倒在地，一只大黄色的狗半压在上面，低沉的鼾声带着哨音

## 二、主交互阶段

> 〔系统注〕阶段②-④ 为主交互阶段。锁点结构：**醒酒锁**（阶段②半睡半醒 + 持【谷物泡水】 → 可叫醒）+ **借梯锁**（大黄已清醒 + E06 已确认缺少垫脚物 → 出现索要短梯）+ **单锁型**（展示插图 ③→④）。  
> **动机链**：发现醉倒大黄与短梯 → 泡水叫醒 → 发现谷仓入口缺垫脚物 → 向清醒大黄索要短梯 → 谷仓入口架短梯 → 翻过小围墙进入平台跳跃 → 攀顶。E06 可在醒酒前或醒酒后发现；它不阻止醒酒，只控制借梯对话是否出现。

### 2-A · 阶段②：半睡半醒（宿醉复述）

> 〔系统注〕初见时自动进入。E04 已在进入本段时成立：玩家已经看到醉倒大黄和被压住的短梯。大黄仍处于宿醉半睡半醒状态，能说出目击片段，但不能可靠配合行动。  
> 本段结束后统一刷新笔记：写入【沉闷带哨音的鼾声】、【大黄身下像压着一架木梯】、【大黄目击：乌鸦叼蛋上屋顶】。随后大黄停留在阶段②，等待玩家用【谷物泡水】醒酒。

大黄："嗝——"  
大黄："谁……"  
玩家："淑芬的蛋不见了。你看到过什么吗？"  
描述：大黄的耳朵动了一下，目光慢慢聚到玩家身上  
大黄："蛋……"  
大黄："乌鸦……叼走了。"  
玩家："乌鸦？"  
大黄："嗯……飞到谷仓屋顶去了。"  
大黄："我追了……跳不上去。"  
大黄："只咬到空气。"  
描述：大黄把脑袋重新压回前爪上，声音低下去  
大黄："我失职了……没有保护好淑芬的蛋。"

〔已持 E06〕  
玩家："你身下压着一架短梯，能先挪一下吗？谷仓入口那道矮墙翻不过去。"  
描述：大黄迷迷糊糊蹬了下腿，短梯反而被压得更实  
玩家："看来得把他叫醒。"

> 〔系统注〕若玩家此时未持【谷物泡水】，转入 **2-C′ · 未持泡水时叫醒失败**；若已持【谷物泡水】，可直接转入 **2-C · 阶段②→③：叫醒大黄**。未持 E06 时不播放本短段，但玩家仍可在阶段②与大黄交互，并按是否持【谷物泡水】进入 2-C 或 2-C′。

### 2-C · 阶段②→③：叫醒大黄

> 〔系统注〕触发条件：大黄处于阶段②半睡半醒 + 玩家持【谷物泡水】，与大黄交互。E06 不是醒酒前置，只影响醒酒后是否出现索要短梯选项。  
> 提示来源（三源任一；泡水位置无拼合要求）：小鸡阶段一·叫醒帮忙 / 淑芬阶段 1 / 老鼠一档 #7（1 块奶酪）。  
> 辅助环境：**狗窝**旧木桶 E11 可作为泡水方向暗示，非硬锁；具体观察文本见 [13-玩家线索与交互点总表](../13-玩家线索与交互点总表.md)。

玩家："大黄，先喝点这个。"  
描述：大黄眯着眼嗅了嗅，慢慢伸出舌头，喝了几口，停住，再喝  
描述：片刻后，大黄猛地甩了下脑袋——湿树叶从额头飞出去  
大黄："呕——"  
大黄："嗝——"  
描述：大黄使劲眨眼，目光开始聚焦  
大黄："……我刚才说什么来着？"  
玩家："你说乌鸦叼走了蛋，飞到谷仓屋顶。"  
描述：大黄低下头，耳朵耷拉着  
大黄："对。"  
大黄："我没拦住。"  
描述：大黄深吸一口气，四肢从梯子上撑起来，摇摇晃晃站稳

> 〔系统注〕进入阶段③。大黄只是从宿醉中清醒，不恢复嗅觉自信，不出现“我的鼻子回来了”。若已持 E06，则解锁 2-E；若未持 E06，只进入清醒状态，不出现索要短梯选项，待玩家之后发现 E06 再解锁 2-E。

### 2-C′ · 未持泡水时叫醒失败

> 〔系统注〕大黄处于阶段②半睡半醒，但玩家尚未持【谷物泡水】时，与大黄交互不进入 2-C。可提示去找能醒酒的东西；若玩家已持 E06，可加强「醒了再借梯」动机。

玩家："大黄，醒醒。"  
描述：大黄哼了一声，把头埋进前爪里  
大黄："嗝……别吵……"

〔未持 E06〕  
玩家："这样叫不醒。得找点能让他清醒的东西。"

〔已持 E06〕  
玩家："这样叫不醒。得先让他清醒过来，短梯才借得出来。"

### 2-D · 阶段③：清醒后（未知假蛋）

> 〔系统注〕大黄已清醒，但仍坚信乌鸦偷走的是真蛋；他不重复完整目击，不恢复嗅觉，不主动借梯。若未持 E06，只提供默认回访；若已持 E06，出现 2-E 索要短梯选项。

大黄："我清醒多了。"  
大黄："刚才说的是真的。"  
大黄："乌鸦叼着那个白色圆东西，飞到谷仓屋顶去了。"  
描述：大黄低头看着地面，爪尖刮了下泥  
大黄："我没拦住。"  
大黄："这事得查清楚。"

> 〔系统注〕大黄此时仍处在错误认知内：他把乌鸦叼走的白色圆物当成真蛋，也还没意识到自己把时间搞错。时间错位由小鸡招供回收，假蛋认知由 E10 展示插图回收。

大黄："他在听的。"
