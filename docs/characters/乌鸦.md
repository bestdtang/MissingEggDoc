# 囤积癖乌鸦

## 人设

- **视觉**：贼眉鼠眼黑乌鸦；栖息全农场最高点（谷仓屋顶）。巢极大，堆满「财宝」（啤酒瓶盖、碎玻璃、糖纸等）。身下压着惹眼的 **【画爱心的白蛋】**——实为石头。
- **性格**：极度贪婪、眼光极差的废品收藏家；领地意识极强；极度嚣张。
- **关系**：视底下 **大黄** 为「跳跃能力为零的废物」；与 **黑猫** 有多年宿怨（凌晨嘎嘎叫扰猫睡眠）。
- **动机**：无论何物，进巢即 **绝版手办**；谁也别想拿走，尤其戴帽侦探。

## 时间线（事实链）

1. **贼眼识「宝」（T-2 清晨）**  
   - 乌鸦晨巡农场找反光小物；发现**鸡舍外草丛**里反光——实为 **小鸡遗弃的、涂劣质反光白漆的石头假蛋**（上有小鸡所画爱心等标记）。  
   - 俯冲叼起；**大黄**狂吠扑上，体型笨重够不着；乌鸦绕顶炫耀，带「战利品」回屋顶安全区。  
   - 回巢翻面：除爱心侧外，有小鸡画的 **吐舌鬼脸** 等涂鸦；乌鸦惊为「充满神秘力量的部落图腾宝石」，供在巢中央。

2. **替人背锅的「大 Boss」（T0 开局）**  
   - 真蛋已于昨夜被主人拿走；今早全农场找「画爱心的白球」。  
   - 乌鸦在屋顶不知内情，只顾向全场炫耀新收藏。  
   - **大黄**宿醉断片、时间线记混，痛哭指认乌鸦掠走「蛋」；另有**小鸡**池塘误导、**老鼠**可选情报等，把玩家推上「上屋顶讨蛋」的强目标。  
   - 结果：乌鸦在懵逼又嚣张的状态下，无辜成为**第一章终极反派位**。

## 交互与台词

### 被捧上神坛的嫌疑人（间接）

三方来源将玩家目标收拢至「上屋顶夺蛋」：

1. **大黄**（宿醉断片证词）：「亲眼」看到黑色恶魔掠走白色宝贝——时间线模糊，但语气确凿。大黄的证词有一半来自 **黑猫在 T-2 清晨** 漫不经心说的那句「那乌鸦把草丛里白色圆球叼到屋顶去了」，被大黄的内疚记忆放大成「我亲眼见证了失职」。
2. **黑猫**（树上独语）：「屋顶那只蠢东西……守着他的宝贝……」——表面抱怨乌鸦噪音，实则有意把玩家注意力引向屋顶（黑猫已知石头是假蛋，选择隐瞒以报多年宿怨）。
3. **小鸡侦探团**（前期误导链）：水怪→池塘→蛙→此路不通→最终被玩家自行推断至屋顶乌鸦。
4. （可选）**老鼠兄弟**收费闲话亦可强化「屋顶白球」方向。

### 屋顶巅峰对峙

- **位置**：玩家经推箱、干草垛等互动爬上谷仓屋顶。  
- **系统**：获得【假蛋】类条目——**【画鬼脸的沉重石头】**（或 UI 等价命名）。  
- **乌鸦**：「就算你拿到了，也别想抹去上面的神秘图腾！」  
- **玩家体验**：从「破案狂喜」跌入「被当猴耍」——意识到主要被**老实狗乌龙证词 + 熊孩子石头**忽悠；**真蛋仍下落不明**。
