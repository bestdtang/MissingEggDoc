# 囤积癖乌鸦

## 人设

- **视觉**：贼眉鼠眼黑乌鸦；栖息全农场最高点（**谷仓屋顶**）。巢极大堆满"财宝"（啤酒瓶盖、碎玻璃、糖纸）；身下压惹眼的**【画爱心的白蛋】**——实为石头。
- **性格**：极度贪婪、眼光极差的废品收藏家；领地意识极强；极度嚣张。
- **关系**：视底下大黄为"跳跃能力为零的废物"；与黑猫多年宿怨（凌晨嘎嘎扰猫）。
- **动机**：无论何物进巢即"绝版手办"；谁也别想拿走，尤其戴帽侦探。

## 时间线（事实链）

1. **T-2 清晨 · 贼眼识"宝"**：晨巡发现**鸡舍外草丛**里的天然白色圆石（实为小鸡遗弃的石头假蛋）；俯冲叼起，大黄扑咬跳不高够不着；乌鸦绕顶炫耀带"战利品"回屋顶。回巢翻面发现小鸡用木炭画的爱心和吐舌鬼脸，**惊为"充满神秘力量的部落图腾宝石"，供巢中央**。
2. **T-2 正午～下午 · 玻璃眩光公害**：谷仓顶摆弄玻璃珠、碎玻璃等闪亮收藏，反射光扫向窗台下/谷仓边午睡的黑猫，**毛尖险燎**——为黑猫稍后**一次**上顶对峙埋雷。
3. **T0 开局 · 替人背锅的"大 Boss"**：真蛋已于昨夜被主人拿走；今早全农场找的是淑芬那颗画爱心的蛋。乌鸦不知内情，只顾向全场炫耀巢里的画爱心和鬼脸白石头。三方误导（**大黄清醒目击 + 宿醉后复述** + **黑猫树上独白引导** + **小鸡水怪转移**）把玩家推上"上屋顶讨蛋"的强目标——乌鸦在懵逼又嚣张中无辜成为**第一章终极反派位**。

---

## 玩家目标收拢（间接铺垫）

三方汇聚把玩家方向收到"上屋顶夺蛋"：

1. **大黄**（**清晨清醒**形成核心、**宿醉后**向玩家复述）：咬死「乌鸦把淑芬的蛋叼上屋顶、自己没拦住」；醉话里可嘴瓢昼夜，**不出现「黑色恶魔」**（该修辞**仅**悲伤蛙的史诗口吻）。
2. **黑猫**（树上独白）："屋顶那只蠢东西……守着他的宝贝……"——表面抱怨噪音，实则有意引导玩家（黑猫已知是石头，选择隐瞒报多年宿怨）。
3. **小鸡侦探团**（前期误导链）：水怪→池塘→蛙→此路不通→最终被玩家自行推断至屋顶乌鸦。
4. （可选）**老鼠收费闲话**强化"屋顶爱心石头"方向。

---

## 屋顶巅峰对峙

- **位置**：第一章末玩家向大黄索要【干草垛】（大黄 ③ 起身后），走到谷仓底部交互点放置，完成平台跳跃路径后爬上谷仓屋顶。
- **系统**：获**【画鬼脸和爱心的沉重白石头】**（或 UI 等价命名）。
- **乌鸦台词 · 必带黑猫上顶信息**：对峙中须有一句点明 **T-2 下午黑猫已经上来过**（起因可回扣**玻璃珠眩光燎毛**；乌鸦主观仍误读成嫉妒/怂），与事实链（黑猫瞥见假蛋却沉默）形成**表里反差**；玩家事后持老鼠 B 组 #10 或听黑猫自白时，可回扣「原来乌鸦嘴里那只黑毛就是知情者」。
  - **示例方向**（可拆两句轮播）：`"本王的玻璃珠阵法差点给那只黑毛烫个新发型——他还敢爬上来瞪我的部落宝石！"` → `"……你又是什么？就算你从本王爪下抠走了，也别想抹去上面的神秘图腾！"`
- **玩家体验**：从"破案狂喜"跌入"被当猴耍"——意识到被**老实狗乌龙证词 + 熊孩子石头**忽悠；**真蛋仍下落不明**。
- **设计要点**：谷仓顶访问由**第一章关键质询锁**控制（解酒 + 发现干草垛 + 主动索要 + 放置缺一不可），避免"开局即上谷仓"，让"上屋顶 = 多源误导汇聚 + 主动行动"的戏剧节奏成立。
