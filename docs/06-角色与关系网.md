## 6.4 角色细化与交互脚本

人设、时间线节点、分阶段交互与台词等**事实级**正文集中在 `docs/characters/` 下各 `.md` 详档；上表"详档"列与各角色文件一一对应。

**增删角色须同步四处**：本文件 6.2 关系图与 6.3 主要角色表、对应 `docs/characters/*.md` 详档、`docs/00-文档索引.md`、根目录 `README.md` 入口表。

## 6.5 术语速查

- **圆形物术语**：不设通用抽象词。玩家、淑芬、小鸡、大黄等谈案件时统一说**蛋 / 真蛋 / 画爱心的蛋**；策划说明直写**石头假蛋 / 月影**；动物台词按各自口吻另取形容（如悲伤蛙说"玉盘"），避免所有角色共用同一套说法。

## 6.6 时间线与关键事件

- **策划全知轴**：以 `05` 的 **5.6 上帝视角真相时间线** 为唯一时间基准（完整事实链总表见 `10-NPC主时间线.md`）；对白中"昨天 / 前天"须逐句对照校准。
- **玩家 Reveal 顺序**：以 `07` 两章主线为准。
