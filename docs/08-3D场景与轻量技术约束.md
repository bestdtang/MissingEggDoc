# 08 3D 场景与轻量技术约束

## 8.1 场景规模与镜头语言

- **Lowpoly 微缩农场**：高度集约、色彩明快；强调「**盒子盆景**」式构图，利于移动端性能与镜头可读性。
- **关键地标**：
  - **谷仓**（**全图最高点**、乌鸦巢；屋顶为第一章平台跳跃挑战目标）；
  - **鸡舍**（淑芬常驻、第二章小鸡门口愧疚待命、水槽内放谷物泡水）；
  - **池塘**（悲伤蛙常驻、薄荷鱼藏匿点、T-1 夜主人舀水）；
  - **红顶屋**（紧闭大门 + **专为黑猫凿刻的精美猫门** + **屋顶天窗**；玩家进屋走天窗，是第二章末平台跳跃挑战目标）；
  - **大橡树**（黑猫栖居 · 第一章独白触发源 · 第二章首次现身位 · 第二章大黄摇树召唤位）；
  - **谷仓墙缝**（老鼠墙缝双道裂口情报窗口）；
  - **狗窝**（大黄第一/二章初任务点 + 旁置旧木桶环境物证 + 苹果渣残迹）。
- 拖拉机等非主线地标可作环境远景。

## 8.2 交互复杂度上限

- **主交互**：点击触发气泡；少量**物证出示**；环境可拾取关键物。
- **垂直关卡**（两次屋顶 · 首尾呼应）：
  - **第一章末 · 谷仓屋顶**：推箱 / 干草垛堆叠等轻量平台跳跃感，目标对峙乌鸦取假蛋；
  - **第二章末 · 红顶屋顶**：玩家循"屋檐 / 雨水桶 / 旁侧木箱堆叠 → 屋脊 → 天窗"等组合跳跃登顶，从被黑猫推开的**天窗下降进入卧室**（具体跳跃路径与卡点待美术与关卡定稿）。
  - 两次均为休闲向，**失败惩罚宜轻**（不掉血、不读条），失败后自动落回起跳点或最近平台。
- **同屏信息**：避免同时高亮过多可互动物；叙事高潮处控制 NPC 同屏数量。

## 8.3 性能与包体友好原则

- **3D + 2D 分层**：环境为 **3D Lowpoly**；角色情绪高光可用 **2D 动态立绘** 补强（效率与表现力平衡，见 `11`）。
- 面数/贴图/骨骼数等预算以工程量化为准；策划侧默认「单场景可拆区加载」友好。

## 8.4 与推理玩法的接口

- **可读现场**：屋顶对峙路径需清晰可读；若保留任何「跳台视线差」机关，以 `07` 单案为准。
- **检查点**：建议章首自动存档；**第一章谷仓屋顶对峙、第二章红顶屋顶天窗下降**前各设检查点。
