# 08 场景与布置

## 8.1 文档定位

本文只描述农场空间、关键地标、路径组织与叙事布置原则。性能、包体、面数、加载等工程指标不在策划文档中展开，后续由工程规范承接；本文仅保留会影响玩法与美术落地的空间要求。

## 8.2 场景整体气质

- **Lowpoly 微缩农场**：高度集约、色彩明快；强调「**盒子盆景**」式构图，让玩家一眼读懂地标、路径和动物活动区。
- **一张可反复回看的案发现场**：场景不是单纯背景，而是案件信息的容器。每个主要地标都应有可观察细节、动物行动痕迹或后续回扣。
- **童话感与案情感并存**：整体可爱、温暖，但局部布置允许出现“看起来很可疑”的反差点，如烧焦稻草、泥脚印、空水桶、深色兽毛。

## 8.3 关键地标与布置要点

### 鸡舍

- 淑芬常驻点，也是案件情绪起点。
- 周边需容纳小鸡搞事碎片：短木炭、散落羽毛、小鸡偷听点、水槽与谷物泡水。
- 第二章小鸡可在门口愧疚待命，形成“案发地回访”的情绪回响。

### 狗窝

- **T0 开局空窝**：大黄未回窝，醉倒在**谷仓旁**；玩家路过可发现【狗窝里不见大黄】（E22），项圈/干草窝仍在，形成「保安不在岗」的疑点。
- 可保留旧木桶等环境物（醒酒暗示），但**无大黄、无木梯**。
- 位置连接鸡舍与谷仓，方便玩家从鸡舍线自然路过空窝再往谷仓寻鼾声。

### 池塘

- 悲伤蛙常驻点，承担水怪误导、主人取水证词和薄荷鱼回扣。
- 需要保留荷叶、绿垫、池边藏物等可点击点位；薄荷味绿垫应视觉上足够特殊，便于玩家在老鼠情报后想起并回访。

### 谷仓

- **全图最高点**，乌鸦巢所在处，也是第一章平台跳跃挑战目标。
- **墙根**：T-2 大黄踩梯抓鸟导致**侧倒木梯** + 醉倒大黄（E04）；苹果渣、湿树叶散落一旁；底部 E06 攀梯点「无梯上不去」。
- 玩家从谷仓旁借来【木梯】后在 E06 架设攀顶。E06 缺梯 + E04 像木梯 → 拼合后确立「叫醒大黄借梯」。
- 屋顶对峙区要能容纳乌鸦巢、白石头假蛋、动物爪印 / 烧焦稻草等误导回收点。

### 大橡树

- 黑猫栖居点，第一章可作为独白触发源，第二章成为首次现身位与大黄摇树召唤位。
- 树冠、枝杈、阴影和午睡痕迹可作为“黑猫一直在场但未正面出现”的视觉暗线。

### 红顶屋

- 第二章视觉锚点：远看温暖，近看封闭。
- 外部必须包含紧闭大门、暖黄窗光、精美猫门 / 专属钥匙孔、屋顶天窗、陶瓷碗、雨靴泥脚印、空水桶、深色兽毛等信息点。
- 卧室内部不做 3D 实时场景，终局揭晓由手帐漫画承接；红顶屋外部布置需要为漫画里的内部真相提供足够期待。

### 红顶屋墙缝

- 老鼠兄弟情报窗口。
- 墙缝可设计为双道裂口，既方便角色藏身，也让玩家理解“他们知道很多墙内墙外的八卦”。

## 8.4 路径与视线组织

- **主线视线**：鸡舍 → 狗窝 / 池塘 → 谷仓 → 红顶屋，应能形成从案发地到真相地的清晰空间迁移。
- **两次登高呼应**：
  - 第一章末：谷仓屋顶，目标是对峙乌鸦并回收假蛋误导。
  - 第二章末：红顶屋屋顶，目标是从天窗进入终局漫画。
- **回访动机**：池塘、鸡舍、红顶屋外等地点需要在不同阶段显示不同意义，避免一次性用完。
- **可读动线**：玩家不应靠猜地图推进；关键路径用地标、光源、动物站位、可点击物和笔记本过程行共同提示。

## 8.5 交互布置原则

- **主交互**：点击触发气泡；少量物证出示；环境可拾取关键物。
- **同屏信息**：避免同时高亮过多可互动物；叙事高潮处控制 NPC 同屏数量。
- **平台跳跃**：两次屋顶挑战均为休闲向，失败惩罚宜轻（不掉血、不读条），失败后自动落回起跳点或最近平台。
- **检查点**：建议章首自动存档；第一章谷仓屋顶对峙、第二章红顶屋顶天窗下降前各设检查点。
