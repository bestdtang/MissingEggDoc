# 08 场景与布置

## 8.1 文档定位

本文只描述农场空间、关键地标、路径组织与叙事布置原则。性能、包体、面数、加载等工程指标不在策划文档中展开，后续由工程规范承接；本文仅保留会影响玩法与美术落地的空间要求。

## 8.2 场景整体气质

- **Lowpoly 微缩农场**：高度集约、色彩明快；强调「**盒子盆景**」式构图，让玩家一眼读懂地标、路径和动物活动区。
- **一张可反复回看的案发现场**：场景不是单纯背景，而是案件信息的容器。每个主要地标都应有可观察细节、动物行动痕迹或后续回扣。
- **童话感与案情感并存**：整体可爱、温暖，但局部布置允许出现“看起来很可疑”的反差点，如烧焦稻草、泥脚印、空水桶、深色兽毛。

## 8.3 关键地标与布置要点

### 鸡舍

- **结构作用**：案件情绪起点 + 小鸡误导链起点 + 主人取蛋终点回扣。
- 淑芬常驻点，也是玩家接案、确认空窝与回访情绪的核心位置。
- 周边需容纳小鸡搞事碎片：短木炭、散落羽毛、小鸡偷听点、水槽与谷物泡水。
- 鸡舍门槛需保留压平稻草、旧蛋壳碎片等氛围细节；这些只增加案发地生活痕迹，不替代主线物证。
- 第二章小鸡可在门口愧疚待命，形成“案发地回访”的情绪回响。

### 院中主路 / Flash 宽叶

- **结构作用**：全案目击轴。小鸡抱蛋、乌鸦俯冲叼石头、主人抱真蛋走向红顶屋，都从这条主路或其近旁经过。
- Flash 的宽叶位于鸡舍外侧小路旁，夹在鸡舍出口与院中主路交汇处；活动半径极小，但视野覆盖关键路过事件。
- 宽叶需要可点击粘液轨迹，表现 Flash 的慢速与“看见很多但说得很慢”的角色功能。
- 草丛边保留 T-3 石头假蛋被丢下、T-2 乌鸦俯冲叼走的空间记忆；不必做成独立硬锁，但要让乌鸦来源方向在回溯时成立。

### 狗窝

- **结构作用**：保安缺席提示 + 去谷仓找大黄的空间引导。
- **T0 开局空窝**：大黄未回窝，醉倒在**谷仓旁**；玩家路过狗窝（E22 · 氛围观察）可见空窝、项圈与干草窝仍在，形成「保安不在岗」的即时空间提示，但不入笔记本。
- 可保留旧木桶等环境物（醒酒暗示），但**无大黄、无木梯**。
- 位置连接鸡舍与谷仓，方便玩家从鸡舍线自然路过空窝再往谷仓寻鼾声。

### 池塘

- **结构作用**：水怪误导承载区 + 主人取水物证回扣 + 薄荷鱼找回地。
- 悲伤蛙常驻点，承担水怪误导、主人取水证词和薄荷鱼回扣。
- 需要保留荷叶、绿垫、池边小鸡脚印、池边取水痕迹等可点击点位；薄荷味绿垫应视觉上足够特殊，便于玩家在老鼠情报后想起并回访。
- 池边松果、软泥与圆底压印共同表现夜里有人急着取水；它们只强化氛围与互证，不抢空水桶 / 雨靴脚印的主线地位。

### 谷仓

- **结构作用**：第一章登高目标 + 乌鸦误导回收点 + 黑猫薄荷鱼线前置空间。
- **全图最高点**，乌鸦巢所在处，也是第一章平台跳跃挑战目标。
- **墙根**：T-2 大黄踩短木梯追鸟导致**侧倒木梯** + 醉倒大黄（E04）；苹果渣、湿树叶散落一旁。
- **入口小围墙**：E06 是谷仓平台跳跃关卡的起点。未架设短木梯前，玩家不能翻过小围墙，也不能开始谷仓平台跳跃。
- 玩家从谷仓旁借来【木梯】后在 E06 架设短木梯，翻过小围墙进入通往谷仓屋顶的平台跳跃路线。E04 负责发现醉倒大黄与短梯并进入醒酒线；E06 负责确认缺少垫脚物并解锁清醒后的借梯对话。
- 平台路线中段需容纳谷仓午睡点、烧焦稻草 / 皮毛与玻璃珠反光；它们连接黑猫薄荷鱼线、乌鸦玻璃珠阵法和后续追问。
- 屋顶对峙区要能容纳乌鸦巢、白石头假蛋、乌鸦玻璃制品、混杂毛发、巢前动物爪印等误导回收点；乌鸦巢里的黑色细毛与红顶屋外黑毛形成二次疑问。

### 大橡树

- **结构作用**：黑猫隐藏声源 + 第二章召唤舞台 + 下树线索之一。
- 黑猫栖居点，第一章可作为独白触发源，第二章成为首次现身位与大黄摇树召唤位。
- 树冠、枝杈、阴影、树根抓痕可作为“黑猫一直在场但未正面出现”的视觉暗线；树根痕迹计入【下树线索】。黑色毛发集中放在乌鸦巢与红顶屋外，避免树下重复同一信息。
- 第二章黑猫现身后，树下成为案情汇报线、薄荷鱼线与开窗前汇合的对话舞台；附近需要能容纳大黄和黑猫同屏站位。

### 红顶屋

- **结构作用**：第二章目标建筑 + 真相入口 + 终局漫画承接点。
- 第二章视觉锚点：远看温暖，近看封闭。玩家先确认正门进不去，再通过猫门、食碗、兽毛和树根抓痕推到黑猫。
- 外部必须包含紧闭大门、暖黄窗光、精美猫门 / 专属钥匙孔、屋顶天窗、陶瓷碗、雨靴泥脚印、空水桶、深色兽毛、窗台下磨亮爪痕等信息点。
- 红顶屋外平台路线是第二次登高挑战；黑猫从屋内推开天窗后，屋檐、雨水桶、木箱等才成为通向天窗的路线。
- 卧室内部不做 3D 实时场景，终局揭晓由手帐漫画承接；红顶屋外部布置需要为漫画里的内部真相提供足够期待。

### 红顶屋墙缝 / 老鼠洞

- **结构作用**：老鼠兄弟情报入口 + 薄荷鱼付费路径入口 + 红顶屋外围听觉证词来源。
- 墙缝位于红顶屋底层外墙，既是玩家交易窗口，也是老鼠长期居住的墙体裂口；外观上应像“老鼠洞 + 细墙缝”的组合，而不是普通门洞。
- 结构建议保留横向裂口与斜向上细缝：横向裂口用于菜单、算盘、奶酪交易和老鼠探头；斜向上细缝解释他们能听见屋内黄灯嗡声、雨靴脚步，也能看到院中近景与远处谷仓屋脊剪影。
- 这里不能直接看见红顶屋室内，避免提前泄露保温箱、软垫用途和蛋的具体位置；老鼠只提供声光、气味、方向和八卦。
- 黑猫现身后，墙缝还承担“猫鼠旧怨”的轻提示：老鼠紧张、黑猫盯缝，但不在此处直接交代薄荷鱼真相。

## 8.4 路径与视线组织

- **主线视线**：鸡舍 → 狗窝 / 池塘 → 谷仓 → 红顶屋，应能形成从案发地到真相地的清晰空间迁移。
- **院中主路**：串联鸡舍、Flash 宽叶、狗窝、谷仓与红顶屋，是玩家回溯三天行动轨迹的视觉骨架。
- **两次登高呼应**：
  - 第一章末：谷仓屋顶，目标是对峙乌鸦并回收假蛋误导。
  - 第二章末：红顶屋屋顶，目标是从天窗进入终局漫画。
- **回访动机**：池塘、鸡舍、红顶屋外等地点需要在不同阶段显示不同意义，避免一次性用完。
- **可读动线**：玩家不应靠猜地图推进；关键路径用地标、光源、动物站位、可点击物和笔记本过程行共同提示。

## 8.5 交互布置原则

- **主交互**：点击触发气泡；少量物证出示；环境可拾取关键物。
- **奶酪碎**：作为探索奖励散落在地面、轻跳和隐藏点；优先放在自然回访路径、平台跳跃边缘、池塘荷叶终点、屋檐和树洞等位置，不承担主线判断。
- **同屏信息**：避免同时高亮过多可互动物；叙事高潮处控制 NPC 同屏数量。
- **平台跳跃**：两次屋顶挑战均为休闲向，失败惩罚宜轻（不掉血、不读条），失败后自动落回起跳点或最近平台。
- **检查点**：建议章首自动存档；第一章谷仓屋顶对峙、第二章红顶屋顶天窗下降前各设检查点。
