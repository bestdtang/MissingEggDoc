# 12 数据埋点与验收

## 12.1 核心漏斗与关键事件

与 `02` KEI 对齐建议跟踪：

1. 开场接委托 → 首次打开笔记本（第一页 · 淑芬委托便签贴入）
2. 关键质询·小鸡（初次）→ 获得【水怪说】（鸡舍搞事碎片 ≥ 2 条解锁）
3. 关键质询·悲伤蛙 → 获得"黑色恶魔 + 主人舀水"两条线索（持水怪说解锁）
4. 大黄 ②（半睡半醒）→ 自动入「大黄目击：乌鸦叼蛋上屋顶」误导证词 + 干草垛环境 flag
5. 解酒任务完成 → 大黄 ③（清醒）触达
6. 关键质询·向大黄索要干草垛 → 获得【干草垛】道具
7. 放置干草垛 + 首次到达谷仓屋顶（**笔记本翻至第二页**节点 · 重点埋点）
8. 获得假蛋石头 / 与小鸡对质完成（第二页"乌鸦"任务指针划掉演出）
9. 出示假蛋触发大黄 ④ offer 嗅觉追踪（获得进红顶屋动机 · 主章二入口）
10. 支付老鼠奶酪碎（次数与总量 · 同步触发第三页老鼠情报条目贴入）
11. 取得薄荷鱼 / 黑猫推开天窗
12. **跳入红顶屋天窗**（主线收束节拍 · 重点埋点 · 触发漫画演绎）
13. **漫画演绎完播**（手帐合本 · 重点埋点）
14. **进入二周目**（小火人放回红顶屋大门前 · 重点埋点）
15. NPC 结尾对话触发数（每 NPC 一次性，覆盖率统计）
16. 老鼠八卦二周目补完进度（剩余条目购入数 / 全购入）
17. 奶酪碎全收集（一周目 + 二周目刷新合并）
18. 尾声点击蜗牛（**二周目最终收束节拍**）

## 12.3 玩法验收标准

- **逻辑**：两章误导均有**硬回收**物证或空间条件；漫画演绎中的物证归位与 `05` 时间线、`漫画收束-红顶屋卧室情景.md` 无矛盾；二周目 NPC 结尾对话与一周目台词不冲突。
- **时长**：一周目首通 **约 1 h**；二周目释怀盘 + 全收集额外约 15–25 min（与 `01`/`02`/`07` 对齐）。
- **性能**：目标机型帧率与加载上限以工程规范为准；漫画演绎为整段一次性序列演出，加载策略与 3D 场景切换流畅度需单独验收。

## 12.4 测试范围注意

- 弱网、电话中断恢复、不同屏幕比例下「屋顶/猫门」可达性；误触质询边界。
