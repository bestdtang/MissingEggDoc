# 主线终点 · 红顶屋卧室（漫画收束情景）

> **载体**：主线在小火人跳入天窗瞬间淡出至手帐，**卧室内部不在 3D 中实时呈现**；真相与情感落点全部由**手帐内漫画分镜**承担（流程锚点见 `07-7.2` 步骤 9）。
> **主人出场**：仍以剪影 / 局部动作为主，与 `characters/主人.md`「缺席的温柔」一致。

## 漫画必达信息

玩家在攻顶前已通过门廊、窗缝、脚印、动物侧叙拼出"红顶屋 = 主人核心区 + 昨夜有大事"；本段漫画负责把**卧室内**一次性对齐，补足"温柔干预"的语义闭环。

## 核心物证（分镜必出镜）

| 镜头物证 | 叙事功能 |
|----------|----------|
| 桌上**保温箱**（铁丝固定、手工临时搭建） | 承接屋外"粗糙非成品"想象，落实人工孵化 |
| 箱底**天鹅绒软垫** | 与黑猫控诉、情绪线闭合 |
| 软垫上**画爱心的破碎蛋壳** | 真蛋已安全破壳的事实落锤 |
| 桌边**空水桶**（水迹、池塘加湿用途） | 与悲伤蛙"舀水"侧叙对齐 |
| **黄色保温灯**（可仍亮或刚熄的暖光残留） | 与"彻夜黄灯 / 刺眼黄灯"听觉线索和窗缝暖光对齐 |

## 二级物证（温柔人设 · 可分格插入）

| 物证 | 叙事功能 |
|------|----------|
| 床头桌摊开**《母鸡产后护理与孵化指南》**（折角、可有"破壳前温湿度"铅笔批注） | "观望两天再动手"与"蛋本已临近破壳"的书面证据 |
| 桌角**黑猫旧照片（裱框）** | 与"抽垫"形成刺痛对照，不洗白只补厚度 |
| 床头散放**数颗备用猫薄荷球** | 主人长期为黑猫备着的日常细节 |

## 演出节拍（编剧 / 分镜 checklist）

1. **小火人 + 黑猫**共同入画：黑猫第一次看见软垫去向与桌上保温装置（猫**没有原谅**主人，但好奇已满足）。
2. **健康小鸡**已在箱中或刚破壳不久的柔光状态，避免惊悚读法。
3. **温柔回巢**：主人剪影抱送小鸡回淑芬窝，母子团聚为温度高潮；无声或极简旁白，由分镜节奏带情绪。
