# 09 侦探笔记本

> **草稿**：下列仅为「主线会进手帐的内容」清单，版面、顺序、是否合并展示之后还会改。流程节拍见 [`07-关卡与内容结构`](./07-关卡与内容结构.md)；环境交互点见 [`13-玩家线索与交互点总表`](./13-玩家线索与交互点总表.md)。

## 9.1 形式（暂定）

- **规格**：横版 16:9；**主线 2 页 + 老鼠情报 1 页**。
- **气质**：手帐拼贴——文字、插图、大头针、线、胶带等；玩家不手填，新信息用预设演出写入。
- **作用**：线索汇总 + 任务引导一体；新内容写入后笔记本入口可出红点提示。

## 9.2 主线第一页（「锁定乌鸦之前」）

- **时段**：开场委托 → 玩家决定攀谷仓顶之前。
- **翻页**：攀谷仓顶、启动平台跳跃前一瞬翻到第二页；之后本页只读、不再追加。

**会出现的内容（与 `07` 步骤大致对应）**

- 淑芬委托便签（爱心蛋、失踪时间）——**步骤 一-1**
- 小鸡搞事碎片（多条拼贴感）——**步骤 一-2**
- 【水怪说】——**步骤 一-3**
- 【夜里有两脚兽来池塘舀水】——**步骤 一-4**
- 【夜里有黑色生物拍碎了水中玉盘】——**步骤 一-4**
- 【悲伤蛙身下绿垫有刺鼻薄荷味】——**步骤 一-4**（可先记，第二章薄荷鱼线回扣）
- 【大黄沉闷带哨音的鼾声】——**步骤 一-5**
- 【大黄目击：乌鸦叼蛋上屋顶】——**步骤 一-6**（误导，自动入簿）
- 【大黄身下压着干草垛】——**步骤 一-6**
- 【谷物泡水】（道具感）——**步骤 一-7**

**过程行（示例，可改）**

- 「鸡舍的小屁孩在掩盖什么？」→ 碎片够后划掉，旁注水怪烟幕
- 「池塘有水怪？」→ 问过悲伤蛙后划掉，旁注黑色恶魔 / 玉盘
- 「乌鸦偷了蛋？去谷仓顶看看」→ 攀顶前作任务指针
- 「大黄草垛 → 醒酒后再讨」→ 解酒后划掉，旁注去谷仓底垫脚

## 9.3 主线第二页（「进入红顶屋之前」）

- **时段**：谷仓顶对峙 → 跳入天窗之前（含假蛋揭露、第二章跑腿与谈判）。
- **跨页**：「乌鸦偷了蛋？」类指针在第二页起手划掉，旁注小鸡石头恶作剧。
- **收束**：跳入天窗后本页展开为漫画载体；卧室真相与物证以分镜填入（物证清单见 [`漫画收束-红顶屋卧室情景`](./漫画收束-红顶屋卧室情景.md)）；完播后合本。

**会出现的内容**

- 【屋顶白石头插图】——**步骤 一-10**
- 【干草垛】使用记录（若需留在簿上）——**步骤 一-8 / 一-9**
- 【大黄④ 嗅觉追踪证词】（雨靴、黄灯、红顶屋等口径）——**步骤 一-12**
- 红顶屋外勘察相关条目（大门、猫门、食碗、兽毛、空水桶、雨靴泥印等，随玩家交互逐项出现）——**步骤 二-2**
- 【沾蛙味极品猫薄荷鱼】——**步骤 二-5**
- 老鼠侧「薄荷鱼最后掉在池塘边」一类路径 / 交易备忘——**步骤 二-5**
- 漫画段：保温箱、软垫、爱心蛋壳、水桶、黄灯等揭晓物 + 二级温柔细节——**步骤 二-9**（与漫画文档对齐）

**过程行（示例，可改）**

- 「真蛋昨夜从鸡舍消失 → 红顶屋」
- 「大门撞不开 → 猫门 / 天窗」→ 摇树、黑猫开窗后更新措辞

## 9.4 第三页 · 老鼠兄弟情报

- 与主线两页分开；按购入顺序往下堆；可标档次，不显式真/假。
- 真情报若要做「撕下贴回主线」的演出，具体贴哪条主线内容由后续定稿再写。

## 9.5 翻页与回溯

- 可横向翻回看；主线停在当前最新页；老鼠页随买随增。

## 9.6 二周目

- 主线两页：漫画完成后封存，只读不追加。
- 老鼠第三页：仍可继续买八卦并追加。
- NPC 一次性结尾对话：不进笔记本。
- 可做软性进度提示（剩余八卦 / 结尾对话 / 奶酪），不强迫全收集。
