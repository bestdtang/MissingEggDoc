# 07 关卡与内容结构

## 7.1 内容层级命名

- **世界**：微缩农场（单主场景 + 关键子场景：鸡舍区、池塘、谷仓屋顶、红顶屋、大橡树、谷仓墙缝等）。
- **结构**：**起因** + **两章主线** + **尾声**；以「假蛋回收 → 红顶屋终局」为推理场主轴。

## 7.2 玩家游玩主线（与 `05` 时间线对照）

### 起因：焦急的母亲

混乱清晨开场：玩家操控小火人侦探，接 **淑芬** 委托——寻找画爱心且失踪的宝贝蛋；获得 **侦探笔记本**，踏入农场。

### 第一章：天降大锅

- **情绪**：悬疑寻凶 → 爆笑大乌龙。
- **流程摘要**：
  1. 被 **小鸡侦探团**（纸板墨镜）忽悠至池塘，遭遇 **悲伤蛙** 谜语。
  2. **小鸡口径 +（可选）老鼠收费闲话 + 宿醉大黄 + 黑猫树上暗示** 多源把玩家推向「屋顶白球」强目标。
  3. 找宿醉 **大黄** 核实，获「老实人强力证词」（时间线模糊）。
  4. **平台跳跃/场景互动**（推箱、干草垛等）登上**谷仓屋顶**对峙 **乌鸦**。
  5. 发现乌鸦守护的是 **画鬼脸的重石假蛋**；下楼与小鸡对质 → 招供「偷龙转凤」但坚称昨夜前已放回真蛋。
  6. **大黄**出示假蛋后清醒，提供 **雨靴声/黄灯/红顶屋 + 求大橡树黑猫** 的硬指向（承接第二章）。
- **章末认知**：水怪与乌鸦线为乌龙；**真蛋在昨夜从鸡舍「凭空」消失，需进人类屋**。

### 第二章：温柔的反转

- **情绪**：跨服聊天收束 + 温柔治愈。
- **流程摘要**：
  1. 红顶屋大门紧闭 → 通过 **老鼠** 口碑或环境可读信息，得知仅 **黑猫** 能进人类屋；**大橡树**下解锁黑猫交互（条件见 `characters/黑猫.md`）。
  2. 黑猫索要 **老鼠偷走的薄荷玩具** → **跑腿链**：谷仓墙缝老鼠要价「三块奶酪碎」→ 情报指向池塘；**悲伤蛙** 屁股下抽出 **沾蛙味的薄荷鱼**；交黑猫 → 猫「醉」开**窗**。
  3. 卧室内物证总览：保温箱 + **黑猫天鹅绒垫** + **黄灯** + **池塘水**。
  4. 健康小鸡刚破壳（或等价温柔落点）。
- **结案语义**：主人心疼淑芬焦虑，半夜温柔介入人工孵化；无阴谋。

### 尾声

玩家合笔记本出屋；路遇 **闪电蜗牛** 慢速补刀一句（见 `06`）。

## 7.3 难度与内容量规划

- **目标总时长**：**约 1–1.7 h**（占位，与甲方对齐后填死数）。
- **难度**：休闲向；硬卡关依赖 **物证链/空间位置**（屋顶等）而非操作极限。
- **内容量**：动物 NPC 以 `06` 为准（**约 8 组**）；主案件仍为 1。
