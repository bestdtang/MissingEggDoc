# 07 关卡与内容结构

## 7.1 文档定位

本文记录玩家主线流程：游玩顺序、情节节点、卡点与质询锁。质询锁指对话选项的出现条件；条件不足时，选项不显示。

环境点位置与反馈见 [13-玩家线索与交互点总表](./13-玩家线索与交互点总表.md)。场景布置见 [08-场景与布置](./08-场景与布置.md)。事实时间线见 [05-叙事与世界观](./05-叙事与世界观.md) 与 [10-NPC主时间线](./10-NPC主时间线.md)。

## 7.2 第一章：天降大锅

**步骤 一-1，淑芬委托。** 玩家在鸡舍侧接到淑芬委托：今早，窝里那颗画了爱心的蛋不见了。此处无质询锁。

**步骤 一-2，鸡舍周边调查。** 玩家调查鸡舍附近的小鸡搞事痕迹：短木炭、散落羽毛碎屑、小鸡身后偷听点。收集任意两条后，解锁小鸡关键质询。

**步骤 一-3，质问小鸡侦探团。** 条件满足后，出现选项「你们到底在搞什么鬼？」。小鸡抛出水怪说，把玩家指向池塘。获得水怪说后，池塘悲伤蛙的水怪质询开放。

**步骤 一-4，池塘与悲伤蛙。** 未持水怪说时，悲伤蛙只提供默认独白；玩家可观察其身下绿垫。持有水怪说后，出现质询「水怪是不是真的？昨晚池塘到底发生了什么？」。悲伤蛙给出两条证词：夜里有两脚兽来池塘舀水；夜里有黑色生物拍碎水中玉盘。

**步骤 一-5，狗窝大黄初醒。** 玩家靠近狗窝触发大黄鼾声。交互后，大黄从完全睡着进入半睡半醒阶段。

**步骤 一-6，大黄半睡半醒。** 大黄复述清晨目击：乌鸦把淑芬的蛋叼上谷仓屋顶。玩家可观察到大黄身下压着干草垛。此时大黄仍未清醒，索要干草垛选项不出现。

**步骤 一-7，解酒。** 玩家从鸡舍水槽边取得老谷物泡水，交给大黄。解酒提示来源为小鸡、淑芬或老鼠一档情报任一条。交付后，大黄进入清醒阶段。

**步骤 一-8，索要干草垛。** 玩家已观察干草垛，且大黄已清醒时，出现索要干草垛选项。完成后获得干草垛。

**步骤 一-9，攀上谷仓。** 玩家在谷仓底部放置干草垛，打开通往谷仓屋顶的平台跳跃路线。攀爬途中可经过谷仓午睡点、烧焦稻草 / 皮毛、动物爪印等观察点；其中午睡点与烧焦痕迹是第二章薄荷鱼线的前置条件。

**步骤 一-10，谷仓顶乌鸦对峙。** 玩家登上谷仓屋顶，看见乌鸦巢中供着一颗画鬼脸和爱心的白色石头。乌鸦对峙台词须交代：黑猫在 T-2 下午上过谷仓顶。

**步骤 一-11，质问小鸡。** 玩家完成谷仓顶观察后，可向小鸡出示屋顶所见。小鸡招认假石头与换蛋经过，并回收水怪说。

**步骤 一-12，质问大黄。** 玩家完成谷仓顶观察，且大黄已清醒后，可向大黄出示屋顶所见。大黄用嗅觉确认蛋味指向红顶屋，任务点迁至红顶屋外。

第一章结束时，玩家已确认屋顶是石头假蛋，真蛋昨夜从鸡舍消失。下一步调查红顶屋。

## 7.3 第二章：温柔的反转

**步骤 二-1，抵达红顶屋外。** 玩家随大黄来到红顶屋外。大黄二次确认蛋味在屋内。大门无法直接进入。

**步骤 二-2，红顶屋外勘察。** 玩家调查红顶屋外环境：紧闭大门、猫门、食碗、深色兽毛、门旁空水桶、雨靴泥脚印。下树线索为大门、猫门、食碗、兽毛四项，完成任意两项后，解锁摇树召唤黑猫。案情汇报物证为空水桶与雨靴泥脚印，两项必须齐备；雨靴印为夜里踩出，方向朝鸡窝。物证缺一时，黑猫汇报线停在追问处，玩家需回屋外补全。

**步骤 二-3，摇树召唤黑猫。** 下树线索满足后，大黄处出现摇树选项。触发后，黑猫从大橡树下来，停留在树下。

**步骤 二-4，树下谈判。** 黑猫掌握猫门与天窗路径。玩家需完成案情汇报线与薄荷鱼线，两线无先后要求；两线都完成后，黑猫才会开窗。

**步骤 二-5，薄荷鱼线。** 玩家须已在第一章攀谷仓途中发现谷仓午睡点与烧焦痕迹，才可质询黑猫丢失物。黑猫承认与老鼠有关后，玩家到红顶屋墙缝找老鼠兄弟，支付八块奶酪碎，得知薄荷鱼最后掉在池塘边。玩家回池塘向悲伤蛙索要身下绿垫，取得沾蛙味的猫薄荷鱼并交还黑猫。交还后，老鼠二档后组情报开放。

**步骤 二-6，案情汇报线。** 玩家向黑猫汇报事实：淑芬今早空窝且无打斗痕迹；大黄确认蛋味仍在红顶屋；悲伤蛙曾说夜里有两脚兽到池塘；门旁空水桶带池塘泥沙；雨靴泥脚印朝鸡窝。空水桶与雨靴印缺一时，汇报无法完成。

**步骤 二-7，汇合开窗。** 案情汇报线完成，且薄荷鱼已交还后，黑猫钻猫门进屋，从内推开屋顶天窗。揭穿黑猫「早知石假蛋」的早触发条件见 [黑猫](./characters/黑猫.md)。

**步骤 二-8，登顶入屋。** 玩家沿红顶屋外平台跳跃路线登上屋面，从天窗跳入。卧室不作为 3D 可探索关卡，跳入后进入漫画收束演出。

**步骤 二-9，漫画收束。** 漫画分镜呈现室内真相：保温箱、软垫、蛋壳、农场主介入、小鸡回巢与淑芬团聚。室内分镜清单见 [漫画收束-红顶屋卧室情景](./漫画收束-红顶屋卧室情景.md)。

**步骤 二-10，二周目释怀盘。** 漫画结束后，玩家回到红顶屋门前。二周目开放 NPC 结尾对话、老鼠情报补买与奶酪碎刷新。无主线硬卡关。

**步骤 二-11，揭穿黑幕晚触发。** 若玩家未在开窗前追问黑猫，可在漫画收束后回大橡树下追问「早知石假蛋」对白。是否与二周目黑猫释怀台词互斥，以 [黑猫](./characters/黑猫.md) 为准。

第二章主干为：红顶屋外勘察 → 摇树召唤黑猫 → 案情汇报线与薄荷鱼线并行 → 黑猫开窗 → 登顶入屋 → 漫画收束 → 二周目。

## 7.4 尾声与内容量

鸡舍外宽叶上的闪电蜗牛 Flash 负责尾声慢语速节拍，详见 [闪电蜗牛-Flash](./characters/闪电蜗牛-Flash.md)。

一周目主线约一小时；二周目释怀与补情报约十五到二十五分钟。主线无失败惩罚。农场可对话动物共八组，主案件单线。
