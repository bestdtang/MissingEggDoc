# 07 关卡与内容结构

## 7.1 内容层级命名

- **世界**：微缩农场（单主场景 + 关键子场景：鸡舍区、池塘、谷仓屋顶、红顶屋、大橡树、谷仓墙缝等）。
- **结构**：**起因** + **两章主线** + **尾声**；以「假蛋回收 → 红顶屋终局」为推理场主轴。

## 7.2 玩家游玩主线（与 `05` 时间线对照）

### 起因：焦急的母亲

混乱清晨开场：玩家操控小火人侦探，接 **淑芬** 委托——寻找画爱心且失踪的宝贝蛋；获得 **侦探笔记本**（**手帐式 · 横版 16:9 · 主线 2 页 + 老鼠情报 1 页**，详见 `09-9.2`），淑芬委托便签自动贴入第一页左上槽位，踏入农场。

> **淑芬全程可选回访**：玩家在第一章调查途中、小鸡认罪后、第二章锁定主人方向后、揭示后，均可回鸡舍与淑芬交谈。她的对话随进度更新，不提供硬线索，但每次都为情感线补充温度（详见 `characters/淑芬.md`）。第二章前往鸡舍取谷物泡水时，玩家必然路过淑芬——是自然的强制情感节拍。

### 第一章：天降大锅

- **情绪**：悬疑寻凶 → 爆笑大乌龙。
- **流程摘要**：
  1. 被 **小鸡侦探团**（纸板墨镜）忽悠至池塘，遭遇 **悲伤蛙** 谜语。
  2. **小鸡口径 +（可选）老鼠收费闲话 + 宿醉大黄 + 黑猫树上暗示** 多源把玩家推向「屋顶白球」强目标。
  3. 找宿醉 **大黄** 核实，获「老实人强力证词」（时间线模糊）。
  4. **平台跳跃/场景互动**（推箱、干草垛等）登上**谷仓屋顶**对峙 **乌鸦**——攀顶前一瞬笔记本**自动翻至第二页**（第一页"锁定乌鸦之前"留存为已写就状态，可回溯不可追加）。
  5. 发现乌鸦守护的是 **画鬼脸的重石假蛋**；下楼与小鸡对质 → 招供「偷龙转凤」但坚称昨夜前已放回真蛋。**石头假蛋拓片**贴入第二页左上槽位；第一页"乌鸦偷了蛋？"任务指针被**划掉**，旁注「画鬼脸的石头！是小鸡的恶作剧」（跨页回响）。
  6. **大黄**出示假蛋后清醒，提供 **雨靴声/黄灯/红顶屋 + 求大橡树黑猫** 的硬指向（承接第二章），证词便签贴入第二页中上槽位。
- **章末认知**：水怪与乌鸦线为乌龙；**真蛋在昨夜从鸡舍「凭空」消失，需进人类屋**。

### 第二章：温柔的反转

- **情绪**：寻线 → 碰壁 → 谈判 → 跑腿 → 攻顶 → 温柔治愈。
- **章节结构**：**串行主链 + 跑腿子链**。第一章末大黄给出的"先去找黑猫"压缩为单一情绪台词，**黑猫的首次现身被推后到红顶屋外卡门 + 大黄摇树召唤之后**，让"找方向 → 碰壁 → 呼唤 → 谈判"形成完整戏剧曲线。

#### 开场：嗅觉恢复线

1. **大黄（狗窝）**嗅觉未恢复——情绪低落，**只说"鼻子还是死的，得喝点鸡舍那边的老谷物泡水"**，**不直接点名黑猫**（章节入口提示压到步骤 7 由大黄统一给出）。
2. 玩家在大黄狗窝旁发现**旧木桶**（环境物件 · 桶底谷物渣 + 谷糠气味），结合大黄台词推出去**鸡舍**取解酒方案。
3. **鸡舍途中路过淑芬**（强制情感节拍，详见 `characters/淑芬.md` 阶段 1）→ **小鸡侦探团**在鸡舍门口（愧疚待命，不主动靠近）→ 玩家主动搭话询问大黄解酒方法 → 小鸡指向**水槽边的【谷物泡水】**。
4. 玩家把【谷物泡水】交大黄 → **嗅觉恢复**，神情骤然郑重：**「淑芬的蛋——气味从那栋红顶屋飘出来的，就是那里。我跟你过去守着，离近一点。」** → **大黄任务点从狗窝迁移至红顶屋外**。

#### 主链：红顶屋外卡门

5. 玩家前往**红顶屋外**，遇见已迁移到此的大黄 → 大黄抬头嗅 + 二次确认：**「就在屋子里。错不了。」**
6. 玩家在红顶屋外探查环境（**自由探索 · 第二页"红顶屋外勘察平面图"槽位逐项拼贴**）：
   - **紧闭大门**（玩家推不动）；
   - **暖黄灯光**从窗缝渗出；
   - **专为黑猫凿刻的精美猫门**（边缘可见手工凿痕）——小火人侦探**钻不过去**；
   - 屋顶疑似有**天窗轮廓**（埋伏笔 · 步骤 14 攻顶目标）。
7. 玩家回到大黄处汇报"屋子进不去，但有一个修葺精美的猫门" → **大黄抬头看大橡树**：**「……树上那位倒是钻得过去。」** 大黄走到大橡树下，**用爪子撩拨树干 / 摇了摇树**：**「猫大爷！下来！这事得你出马！」**

#### 黑猫首次现身：薄荷鱼谈判

8. 大橡树枝头**黑猫现身**（第一章树上独白至此结束，第二章首次可交互）→ 黑猫低沉警告：「滚开！没看到本喵正因感情欺骗疗伤吗？……」（详见 `characters/黑猫.md`"大橡树下唤醒"）。
9. 玩家请求开门 → 黑猫提条件：**先夺回老鼠偷走的【极品薄荷鱼玩具】**，方肯出力。

#### 跑腿子链：薄荷鱼回收

10. 谷仓墙缝**老鼠兄弟** → 付「**八块奶酪碎**」（全场最贵，唯一硬必买）→ 获薄荷鱼下落情报；同时**解锁老鼠二档后组 6 条深夜情报**（3 真 + 3 bullshit · 盲盒抽取 · 命中率 50%，详见 `characters/老鼠兄弟.md`）。
11. 池塘 → 从**悲伤蛙**屁股下取回**沾蛙味的薄荷鱼**。

#### 汇合：案情汇报 + 推开天窗

12. 返回大橡树（大黄仍在树下）：将薄荷鱼交黑猫 → 猫开口倾听。
13. 玩家**只列事实清单、不下结论**地汇报案情。**硬必两条**（淑芬今早失蛋 + **大黄鼻子实时确认蛋仍在红顶屋**）+ **玩家手中持有的可选拼接条目**（老鼠雨靴/黄灯·若买到 / 悲伤蛙舀水·若交互过 / 其他 NPC 对话扩写后的额外旁证）→ **激活黑猫的首次顿悟**：黑猫半睡半醒间记得"主人昨晚深夜出门一趟 + 回来后摆弄桌上装置 + 夺垫 + 开黄灯"这一串怪行——但**他完全没看见过蛋**，是玩家的事实汇报让他第一次把"主人怪行"和"偷蛋"挂上钩 → 大黄从树下撩猫（"被当面骗了一手 + 冬天不让你蹭我狗窝"）配合好奇 + 自尊 + 物权三重驱动 → 黑猫"亲眼进屋看个清楚那家伙到底搞了什么名堂 + 顺便看一眼那颗破蛋"为由下树 → **钻猫门进屋，从内啪嗒推开屋顶天窗**——这声响是玩家攻顶的目标音效。黑猫此刻仍**不知道蛋的具体位置、桌上是什么装置、软垫去向、主人为何要这么做**——这些**全部保留至步骤 15 红顶屋谜底一次性揭晓**。
    - **可选高光分支（早触发）**：玩家若已从老鼠一档 B 组 #10 获悉"黑猫 T-2 夜上过乌鸦屋顶"，此处对话菜单多出一项「你那天晚上去屋顶看到什么了？」→ 黑猫承认"涂漆石头一眼就认出来了 / 让侦探多转几圈相当公平"，**与第一章树上伏笔独白"傻子在炫耀，瞎子在到处问"形成回响**（伏笔句已在玩家首次靠近大橡树时硬触发首播，玩家必听过；详见 `characters/黑猫.md`"揭穿黑幕"与"树上的神秘声音"触发规则）。

#### 攻顶：平台跳跃挑战 · 主线收束节拍

14. 玩家通过红顶屋外的**平台跳跃挑战**登上红顶屋顶面 → 在天窗边沿做出**跳跃下降**的动作——**主线在小火人侦探跳入天窗的那一瞬间收束**。
    - **关卡定位**：第二章末尾硬空间闸，与第一章谷仓屋顶平台跳跃形成首尾呼应；难度休闲向、失败惩罚轻。
    - **跳跃路径与卡点**：待美术与关卡定稿；可参考路径如外墙雨水桶 → 旁侧木箱堆叠 → 屋檐 → 屋脊 → 天窗。
    - **演出衔接**：跳入瞬间画面**淡出至手帐特写**——笔记本在镜头前自动展开，进入步骤 15 的**漫画演绎**段。**卧室内部不在 3D 场景中实时呈现**，谜底揭晓全部由漫画承担。

#### 手帐内的漫画演绎：真相揭晓 + 小鸡回巢

15. 手帐展开为漫画载体，**整段卧室揭晓 + 健康小鸡回巢以漫画分镜演绎**——玩家以观众视角看完小火人侦探 + 黑猫共同进入卧室、发现真相、健康小鸡被温柔送回淑芬窝的完整段落。
    - **核心物证（5 件 · 必出现的漫画镜头）**：桌上**保温箱**（铁丝固定 · 手工搭建）+ 箱底铺**黑猫天鹅绒软垫** + 软垫上**画爱心的破碎蛋壳** + 桌边**空水桶**（池塘水加湿用）+ **黄色保温灯**仍亮着。
    - **二级物证（3 件 · 温柔人设细节，详见 `characters/主人.md`）**：床头桌摊开的**《母鸡产后护理与孵化指南》**（有折角、批注）+ 桌角**黑猫旧照片（裱框）** + 床头散放**数颗备用猫薄荷球**（主人长期为黑猫备着）——可分入若干小分镜，作为漫画质感的温度细节。
    - **黑猫的归位（漫画分镜）**：黑猫"那夜怪行 ↔ 主人温柔孵化"在脑内一次性接通——软垫去向、桌上装置、黄灯用途、蛋的位置全部归位；黑猫**没有原谅**主人但好奇心已被满足（与 `characters/黑猫.md`"卧室物证呼应"一致）。
    - **温柔回巢（漫画末段 · 情感高潮）**：健康小鸡已破壳 + 主人**温柔将小鸡送回淑芬窝**，淑芬与小鸡团聚的画面作为本作温度落点。**主人仍只以剪影 / 局部动作出场**（手、身影），不正面亮相，与"缺席的核心角色"定位保持一致；具体分镜与无声台词由美术 / 编剧定稿。
    - **手帐与漫画的关系**：第二页此前预留的"揭晓填空"槽位**随漫画镜头节奏陆续被分镜插画填入**——**漫画本身即第二页揭晓段的"分格画稿"**。漫画完播后，第二页全部写就，整本主线两页完结，"合上手帐"作为漫画收束的仪式节拍。

#### 二周目：释怀盘 · 全收集 · 全 NPC 结尾对话

16. 手帐合上后，小火人侦探**被放置在红顶屋大门前**——主线已通关，进入**二周目"释怀盘"阶段**。
    - **农场状态变化**：主案件已破，全农场 NPC **进入"结尾状态"**；每个可交互 NPC 都会有**一段一次性结尾对话**（释怀的、调侃的、meta 的，结合各自性格），与一周目调查阶段台词**完全分离**。触发条件为"二周目期间首次靠近"——各 NPC 结尾对话详见 `characters/` 对应详档"二周目结尾对话"段（首轮为占位 stub，待美术 / 编剧定稿）。
    - **老鼠兄弟 · 八卦补完**：盲盒情报清单仍开放，玩家可继续付费抽取一周目未购入的剩余条目（一档 A/B 未抽 + 二档前组 / 后组未抽，含 bullshit / 真情报混抽，详见 `characters/老鼠兄弟.md`）；薄荷鱼分支已完成、不再出现，回访可触发彩蛋台词。
    - **奶酪碎二周目刷新**：场景内**新刷新一批奶酪碎**（覆盖原拾取点位 + 新增点位），总量补足"买光所有剩余八卦 + 留出余裕"的水平；具体刷新策略与点位待美术 / 关卡定稿，**设计原则**：让全收集玩家在合理探索半径内拿满，不必强行重玩主线（详见 `03-3.5`、`characters/老鼠兄弟.md` "奶酪碎经济与拾取设计 · 二周目刷新"段）。
    - **闪电蜗牛 Flash**：仍在叶尖等待玩家最后一次点击，慢语速 meta 补刀作为**二周目尾声节拍**（详见 `characters/闪电蜗牛-Flash.md` 阶段 4）。
    - **手帐在二周目的状态**：主线两页**封存为已写就**，可回溯不可追加；老鼠情报第三页**仍可继续追加新购入条目**；NPC 结尾对话**不进笔记本**——它们是释怀盘的演出节拍，不是新增情报（详见 `09-9.2` 二周目手帐状态段）。
    - **退出 / 收束**：二周目无硬卡关、无失败惩罚；玩家可随时退出（保留进度），亦可一气把全收集做完。

17. **可选高光分支（晚触发）**：未在步骤 13 触发"黑猫知情隐瞒"分支的玩家，可在二周目阶段回大橡树下追问黑猫，由黑猫亲口承认（兜底路径）。

- **结案语义**：主人心疼淑芬焦虑，半夜温柔介入人工孵化；无阴谋。
- **串行说明**：撤销之前的"两条并行任务线"方案，改为单链串行 — 嗅觉线必先于黑猫现身，黑猫现身后才启动薄荷鱼跑腿。这条曲线把"找方向 → 碰壁 → 呼唤 → 谈判"的戏剧节奏拉满，且消除了"玩家在不知道为什么要拿薄荷鱼的时候已经被卡在墙缝外"的旧版尴尬。
- **黑猫追问双触发说明**：步骤 13 早触发奖励推理流玩家（两条独立事实自己拼接 → 主动发起追问）；步骤 17 晚触发为休闲玩家兜底（二周目阶段回树下追问），二者对白相同、但早触发额外多一句"我在树上不是说过吗"的伏笔回响（伏笔句首次必触发，所有玩家均有此回响基础）。

### 尾声

- 一周目主线在漫画末段"小鸡回巢、淑芬团聚"达到温柔情绪落点；手帐合上 = 案件了结的仪式收束。
- 二周目作为**收集 / 释怀盘**展开，玩家可补齐老鼠八卦、走完每个 NPC 的结尾对话、完成奶酪碎全收集；**闪电蜗牛 Flash** 极慢语速补刀作为最终收束（与 `06`/`characters/闪电蜗牛-Flash.md` 一致）。

## 7.3 难度与内容量

- **目标总时长**：**一周目主线约 1 h**；二周目释怀盘 + 全收集额外约 15–25 min（视玩家探索深度）。
- **难度**：休闲向；硬卡关依赖 **物证链/空间位置**（屋顶等）而非操作极限；二周目无硬卡关。
- **内容量**：动物 NPC 以 `06` 为准（**8 组**）；主案件 1 条；二周目结尾对话每 NPC 一段（占位见 `characters/` 各详档）。
