# 07 关卡与内容结构

## 7.1 文档定位

本文记录玩家主线流程：游玩顺序、情节节点、卡点与质询锁。质询锁指对话选项的出现条件；条件不足时，选项不显示。

环境点位置与反馈见 [13-玩家线索与交互点总表](./13-玩家线索与交互点总表.md)。场景布置见 [08-场景与布置](./08-场景与布置.md)。事实时间线见 [05-叙事与世界观](./05-叙事与世界观.md) 与 [10-NPC主时间线](./10-NPC主时间线.md)。

## 7.2 第一章：天降大锅

**步骤 一-1，淑芬委托。** 玩家在鸡舍侧接到淑芬委托：今早，窝里那颗画了爱心的蛋不见了。此处无质询锁。

**步骤 一-2，鸡舍周边调查。** 玩家调查鸡舍附近的小鸡搞事痕迹：短木炭、散落羽毛碎屑、小鸡身后偷听点。收集任意两条后，解锁小鸡关键质询。

**步骤 一-3，质问小鸡侦探团。** 条件满足后，出现选项「你们到底在搞什么鬼？」。小鸡抛出水怪说，把玩家指向池塘。获得水怪说后，池塘悲伤蛙的水怪质询开放。

**步骤 一-4，池塘与悲伤蛙。** 未持水怪说时，悲伤蛙只提供默认独白；玩家可观察其身下绿垫。持有水怪说后，出现质询「水怪是不是真的？昨晚池塘到底发生了什么？」。悲伤蛙给出三条证词：大前天有浑圆之物被人推入池塘洗涤；夜里有两脚兽来池塘舀水；夜里有黑色生物拍碎水中宝珠。

**步骤 一-5，狗窝与谷仓旁。** 玩家调查**狗窝**（E22）发现空窝——只作为"保安不在岗"的氛围提示，不入笔记本。**大黄醉倒在谷仓墙根旁**（未回窝）；玩家靠近谷仓旁大黄处可闻鼾声，交互后从完全睡着进入半睡半醒。顺序不限：可先查空窝再循鼾声找狗，也可直接循鼾声到谷仓旁找大黄。

**步骤 一-6，大黄半睡半醒。** 大黄复述清晨目击：乌鸦把淑芬的蛋叼上谷仓屋顶。玩家在**谷仓旁**找到醉倒的大黄与被他压住的短木梯（E04），笔记本记入【大黄身下像压着一架木梯】；玩家亦可前往谷仓入口小围墙（E06），笔记本记入【谷仓入口有小围墙，没有木梯翻不过去】。E04 与 E06 顺序不限，但作用不同：E04 是醒酒前置，E06 是后续借梯前置。若玩家已知道 E06 小围墙缺梯，再回到半睡半醒的大黄处，才会追问他能不能挪开短木梯；大黄迷糊拒绝。

**醒酒锁（一-7 前置）。** 须持有【大黄身下像压着一架木梯】并取得【谷物泡水】，即可交给大黄醒酒；不要求玩家已经发现 E06 小围墙缺垫脚物。三源「怎么叫醒」提示仍可获取泡水位置。

**步骤 一-7，叫醒大黄。** 从鸡舍水槽边取得老谷物泡水并交给大黄；提示来源为小鸡、淑芬或老鼠一档情报任一条。交付后，大黄进入清醒阶段，短木梯完整露出。若此时玩家尚未调查 E06，小围墙缺垫脚物的需求尚未成立，不出现索要短梯选项。

**步骤 一-8，索要木梯。** 大黄已清醒，且玩家已持【谷仓入口有小围墙，没有木梯翻不过去】时，出现索要短梯选项。E06 可在醒酒前或醒酒后发现；若醒酒后才发现 E06，回到大黄处再触发「把短梯给我」对话。完成后获得木梯持有状态。

**步骤 一-9，进入谷仓平台跳跃。** 玩家在谷仓入口小围墙架设短木梯，翻过围墙后进入通往谷仓屋顶的平台跳跃路线。攀爬途中可经过谷仓午睡点、烧焦稻草 / 皮毛等观察点；接近乌鸦巢前可发现动物爪印。其中午睡点与烧焦痕迹是第二章薄荷鱼线的前置条件；动物爪印为辅助线索，只补强「不只乌鸦上来过」的怀疑。路线上设置两处乌鸦远程放狠话触发点，只作节奏反馈，不写入笔记本。未架设短木梯前，谷仓平台跳跃关卡不可开始。

**步骤 一-10，谷仓顶乌鸦对峙。** 玩家登上谷仓屋顶，看见乌鸦巢中供着一颗画鬼脸和爱心的白色石头；巢边还夹着鸡羽毛、狗毛、鼠毛与一撮不似乌鸦羽毛的黑色细毛。乌鸦对峙台词须交代：黑猫在 T-2 下午上过谷仓顶。

**步骤 一-11，质问小鸡。** 玩家完成谷仓顶观察后，可向小鸡出示屋顶所见。小鸡招认假石头与换蛋经过，并回收水怪说。

**步骤 一-12，质问大黄。** 玩家完成谷仓顶观察，且大黄已清醒后，可向大黄出示屋顶所见。大黄用嗅觉确认蛋味指向红顶屋，任务点迁至红顶屋外。

第一章结束时，玩家已确认屋顶是石头假蛋，真蛋昨夜从鸡舍消失。下一步调查红顶屋。

## 7.3 第二章：温柔的反转

**步骤 二-1，抵达红顶屋外。** 玩家随大黄来到红顶屋外。大黄二次确认蛋味在屋内。玩家先调查紧闭大门，确认正门无法进入；大黄随后提示玩家想想别的办法进去。首次进入墙缝外围 **E39** 范围时，老鼠兄弟强制吆喝导向墙缝（**0-A**，见老鼠树状）。

**步骤 二-2，红顶屋外勘察。** 正门碰壁后，玩家调查红顶屋外环境：猫门、食碗、深色兽毛、门旁空水桶、雨靴泥脚印；也可在大橡树根发现抓痕。下树线索为猫门、食碗、兽毛、树根抓痕四项，完成任意两项后，玩家回大黄处追问猫门主人，解锁摇树召唤黑猫。若玩家已在乌鸦巢见过黑色细毛，红顶屋外兽毛会二次触发「乌鸦真的长毛吗？」的自问。案情汇报物证为空水桶与雨靴泥脚印，两项必须齐备；雨靴印为夜里踩出，方向朝鸡窝。物证缺一时，黑猫汇报线停在追问处，玩家需回屋外补全。

**步骤 二-3，摇树召唤黑猫。** 下树线索满足后，大黄处出现摇树选项。触发后，黑猫从大橡树下来，停留在树下。

**步骤 二-4，树下谈判。** 黑猫掌握猫门与二层窗路径。玩家需完成案情汇报线与薄荷鱼线，两线无先后要求；两线都完成后，黑猫才会开窗。

**步骤 二-5，薄荷鱼线。** 玩家须已在第一章攀谷仓途中发现谷仓午睡点与烧焦痕迹，才可质询黑猫丢失物。黑猫承认与老鼠有关后，薄荷鱼线正式开启，池塘悲伤蛙处的「索要绿垫」选项以两条路径触发：

- **自主路径**：玩家凭【薄荷鱼线】+【悲伤蛙身下绿垫有奇异草本甜气】自行拼合，笔记本生成【推测：草本甜气绿垫就是那件猫玩具？】后可直接到蛙处尝试。无需支付老鼠奶酪。
- **老鼠路径**：玩家到红顶屋墙缝找老鼠兄弟，支付八块奶酪碎，得知薄荷鱼最后掉在池塘边一带；老鼠看在大单份上，当场开放二档后组情报（写 `Mouse_PremiumPoolUnlocked`）。

两条路径均到达悲伤蛙处，触发**三轮对暗号谜题**（每轮三选一，测试玩家是否接上蛙的存在主义频道）。三轮全对，蛙主动推出薄荷鱼；任意选错蛙说「你不懂」终止对话：

- **自主成功、未付老鼠**：取鱼后回墙缝选「你不是说东西还在手里吗？」→ 老鼠 **1-E-C** 质问；因被骗，**免费**解锁后组贵情报池（`Mouse_PremiumPoolUnlocked`，不扣 8 块、无 **1-G** 兜底资格）。
- **自主失败、未付老鼠**：蛙 **3-B** 失败后回墙缝 hub「那个青蛙，有没有什么办法搞定？」→ **1-G** 付 8（**1-G-P** 开后组池 + **1-G-T** 口诀）→ 蛙 **3-D**。
- **老鼠路径失败**：已付 8 块的玩家蛙 **3-A** 失败后回墙缝 hub「青蛙那边有点麻烦。」→ **1-G-A** 免费「排第七」（含在 8 块内）。

取得沾蛙味的猫薄荷鱼后交还黑猫，完成薄荷鱼线。

**步骤 二-6，案情汇报线。** 玩家向黑猫汇报事实：淑芬今早空窝且无打斗痕迹；大黄确认蛋味仍在红顶屋；悲伤蛙曾说夜里有两脚兽到池塘；门旁空水桶带池塘泥沙；雨靴泥脚印朝鸡窝。空水桶与雨靴印缺一时，汇报无法完成。

**步骤 二-7，汇合开窗。** 案情汇报线完成，且薄荷鱼已交还后，黑猫钻猫门进屋，从内推开二层窗。揭穿黑猫「早知石假蛋」的早触发条件见 [黑猫](./characters/黑猫.md)。

**步骤 二-8，登顶入屋。** 玩家沿红顶屋外平台跳跃路线登上屋面，从二层窗跳入。黑猫开窗后，路线上设置两处黑猫屋内远程放话触发点（**E37** / **E38** → 黑猫 **2-F**），只作节奏反馈，不写入笔记本。卧室不作为 3D 可探索关卡，跳入后进入漫画收束演出。

**步骤 二-9，漫画收束。** 漫画分镜呈现室内真相：保温箱、软垫、蛋壳、农场主介入、小鸡回巢与淑芬团聚。室内分镜清单见 [漫画收束-红顶屋卧室情景](./漫画收束-红顶屋卧室情景.md)。

**步骤 二-10，二周目释怀盘。** 漫画结束后，玩家回到红顶屋门前。二周目开放 NPC 结尾对话、老鼠情报补买与奶酪碎刷新。无主线硬卡关。

**步骤 二-11，揭穿黑幕晚触发。** 若玩家未在开窗前追问黑猫，可在漫画收束后回大橡树下追问「早知石假蛋」对白。是否与二周目黑猫释怀台词互斥，以 [黑猫](./characters/黑猫.md) 为准。

第二章主干为：红顶屋外勘察 → 摇树召唤黑猫 → 案情汇报线与薄荷鱼线并行 → 黑猫开窗 → 登顶入屋 → 漫画收束 → 二周目。

## 7.4 尾声与内容量

鸡舍外宽叶上的闪电蜗牛 Flash 负责尾声慢语速节拍，详见 [闪电蜗牛-Flash](./characters/闪电蜗牛-Flash.md)。

一周目主线约一小时；二周目释怀与补情报约十五到二十五分钟。主线无失败惩罚。农场可对话动物共八组，主案件单线。
