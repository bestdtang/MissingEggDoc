# 05 叙事与世界观

## 5.1 世界观概要

- **空间**：一座高度风格化的 **3D Lowpoly 微缩农场**（「盒子盆景」式精致感），动物拟人社会规则松散、充满抓马。
- **案件**：劳模母鸡 **淑芬** 的最后一颗「爱心蛋」在清晨失踪；全农场在**视角受限**下各自误读，交织成「大忽悠局」。
- **真相内核**：无阴谋——**农场主**因心疼淑芬焦虑，半夜温柔将蛋带回卧室人工孵化；动物与玩家的惊悚推理多为**跨服聊天**叠加。

## 5.2 与「小火人世界」IP 的结合方式

- **玩家身份**：**小火人侦探**；道具 **侦探笔记本** 为系统级核心——**手帐式横版 16:9**，主线 2 页 + 老鼠情报 1 页，固定槽位承载文字 / 插图 / 大头针 + 线 / 胶带等元素，兼具**线索汇总 + 任务引导**双重作用（详见 `09-9.2`）。
- **平台结合**：行走与交互贴合**抖音小火人原生系统**；叙事上预留**高能切片点**（详见 `07`/`09`）。
- **边界**： comedy 可夸张，但终局落点需温暖、可共情，避免真正恶心或低俗羞辱「劳动者动物」（淑芬线保护）。

## 5.3 叙事基调与禁忌

- **语言基调**：冷幽默、跨频道聊天、官方吐槽。
- **禁忌（建议）**：不将儿童向笑点建立在残酷动物伤害结局上；**惨剧误会**仅限误会层，须有喜剧回收（以假蛋乌龙 + 卧室温柔收束为主）。

## 5.4 主线结构（玩家线 × 真相线）

- **玩家线**：见 `07` **两章主线**流程与尾声。
- **真相线**：见下文 **5.6 上帝视角真相时间线**（策划守密用；对白与演出不得提前泄露终局组合）。

## 5.5 网状叙事（Networked Narrative）

解绑「世界中心主义」，采用 **30 / 50 / 20** 黄金叙事比例，将真相拆碎埋在农场各处：

| 比例 | 功能 | 示例 |
|------|------|------------------------|
| **30% 强相关** | 拼凑真相 | 主人半夜开**黄色保温灯**（红顶屋窗侧可见/黑猫控诉）；主人舀水加湿 → 被青蛙误读为「生命之源枯竭」；黑猫碎月 → 青蛙误读「黑恶魔拍碎白圆」等 |
| **50% 弱相关** | 生活气息与人物弧光 | 大黄宿醉自责、淑芬离窝狂奔、老鼠乐子人闲话等 |
| **20% 红鲱鱼** | 高能误导 | 小鸡恶作剧石头假蛋、乌鸦囤积「白球」、老鼠编造水怪等 |

## 5.6 上帝视角真相时间线

> 文中「第一天」=案发前三天（T-3），「第四天」=游戏开局当日（T0）。与 `07` 章节对齐。

### 第一天（T-3）：熊孩子作妖被连环恐吓

**下午（作案）**

- **小鸡侦探团**（半大幼鸡）嫌窝里真蛋无聊，想带「弟弟」出去踢球；找石头画爱心伪装假蛋，将**真蛋**推出窝。
- **淑芬**暂时离窝（觅食/饮水/沙浴等）；回窝一坐发现「蛋」冰冷坚硬——小鸡用窝中散落的羽毛碎屑将石头略作包裹，外形轮廓尚可蒙混，但温度与手感骗不了人。淑芬孵化焦虑已积压数日、心弦绷至极限，这一刻的冰冷刺激直接击穿理性判断，她来不及仔细检视真假，只是本能崩溃：**院内狂奔、大叫、疯狂深蹲**，拼命想用体温把「孩子」救回来。
- **农场主**目击淑芬失控，理解为最后一颗蛋迟迟不破壳的焦虑前兆。

**傍晚（还蛋）**

- 小鸡推沾泥真蛋路遇 **老鼠兄弟**，被恐吓「泥巴球会被池塘水怪吃掉」。
- 小鸡至池塘洗泥，吵醒 **悲伤蛙**；蛙对水中月影发存在主义感慨，小鸡误读为「白圆将沉深渊」＝蛋会被吃。
- 小鸡将洗净真蛋推回窝，**石头假蛋**扔进**鸡舍外草丛**。此时淑芬仍在院中狂奔，窝中无人。

**深夜（伏笔）**

- 淑芬回窝孵到真蛋，感到温度与生机，安心睡去。

### 第二天（T-2）：乌鸦背锅

**清晨**

- **囤积癖乌鸦** 将草丛里反光「白球」（石头假蛋）叼走；**看门狗大黄** 扑咬失败，自责到连续两天偷喝发酵苹果渣宿醉。

**深夜**

- 大黄持续宿醉，院中警戒心下降。

### 第三天（T-1）：人类温柔介入与池塘抓马

**深夜（人类线）**

- 主人至池塘舀水（加湿孵化箱）——悲伤蛙目睹，感叹「生命之源枯竭」。
- 主人回屋抽走 **黑猫** 的天鹅绒软垫铺窝——黑猫感情受骗出走。
- 主人趁淑芬睡着，将**真蛋**温柔抱回红顶屋；开**黄色保温灯**。

**同夜（动物线）**

- 宿醉大黄梦中酒嗝 → 小鸡误当「水怪」。
- 黑猫路过池塘泄愤拍碎水中月影 → 悲伤蛙惊呼「白圆被黑色恶魔拍碎」。

### 第四天（T0，游戏开局）：破壳与全农场大忽悠

**清晨**

- 淑芬发现真蛋消失崩溃大哭；小鸡以为水怪吃蛋，戴墨镜撒谎掩盖。
- 真蛋实于主人保温箱；墙缝老鼠听见 **「咔嚓」碎裂声**——这是蛋壳破裂声（小鸡破壳前兆或破壳瞬间），与木材卡榫声无关。
- **玩家（小火人侦探）** 携笔记本入场。

## 5.7 尾声（漫画收束 + 二周目释怀盘 + meta）

- **主线终点**：小火人侦探**跳入红顶屋天窗**的那一瞬间，画面淡出至手帐特写——**手帐展开为漫画载体**，整段卧室揭晓 + 健康小鸡回巢以**漫画分镜**演绎（小火人 + 黑猫共同进卧室、谜底归位、小鸡破壳、主人**剪影**温柔送回淑芬窝、母子团聚）；**卧室内部不在 3D 场景中实时呈现**，温柔情感落点全部由漫画承担（详见 `07-7.2` 步骤 14–15、`09-9.2` 第二页页面收束段）。
- **二周目释怀盘**：漫画合本后小火人**被放置在红顶屋大门前**进入二周目阶段——农场动物 NPC 进入"结尾状态"，每个可交互 NPC 都会有**一段一次性结尾对话**；老鼠八卦补完仍开放；场景刷新一批奶酪碎以支持全收集（详见 `07-7.2` 步骤 16）。
- **meta 收尾**：农场中央 **闪电蜗牛** 极慢语速收尾——「大——侦——探——，案——子——破——了——吗——？」（与 `06` 蜗牛交互表一致；触达时机为玩家二周目最后一次靠近蜗牛叶尖）。
