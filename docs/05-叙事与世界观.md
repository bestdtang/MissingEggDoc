# 05 叙事与世界观

## 5.1 世界观概要

- **空间**：高度风格化的 **3D Lowpoly 微缩农场**（"盒子盆景"式精致感），动物拟人社会规则松散、充满抓马。
- **案件**：劳模母鸡**淑芬**的最后一颗"爱心蛋"在清晨失踪；全农场在**视角受限**下各自误读，交织成"大忽悠局"。
- **真相内核**：无阴谋——**农场主**因心疼淑芬焦虑，半夜温柔将蛋带回卧室人工孵化；动物与玩家的惊悚推理多为**跨服聊天**叠加。

## 5.2 与"小火人世界" IP 的结合方式

- **玩家身份**：**小火人侦探**；
- **平台结合**：行走与交互贴合**抖音小火人原生系统**；叙事上预留**高能切片点**。
- **边界**：comedy 可夸张，但终局落点需温暖、可共情，避免真正恶心或低俗羞辱"劳动者动物"（淑芬线保护）。

## 5.3 叙事基调与禁忌

- **语言基调**：冷幽默、跨频道聊天、官方吐槽。
- **禁忌**：不将儿童向笑点建立在残酷动物伤害结局上；**惨剧误会**仅限误会层，须有喜剧回收。

## 5.4 主线结构（玩家线 × 真相线）

- **玩家线**：见 `07` **两章主线**流程与尾声。
- **真相线**：见下文 **5.6 上帝视角真相时间线**（策划守密用；对白与演出不得提前泄露终局组合）。

## 5.6 上帝视角真相时间线

> 文中"第一天"=案发前三天（T-3），"第四天"=游戏开局当日（T0）。与 `07` 章节对齐。完整事实链总表见 `10-NPC主时间线.md`。

### 第一天（T-3）：熊孩子作妖被连环恐吓

**下午（作案）**：小鸡侦探团嫌窝里真蛋无聊想带"弟弟"出去踢球；找天然白石头用木炭画爱心和鬼脸伪装假蛋，把**真蛋**推出窝。淑芬暂时离窝，回窝坐到冰冷石头上**没识破**——孵化焦虑积压数日心弦绷至极限，本能崩溃**院内狂奔、深蹲、大叫**。农场主目击淑芬失控，理解为最后一颗蛋焦虑前兆。

**傍晚（还蛋）**：小鸡推沾泥真蛋路遇老鼠被恐吓"水怪"→ 池塘洗泥吵醒悲伤蛙、对水中月影发存在主义感慨被小鸡误读为"玉盘将沉深渊" → 趁主人回屋翻书、淑芬仍在水盆边缓气，小鸡从鸡舍侧口将真蛋推回窝，**石头假蛋扔进鸡舍外草丛**。淑芬回窝以真蛋温度为安。

### 第二天（T-2）：乌鸦背锅

**清晨**：囤积癖乌鸦将草丛里的画爱心和鬼脸白石头叼走；大黄院中追击不及，**认定**那便是淑芬的蛋被衔走；**乌鸦**绕顶盘旋时嚣张叫大黄「**地毯**」；**黑猫**旁观后对大黄进行嘲讽，径自走开。大黄失职崩溃，于是偷喝发酵苹果渣，连日宿醉。

**下午**：**正午前后**乌鸦在谷仓顶摆弄玻璃珠等反光收藏，玻璃珠聚光**险些燎着**午睡黑猫的黑毛；黑猫上谷仓顶找乌鸦理论，**顺带认清**假蛋，但是仍**选择沉默**；老鼠趁其在谷仓顶偷薄荷鱼。黑猫下顶堵墙缝未果记仇。

### 第三天（T-1）：人类温柔介入与池塘抓马

**深夜（人类线）**：主人被大黄酒嗝鼾声惊醒，感受到倒春寒寒意，于是至池塘**舀水**为孵化箱加湿（悲伤蛙误读"生命之源枯竭"）→ 回屋抽走**黑猫天鹅绒软垫**铺保温箱底（黑猫感情诈骗出走）→ 趁淑芬睡着将**真蛋**温柔抱回红顶屋 → 开**黄色保温灯**。

**同夜（动物线）**：宿醉大黄梦中酒嗝 → 小鸡误当"水怪"；黑猫路过池塘泄愤拍碎水中月影 → 悲伤蛙惊呼"玉盘被黑色恶魔拍碎"。

### 第四天（T0，游戏开局）：破壳与全农场大忽悠

**清晨**：淑芬发现真蛋消失崩溃大哭；小鸡以为水怪吃蛋戴墨镜撒谎掩盖。真蛋实于主人保温箱；墙缝老鼠听见"咔嚓"碎裂声——蛋壳破裂声。玩家（小火人侦探）携笔记本入场。

## 5.7 尾声（漫画收束 + 二周目释怀盘 + meta）

- **主线终点**：小火人**跳入红顶屋天窗**瞬间画面淡出至手帐特写——**手帐展开为漫画载体**，整段卧室揭晓 + 健康小鸡回巢以**漫画分镜**演绎（小火人 + 黑猫共同进卧室、谜底归位、小鸡破壳、母子团聚）；**卧室内部不在 3D 中实时呈现**。
- **二周目释怀盘**：漫画合本后小火人**被放置在红顶屋大门前**——农场 NPC 进入"结尾状态"，每个可交互 NPC 都有一段**一次性结尾对话**；老鼠八卦补完仍开放；场景刷新一批奶酪碎支持全收集。
- **meta 收尾**：农场中央**闪电蜗牛**极慢语速收尾——「大——侦——探——，案——子——破——了——吗——？」（二周目最后一次靠近触达）。
