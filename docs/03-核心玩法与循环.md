# 03 核心玩法与循环

## 3.1 核心乐趣一句话

在「全员跨服聊天」的农场里，用**侦探笔记本**把碎片拼回真相：前期荒诞误导拉满，终局以**人类温柔介入**完成情绪反转。

## 3.2 单局 / 单案件结构（本作 = 单主案件长线）

本作以**单主案件**贯穿（淑芬丢蛋），按**两章**推进：

| 阶段 | 玩家情绪目标 | 玩法重心 |
|------|----------------|----------|
| 起因 | 焦急共情 + 轻喜剧 | 接委托、开笔记本、走访 |
| 第一章：天降大锅 | 悬疑寻凶 → 爆笑乌龙 | 池塘误导链、谷仓屋顶对峙乌鸦、假蛋回收 |
| 第二章：温柔的反转 | 跨服聊天收束 | 黑猫特权线、老鼠奶酪、青蛙薄荷鱼、红顶屋物证总览 |
| 尾声 | 吐槽式 meta 释放 | 闪电蜗牛慢速补刀 |

## 3.3 主循环与外循环

- **主循环（分钟–小时级）**：探索 3D 农场 → 点击触发气泡对话 → 发现物证/矛盾 → **侦探笔记本自动记录** →（可选）**物证质询**解锁进阶对话 → 推进空间（上谷仓等）→ 章节真相局部回收。
- **外循环（日回，若做运营版再展开）**：当前文档以**一次性剧情体验**为主；代币收集（如奶酪碎）属于章节内子循环，见 `10`。

## 3.4 心流与节奏

- **节拍**：误导升级（水怪/乌鸦）→ 证据打脸（假蛋石头）→ **章间重置**（真蛋昨夜失踪、指向红顶屋）→ 温柔终局（保温箱与小鸡破壳）。
- **兜底**：谜语人 **悲伤蛙** 提供「情绪惊悚」而非硬逻辑锁；**老鼠情报**可承担防跳关与喜剧调剂（见 `06`/`10`）。

## 3.5 核心机制（玩法落地）

1. **原生行走与交互**：基于抖音原生系统，玩家以简单**点击**触发气泡对话，**拒绝繁琐 UI**（与 `09` 对齐）。
2. **侦探笔记本（核心逻辑）**
   - **自动记录**：发现物证或关键信息后，笔记本生成**精美手绘条目**。
   - **线索回溯**：整理动物口供矛盾，支撑「跨服聊天」喜剧与推理。
3. **物证质询**：向 NPC **出示特定物品**（如假蛋石头等）解锁进阶剧情。
