# 03 核心玩法与循环

## 3.1 核心乐趣

在"全员跨服聊天"的农场里，用**侦探笔记本**把碎片拼回真相：前期荒诞误导拉满，终局以**人类温柔介入**完成情绪反转。

## 3.2 单局结构

**单主案件**贯穿（淑芬丢蛋），按**两章主线 + 漫画收束 + 二周目释怀盘**推进：

| 阶段 | 玩家情绪 | 玩法重心 |
|------|---------|----------|
| 起因 | 焦急共情 + 轻喜剧 | 接委托、开笔记本、走访 |
| 第一章：天降大锅 | 悬疑寻凶 → 爆笑乌龙 | 池塘误导链、谷仓屋顶对峙乌鸦、假蛋回收 |
| 第二章：温柔的反转 | 跨服聊天收束 | 黑猫特权线、老鼠奶酪、青蛙薄荷鱼、红顶屋外勘察、攻顶 |
| **漫画收束（主线终点）** | **温柔释然 + 情感高潮** | 跳入天窗 → 手帐展开为漫画载体 → 黑猫共同发现真相 + 小鸡回巢淑芬团聚 → 合上手帐 |
| **二周目：释怀盘** | **吐槽式 meta 释放 + 全收集满足** | 小火人放回红顶屋门前；全 NPC 一次性结尾对话；老鼠八卦补完；奶酪碎二周目刷新支持全收集 |
| 尾声 | meta 收束 | 闪电蜗牛慢速补刀（二周目期间触达） |

## 3.3 主循环

探索 3D 农场（步行 + **平台跳跃**）→ 点击触发气泡对话 → 发现物证 / 矛盾 → **侦探笔记本对应固定槽位自动书写 / 拼贴** →（可选）**物证质询**解锁进阶对话 → 推进空间（上谷仓等）/ 拾取奶酪碎 → 章节真相局部回收。

代币收集（奶酪碎）属于章节内子循环，详见 `characters/老鼠兄弟.md`。

## 3.4 心流与节奏

- **节拍**：误导升级（水怪/乌鸦）→ 证据打脸（假蛋石头）→ **章间重置**（真蛋昨夜失踪、指向红顶屋）→ 攻顶跳入天窗 → **手帐展开漫画演绎**（黑猫共同发现真相 + 小鸡回巢淑芬团聚）→ 温柔合本 → **二周目释怀盘**（全 NPC 结尾对话 + 八卦补完 + 全收集）。
- **兜底**：谜语人**悲伤蛙**提供"情绪惊悚"而非硬逻辑锁；**老鼠情报**可承担防跳关与喜剧调剂；二周目无硬卡关、无失败惩罚，玩家可随时退出。

## 3.5 核心机制

> **机制根基**：本作是一款由**平台跳跃（跑酷机制）支持的轻度推理游戏**。点击对话是推理的接口，平台跳跃是探索与代币收集的接口；二者共同支撑"3D 农场探案"，**不偏重操作极限**。

1. **原生行走与平台跳跃**：基于抖音原生系统；空间挑战（屋顶、枝杈、栅栏顶等）作为推理进度的**硬空间闸**与**奶酪碎拾取**的载体。

2. **点击交互**：以简单**点击**触发 NPC 气泡对话与物件互动，**拒绝繁琐 UI**。

3. **侦探笔记本（手帐式 · 横版 16:9）**：
   - 形式：主线 2 页跨页 + 独立第三页汇集老鼠兄弟情报；信息以**固定槽位**的**文字 / 插图 / 大头针 + 线 / 胶带**等手帐元素拼贴，由侦探"自动书写"落入相应位置（详见 `09-9.2`）。
   - **主线两页边界**：第一页覆盖"锁定乌鸦之前"（章一误导期）；第二页覆盖"进入红顶屋之前"（章一末揭露 + 章二跑腿与谈判，卧室揭晓时第二页预留槽位顺势归位）。
   - **过程行笔记**：除事实条目外登记当前假设与下一步去向；假设被推翻后**当场划掉、旁补新结论**——划掉即推理过程的可视化。
   - **双重作用**：线索汇总 + 任务引导一体，玩家不必另开任务栏。

4. **物证质询**：向 NPC 出示特定物品（如假蛋石头等）解锁进阶剧情。

5. **奶酪碎拾取与情报购买（推理子循环）**：场景内**散落限量奶酪碎**（一周目一次性、无重生），分布于平台跳跃挑战点（屋檐、枝杈、谷仓上层、池塘荷叶连跳等）；玩家在红顶屋墙缝**老鼠兄弟**处分档兑换情报（**一档 1 块 / 二档 5 块**；二档按剧情阶段拆**前组 / 后组**，后组在薄荷鱼支付后才解锁）。**薄荷鱼最后掉落方向 = 唯一硬必买 = 8 块（全场最贵）**，由【薄荷鱼线】处理；具体取回位置由玩家回扣悲伤蛙处的薄荷味绿垫自行判断。用探索深度而非时间投入控制代币积累速度。详见 `characters/老鼠兄弟.md`。

6. **漫画收束（主线终点 · 一次性演出）**：玩家跳入红顶屋天窗的瞬间画面淡出至手帐特写——**手帐展开为漫画载体**，整段卧室揭晓 + 健康小鸡回巢以**漫画分镜**演绎（小火人侦探 + 黑猫共同发现真相 → 小鸡破壳 → 主人剪影温柔送回淑芬窝 → 母子团聚）；**卧室内部不在 3D 场景中实时呈现**。

7. **二周目释怀盘（主线后 · 收集 / 全 NPC 结尾对话）**：漫画完播后小火人放回**红顶屋大门前**，进入二周目阶段——
   - **全 NPC 一次性结尾对话**：每个可交互 NPC 在二周目首次靠近时触发一段结尾对话（与一周目台词完全分离）；
   - **老鼠八卦补完**：盲盒情报清单仍开放；
   - **奶酪碎二周目刷新**：场景内新刷新一批奶酪碎，总量补足"买光所有剩余八卦 + 留出余裕"；
   - **无硬卡关 / 无失败惩罚**：玩家可随时退出（保留进度）；**闪电蜗牛 Flash** 慢语速 meta 补刀作为最终收束节拍。
