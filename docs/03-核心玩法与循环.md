# 03 核心玩法与循环

## 3.1 核心乐趣一句话

在「全员跨服聊天」的农场里，用**侦探笔记本**把碎片拼回真相：前期荒诞误导拉满，终局以**人类温柔介入**完成情绪反转。

## 3.2 单局结构（单主案件长线）

本作以**单主案件**贯穿（淑芬丢蛋），按**两章**推进：

| 阶段 | 玩家情绪目标 | 玩法重心 |
|------|----------------|----------|
| 起因 | 焦急共情 + 轻喜剧 | 接委托、开笔记本、走访 |
| 第一章：天降大锅 | 悬疑寻凶 → 爆笑乌龙 | 池塘误导链、谷仓屋顶对峙乌鸦、假蛋回收 |
| 第二章：温柔的反转 | 跨服聊天收束 | 黑猫特权线、老鼠奶酪、青蛙薄荷鱼、红顶屋物证总览 |
| 尾声 | 吐槽式 meta 释放 | 闪电蜗牛慢速补刀 |

## 3.3 主循环

探索 3D 农场（步行 + **平台跳跃**） → 点击触发气泡对话 → 发现物证/矛盾 → **侦探笔记本自动记录** →（可选）**物证质询**解锁进阶对话 → 推进空间（上谷仓等）/ 拾取奶酪碎 → 章节真相局部回收。

代币收集（奶酪碎）属于章节内子循环，详见 3.5.5 与 `characters/老鼠兄弟.md`。

## 3.4 心流与节奏

- **节拍**：误导升级（水怪/乌鸦）→ 证据打脸（假蛋石头）→ **章间重置**（真蛋昨夜失踪、指向红顶屋）→ 温柔终局（保温箱与小鸡破壳）。
- **兜底**：谜语人 **悲伤蛙** 提供「情绪惊悚」而非硬逻辑锁；**老鼠情报**可承担防跳关与喜剧调剂（见 `06`/`characters/老鼠兄弟.md`）。

## 3.5 核心机制

> **机制根基**：本作是一款由**平台跳跃（跑酷机制）支持的轻度推理游戏**。点击对话是推理的接口，平台跳跃是探索与代币收集的接口；二者共同支撑"3D 农场探案"的核心体验，**不偏重操作极限**。

1. **原生行走与平台跳跃**：基于抖音原生系统，玩家可在场景内自由行走与**跳跃 / 跑酷**；空间挑战（屋顶、枝杈、栅栏顶等）作为推理进度的**硬空间闸**与**奶酪碎拾取**的载体（详见第 5 条）。
2. **点击交互**：以简单**点击**触发 NPC 气泡对话与物件互动，**拒绝繁琐 UI**（与 `09` 对齐）。
3. **侦探笔记本**
   - **自动记录**：发现物证或关键信息后，笔记本生成**精美手绘条目**。
   - **线索回溯**：整理动物口供矛盾，支撑「跨服聊天」喜剧与推理。
4. **物证质询**：向 NPC **出示特定物品**（如假蛋石头等）解锁进阶剧情。
5. **奶酪碎拾取与情报购买（推理子循环）**：场景内**散落限量奶酪碎**（一次性、无重生），分布于平台跳跃挑战点（屋檐、枝杈、谷仓上层、池塘荷叶连跳等）；玩家在谷仓墙缝**老鼠兄弟**处分档兑换情报（**一档 1 块 / 二档 5 块**；二档按剧情阶段拆为**前组 / 后组**，后组在薄荷鱼支付后才解锁）。**薄荷鱼下落 = 唯一硬必买 = 8 块（全场最贵）**，由独立分支 B 处理。**用探索深度而非时间投入控制代币积累速度**，无需额外防刷设计。详见 `characters/老鼠兄弟.md`。
