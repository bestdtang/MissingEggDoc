# 12 音频方向

## 12.1 总体策略

- **无配音**；对话以文字 + 2D 立绘承担情绪（见 `11` §11.4.3），与 `01` 非目标「全语音」一致。
- 以**角色专属音效音色** + **环境音**塑造沉浸；3D 场景以静态 pose 为主，**不做口型同步**。
- 音频与叙事节奏需支持**喜剧 / 惊悚切换**——同一农场在误导期与揭晓期听感应有温差。

## 12.2 角色与事件 cue（首版方向）

| 对象 | 方向 |
| --- | --- |
| 大黄 | 宿醉**酒嗝**、沉闷带哨音**鼾声**（「呼噜噜——咕呱」）；叫醒后恢复常规狗叫 / 哼声 |
| 悲伤蛙 | **长叹气**、池水空寂底噪；对暗号失败时沉默，无夸张 rejection 音效 |
| 闪电蜗牛 Flash | **极慢语速**文本节奏 + 轻「黏」感移动音；喜剧来自慢，不靠变调 |
| 黑猫 | 第一章**树上独白**为不可交互音效（声音来源不明）；落地、开窗可有短促机械感 |
| 乌鸦 | 远距离**嘎嘎**、屋顶对峙时护食式短叫；嚣张但不持续吵耳 |
| 淑芬 | 焦急时轻**咯咯**、克制；不配人类化哭泣 |
| 小鸡侦探团 | 压声**叽叽**、偷听段更低音量；群像对话按说话者略分 pitch |
| 老鼠兄弟 | 墙缝**窸窣**、算盘轻响、奶酪交接短音 |
| 环境 | 农场昼夜底噪、池塘、谷仓空阔感、红顶屋墙缝「听感」隔层；漫画收束段收束为更暖、更静 |

## 12.3 UI 与系统音

- 笔记本：**书写 / 钉入 / 贴入 / 翻页**轻音效，与手帐拼贴风匹配，不抢戏。
- 线索获得、奶酪计数、对话叹号提示：短促、统一手绘感音色家族。
- 锁条件 / 失败反馈：轻量 negative cue，不惩罚感过重。
