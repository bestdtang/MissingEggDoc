# 09 UI/UX 与抖音侧能力

## 9.1 核心界面流程

- **主路径**：进入农场 → 对话气泡驱动 → 笔记本条目渐进解锁 →（质询）出示物证 → 空间推进 → 章节收束演出。
- **弱 UI 原则**：拒绝复杂菜单堆砌；高频操作集中在 **点击** 与少量「出示/记录」入口。

## 9.2 笔记本结构（手帐式 · 横版 16:9 · 主线 2 页 + 老鼠情报 1 页）

笔记本是整局唯一的信息中枢，**线索汇总 + 任务引导双重作用一体**——不另开任务栏。整体以**手帐**形式呈现：横版 16:9 跨页、信息出现在**固定位置**，由**文字 + 插图 + 大头针 + 线 + 胶带**等元素拼贴而成。玩家不手填，而是看着笔记本在小火人侦探镜头里"自己"被画上 / 钉上 / 贴上——每条新线索激活对应槽位的预设视觉演出（钢笔写入、便签贴上、红线连接等）。

除事实条目外，笔记本以**过程行笔记**登记当前假设与下一步去向（如「乌鸦偷了蛋？去谷仓顶看看」）；假设被推翻后**当场划掉、旁补新结论**——划掉本身就是侦探推理过程的可视化证据，也是手帐质感的来源。

### 主线 · 第一页（"锁定乌鸦之前"）

- **覆盖时段**：游戏开场（淑芬委托）→ 玩家在多源证词（小鸡侦探团 / 悲伤蛙 / 大黄醉证 / 树上黑猫剪影 / 老鼠一档情报）汇聚下**锁定乌鸦为嫌疑、决定攀谷仓屋顶**之前的全部信息。
- **固定槽位（示例 · 美术待定稿）**：淑芬委托便签（左上 · 爱心蛋插图 + 失踪时间）/ 池塘谜语剪贴（左下 · 悲伤蛙"白圆将沉深渊"）/ 小鸡侦探团口径与水怪传闻（中上）/ 大黄醉证（中下 · "黑色恶魔掠走白色宝贝"）/ 谷仓屋顶剪影 + 乌鸦插画（右上）/ 大橡树暗影 + 黑猫疑问号（右下）。
- **过程行笔记（示例）**：
  - 「池塘有水怪？」→ 走访悲伤蛙后**划掉**，旁注「只是谜语人」。
  - 「乌鸦偷了蛋？去谷仓顶看看」→ 作为**当前任务指针**保留，未划掉。
- **页面收束**：玩家做出"攀谷仓屋顶"决定、启动平台跳跃挑战的那一瞬，笔记本**翻至第二页**（与 `07-7.2` 第一章步骤 4 对齐）；第一页保持已写就状态留作回溯，不再追加。

### 主线 · 第二页（"进入红顶屋之前"）

- **覆盖时段**：谷仓屋顶起 → 玩家从天窗下降进入红顶屋卧室**之前**的全部信息，贯穿第一章末假蛋揭露、第二章全部跑腿与谈判节点。
- **固定槽位（示例 · 美术待定稿）**：石头假蛋拓片（鬼脸 + 爱心 · 左上 · 第一章末贴入）/ 大黄清醒证词便签（雨靴两趟 · 黄灯 · 红顶屋指向 · 中上）/ 谷物泡水 + 嗅觉恢复时间戳（左中）/ **红顶屋外勘察平面图（正中 · 第二章核心拼贴）**：紧闭大门 · 暖黄窗光 · 精美猫门 · 屋顶天窗轮廓 · 大黄蹲守位 / 大橡树 + 黑猫薄荷鱼委托（右上）/ 池塘 → 悲伤蛙屁股下 → 沾蛙味薄荷鱼（右下 · 跑腿子链）/ 老鼠墙缝硬必买"薄荷鱼下落"路径条（右中）。
- **跨页回响（示例）**：第一页"乌鸦偷了蛋？"这条任务指针，在第二页起手处被**划掉**，旁注「画鬼脸的石头！是小鸡的恶作剧」——同一条假设跨页回响，凸显推理过程。
- **过程行笔记（示例）**：
  - 「真蛋昨夜从鸡舍消失 → 目标转向红顶屋」→ 第二章主锚点。
  - 「红顶屋大门撞不开 → 找猫门钥匙？」→ 大黄摇树 → 黑猫现身后**划掉**，更新为「黑猫钻猫门，从内推开屋顶天窗 → 我从屋顶下去」。
- **页面收束 · 手帐展开为漫画载体**：**主线在小火人侦探跳入天窗的那一瞬间收束**——画面淡出至手帐特写，第二页**展开为漫画载体**，整段卧室揭晓 + 健康小鸡回巢以**漫画分镜**演绎（玩家以观众视角看完，**卧室内部不在 3D 场景中实时呈现**）。第二页此前预留的"红顶屋内部 · 桌上装置 · 软垫去向 · 蛋的位置"等槽位**随漫画镜头节奏陆续被分镜插画填入**——**漫画本身即第二页揭晓段的"分格画稿"**（核心 5 件 + 二级 3 件 + 温柔回巢落点，详见 `07-7.2` 步骤 15）。漫画完播后第二页全部写就，"合上手帐"即案件了结的仪式收束（与 `07-7.2` 步骤 15–16 与尾声、`characters/闪电蜗牛-Flash.md` 阶段 4 呼应）。

### 辅页 · 老鼠兄弟情报（独立第三页 · 横版 16:9）

- **独立成页**，**不混入主线两页**——避免老鼠一档八卦（如"大黄项圈镀银"、"蜗牛是商业间谍"）稀释主信噪比；玩家自行翻阅这一页对老鼠情报做真假判断。
- **形式**：仍用手帐拼贴，但视觉上做出区隔（如纸张更脏旧、便签边缘老鼠咬痕、字迹采用"老鼠口述、侦探转录"风格），与主线两页的整洁度形成对照。
- **排列**：按购入顺序自上而下排列；每条标档次（一档 A / 一档 B 浅层做证 / 二档前组 / 二档后组），**不显式标"真 / 假"**——把"咦居然是真的"的小惊喜与"果然又被坑了"的笑点留给玩家亲历验证（详见 `characters/老鼠兄弟.md` "老鼠兄弟情报" 清单与设计意图）。
- **跨页穿线**：经玩家亲历验证为真的老鼠情报，可由侦探"撕下贴到主线两页对应槽位"的视觉演出**复用到主线**（如 B 组 #10「黑猫 T-2 夜上屋顶」撕贴到第二页大橡树槽位），让"老鼠情报 → 主线推理钩子"在视觉上成立。

### 回溯与翻页

- 任意时刻玩家可**横向翻页**回看任意一页；当前主线进度自动停留在最新主页，老鼠页随购入即时更新。
- 回溯支持快速跳转到任意已写入条目，避免玩家手写笔记负担。
- **弱网**：对话与存档请求需可重试。

### 手帐在二周目（释怀盘）的状态

- **主线两页封存**：第一页"锁定乌鸦之前" + 第二页"漫画演绎完成态"作为已写就页面留存，**可回溯不可追加**。
- **老鼠情报第三页仍开放**：玩家可继续在谷仓墙缝处付费抽取未购入的剩余八卦（含一档 A/B 未抽 + 二档前组 / 后组未抽 · 含 bullshit / 真情报混抽），新购条目按购入顺序追加；详见 `characters/老鼠兄弟.md` "奶酪碎经济与拾取设计 · 二周目刷新"段。
- **NPC 结尾对话不进笔记本**：每个可交互 NPC 在二周目首次靠近时触发的一次性结尾对话**作为场景演出节拍**呈现，不登记到手帐——避免笔记本在主线已破后继续累加内容、稀释"合上手帐"的仪式感（占位见 `characters/` 各详档"二周目结尾对话"段）。
- **任务指针软化**：二周目无硬卡关、无明确"下一步"指针；手帐封面或第三页页眉给出**软性进度提示**（如"剩余八卦 N 条 / 剩余 NPC 结尾对话 N 段 / 剩余奶酪碎 N 块"），不强迫全收集。

## 9.3 抖音能力与二创点

- **名场面切片**：
  - **闪电蜗牛** 慢语速跨阶段补刀；
  - **悲伤蛙** 史诗级谜语误导；
  - **大黄** 醉狗痛哭与假蛋后「保安归位」；
  - **黑猫** 薄荷鱼上头开窗等。
- **能力对接**：分享、挑战任务、订阅等以平台实际可接能力为准。

## 9.4 无障碍与误触

- 点击热区宜大于视觉边界；关键质询前可有一次**确认**（防误触）。
