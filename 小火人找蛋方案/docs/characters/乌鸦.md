# 囤积癖乌鸦

## 人设

- **视觉**：贼眉鼠眼黑鸦；谷仓顶巨巢堆「财宝」（瓶盖碎玻璃糖纸）；身下压 **画爱心白蛋（实为石）**。
- **性格**：贪婪、废品收藏家、领地意识强、嚣张。
- **关系**：与张阿菊互骂；鄙视大黄跳跃。
- **故事线**：T-2 清晨俯冲叼草丛假蛋，大黄拦失败；翻面见小鸡鬼脸涂鸦惊为「部落图腾宝石」。T0 真蛋失踪当天仍炫耀收藏 → 被黛西恶意攀咬 + 大黄断片证词捧为 **第一章 Boss**。

## 交互

- 屋顶对峙；获得【画鬼脸沉重石头】；乌鸦：「别想抹去神秘图腾！」
