# 雌竞鸡 · 张阿菊（自称黛西）

## 人设

- **视觉**：羽毛顺滑闪亮；碎玻璃当镜子；美容道具围绕。
- **性格**：极度自恋、毒舌、八卦枢纽；痛恨乡土本名；与乌鸦死对头（抢闪亮垃圾）。
- **动机**：全县农场小姐选美；别人事不关己，碰羽毛记仇一辈子。
- **关键剧情**：T-3 下午冷眼旁观小鸡换蛋；敷衍淑芬「五百深蹲 + 拔羊毛」。T-1 夜拖拉机旁白石灰 SPA，被闭眼绵棉踩尾惨叫；遗留 **白粉坑、挣扎痕、羊蹄印**（统一为绵羊，见 `13`）。

## 交互

1. **借刀杀鸟（指乌鸦）**  
   - 玩家问爱心蛋。黛西（照镜）：「查查屋顶黑心贼鸟，昨天傍晚叼走反光白圆球回窝！」

2. **白粉现场**  
   - 玩家见白粉 + 羊蹄印 → 找绵棉。

3. **破解惊悚**  
   - 玩家持绵羊证词 + 【沾泥白色母鸡羽毛】质问。  
   - 黛西（大白眼）：「什么踩碎蛋？你让那头**蠢羊**睁开眼睛！被踩到打滚的白球是本小姐！极品白泥 SPA！白粉是泥！羊踩的是我尾羽！」
