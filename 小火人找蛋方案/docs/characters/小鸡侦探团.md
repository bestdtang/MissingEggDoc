# 小鸡侦探团

## 人设

- **视觉**：3–4 只半大幼鸡；落叶「侦探帽」、废纸壳墨镜；喜欢首尾相连排队走。
- **性格**：中二、菜又爱玩、自信撒谎精，内心慌。
- **动机**：以为害死未出世弟弟；想用假情报忽悠玩家打怪物掩盖「前科」并救弟。
- **故事线摘要**（与 `05` 对齐）：T-3 下午偷龙转凤 → 傍晚被老鼠吓、被青蛙误读 → 洗净真蛋放回、假蛋扔草丛 → T0 清晨以为水怪成真，组「小鸡侦探团」带偏玩家。

## 交互

1. **阶段一：目击证人（误导池塘）**  
   - 时机/位置：开局鸡舍附近鬼祟开会。  
   - 领头鸡（推墨镜）：「同行你好。暗影侦探团已掌握决定性线索！」  
   - 跟班 A：「我发誓，三个脑袋浑身绿色的水怪把蛋偷去池塘！」

2. **阶段二：对质（假蛋后）**  
   - 时机：玩家持【假蛋】回鸡舍。  
   - 大哭招供前天换石恶作剧，发誓昨夜前已放回真蛋；甩锅乌鸦捡走石头。
