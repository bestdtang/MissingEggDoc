# 看门狗 · 大黄

## 人设

- **视觉**：大体型金毛或田园黄狗；保安金属牌；开局头顶湿树叶宿醉，苹果渣散落。
- **性格**：忠诚、责任心爆棚、不会撒谎、伤心借酒、酒品差。
- **故事线**：T-2 见乌鸦掠走「白球」自责 → 连醉两天 → 防空洞开给信鸽；T-1 深夜酒嗝被小鸡误读水怪。

## 交互

- 玩家听黛西后找大黄核实：酒嗝哭诉时间错乱，但咬定黑恶魔掠走白色宝贝 → 强化锁定乌鸦。  
- 出示假蛋石头后清醒：「原来是石头？我没丢淑芬的孩子！」→ 提供鸡舍外灰色影子/脚印羽毛调查方向。
