# 11 美术与音频方向

## 11.1 视觉策略（与 `08` 对齐）

采用 **「3D 环境 + 2D 立绘」** 表现策略，平衡开发效率与叙事表现力。

- **3D Lowpoly**：构建高度集约、色彩明快的微缩农场，强调「盒子盆景」精致感。
- **2D 动态立绘**：表现情绪与幽默亲和力（夸张反应、吐槽脸）。

## 11.2 角色与场景风格统一性

- **动物**：毛茸茸、拟人神态；小鸡纸板墨镜、绵羊破布芭蕾裙、信鸽微型工牌等**可识别剪影**。
- **与小火人 IP**：玩家小火人侦探造型需与平台原生规范一致（具体约束待美术规范文档 TBD）。

## 11.3 UI 风格

- 轻量、手绘感笔记本 UI 与 Lowpoly 场景形成材质对比；避免厚重拟物边框抢戏。

## 11.4 音频与叙事配合

- **无配音**；以**角色专属音效音色** + **环境音** 塑造沉浸（与 `01` 非目标「全语音」一致）。
- **情绪 cue**：宿醉酒嗝、蜗牛极慢语速、绵羊崩溃大哭等，音频与节奏需支持**喜剧/惊悚切换**。
