# 07 关卡与内容结构

## 7.1 内容层级命名

- **世界**：微缩农场（单主场景 + 关键子场景：鸡舍区、池塘、谷仓屋顶、拖拉机旁、红顶屋、大橡树、谷仓墙缝等）。
- **结构**：**起因** + **三章主线** + **尾声**；章节以「推理场重置」为界（假蛋回收 → 鸡舍内线 → 红顶屋终局）。

## 7.2 玩家游玩主线（与 `05` 时间线对照）

### 起因：焦急的母亲

混乱清晨开场：玩家操控小火人侦探，接 **淑芬** 委托——寻找画爱心且失踪的宝贝蛋；获得 **侦探笔记本**，踏入农场。

### 第一章：天降大锅

- **情绪**：悬疑寻凶 → 爆笑大乌龙。
- **流程摘要**：
  1. 被 **小鸡侦探团**（纸板墨镜）忽悠至池塘，遭遇 **悲伤蛙** 谜语。
  2. 走访 **张阿菊**：攀咬 **乌鸦** 叼走「反光画纹白球」。
  3. 找宿醉 **大黄** 核实，获「老实人强力证词」（时间线模糊）。
  4. **平台跳跃/场景互动**（推箱、干草垛等）登上**谷仓屋顶**对峙乌鸦。
  5. 发现乌鸦守护的是 **画鬼脸的重石假蛋**；下楼与小鸡对质 → 招供「偷龙转凤」但坚称昨夜前已放回真蛋。
- **章末认知**：水怪与乌鸦线为乌龙；**真蛋在昨夜从鸡舍「凭空」消失**。

### 第二章：粉身碎骨的命案

- **情绪**：惊悚绝望 → 极致抓马喜剧。
- **流程摘要**：
  1. **灰色外来羽毛** + 窗外脚印 → 怀疑 **李翠花（翠花）** 勾结外来飞贼。
  2. 绩效榜 + 黛西挑拨建立动机；搜窝得 **灰色羽毛**；质询翠花 → 见 **信鸽**；偷听排除「偷蛋」。
  3. **屋檐二层平台**：需利用箱子**跳**上，旁听「黄色暖灯 = 人工美黑舱」核心碎片。
  4. **拖拉机旁现场**：白粉坑、挣扎痕、中心 **羊蹄印**（与物种一致，见 `13` 勘误）。
  5. 质问 **绵棉** →「自首」踩碎白粉球体；玩家沉重重返现场，于白粉中找到 **沾泥白色母鸡羽毛** 质问张阿菊 → 白粉 SPA 真相回收。
  6. 绵羊报恩线索：**人类雨靴声往红顶屋**；指向 **黑猫** 特权线。

### 第三章：温柔的反转

- **情绪**：跨服聊天完美收束。
- **流程摘要**：
  1. 红顶屋大门紧闭 → 八卦得知仅 **黑猫** 可进人类屋。
  2. 大橡树下求黑猫 → 任务：**找回被老鼠偷走的薄荷玩具**。
  3. **跑腿链**：谷仓墙缝 **老鼠** 要价「三块奶酪碎」→ 情报指向池塘；**悲伤蛙** 屁股下抽出 **沾蛙味的薄荷鱼**；交黑猫 → 猫醉开**窗**。
  4. 卧室内物证总览：保温箱 + **黑猫天鹅绒垫** + **淑芬拔的羊毛** + **黄灯** + **池塘水**；中央 **健康小鸡刚破壳**。
- **结案语义**：主人心疼淑芬「发癫」式焦虑，半夜温柔介入人工孵化；无阴谋。

### 尾声

玩家合笔记本出屋；路遇 **闪电蜗牛** 慢速补刀一句（见 `06`）。

## 7.3 难度与内容量规划

- **目标总时长**：**1.5–2.5 h**（与 `01`/`02` 一致）。
- **难度**：休闲向；硬卡关依赖 **物证链/空间位置**（屋檐、屋顶）而非操作极限。
- **内容量占位**：主案件 1；动物 NPC 群像 ≥10（以 `06` 为准）。
