## 6.4 角色细化与交互脚本

### 小鸡侦探团

- **视觉**：3–4 只半大幼鸡；落叶「侦探帽」、废纸壳墨镜；喜欢首尾相连排队走。
- **性格**：中二、菜又爱玩、自信撒谎精，内心慌。
- **动机**：以为害死未出世弟弟；想用假情报忽悠玩家打怪物掩盖「前科」并救弟。
- **故事线摘要**（与 `05` 对齐）：T-3 下午偷龙转凤 → 傍晚被老鼠吓、被青蛙误读 → 洗净真蛋放回、假蛋扔草丛 → T0 清晨以为水怪成真，组「小鸡侦探团」带偏玩家。

**交互**

1. **阶段一：目击证人（误导池塘）**  
   - 时机/位置：开局鸡舍附近鬼祟开会。  
   - 领头鸡（推墨镜）：「同行你好。暗影侦探团已掌握决定性线索！」  
   - 跟班 A：「我发誓，三个脑袋浑身绿色的水怪把蛋偷去池塘！」

2. **阶段二：对质（假蛋后）**  
   - 时机：玩家持【假蛋】回鸡舍。  
   - 大哭招供前天换石恶作剧，发誓昨夜前已放回真蛋；甩锅乌鸦捡走石头。

---

### 劳模母鸡 · 淑芬

- **视觉**：体态丰满蓬松，母性眼神。
- **性格**：温柔勤恳，为孩子操碎心。
- **动机**：最后一颗蛋迟迟未破壳，画爱心标记视若珍宝。
- **故事线摘要**：T-3 假蛋冰冷 → 狂奔求助（黛西深蹲梗、拔绵羊毛）→ 真蛋被小鸡换回后感到温度安心 → T-1 深夜主人取走真蛋 → T0 清晨发现丢失崩溃。

---

### 雌竞鸡 · 张阿菊（自称黛西）

- **视觉**：羽毛顺滑闪亮；碎玻璃当镜子；美容道具围绕。
- **性格**：极度自恋、毒舌、八卦枢纽；痛恨乡土本名；与乌鸦死对头（抢闪亮垃圾）。
- **动机**：全县农场小姐选美；别人事不关己，碰羽毛记仇一辈子。
- **关键剧情**：T-3 下午冷眼旁观小鸡换蛋；敷衍淑芬「五百深蹲 + 拔羊毛」。T-1 夜拖拉机旁白石灰 SPA，被闭眼绵棉踩尾惨叫；遗留 **白粉坑、挣扎痕、羊蹄印**（统一为绵羊，见 `13`）。

**交互**

1. **借刀杀鸟（指乌鸦）**  
   - 玩家问爱心蛋。黛西（照镜）：「查查屋顶黑心贼鸟，昨天傍晚叼走反光白圆球回窝！」

2. **白粉现场**  
   - 玩家见白粉 + 羊蹄印 → 找绵棉。

3. **破解惊悚**  
   - 玩家持绵羊证词 + 【沾泥白色母鸡羽毛】质问。  
   - 黛西（大白眼）：「什么踩碎蛋？你让那头**蠢羊**睁开眼睛！被踩到打滚的白球是本小姐！极品白泥 SPA！白粉是泥！羊踩的是我尾羽！」

---

### 约会鸡 · 李翠花（翠花） × 装杯信鸽

- **视觉**：翠花别野花、学鸽咕咕；信鸽微型工牌/脚环、鼻孔看人站姿。
- **性格**：翠花恋爱脑、向往城市；信鸽装杯、贬损乡下。
- **动机**：翠花隐瞒网恋怕主人赶走信鸽；信鸽享受崇拜与「学识炫耀」。
- **故事线**：一周前邂逅；翠花睡眠不足致 **母鸡绩效榜** 产蛋暴跌；近两夜狗醉 → 窗外幽会留下 **外来脚印** 与窝中 **灰色羽毛**（定情物藏干草下）。T-1 半夜见卧室黄灯，信鸽解释为「日光美黑舱」。

**交互**

- 绩效榜 + 问黛西 → 黛西挑拨嫉妒动机。  
- 搜窝得【灰色外来羽毛】→ 质询翠花破防 → 领玩家见信鸽洗清「偷蛋」。  
- **屋檐二层**：鸡舍屋檐可跳；底下看似发呆，跳上旁听：  
  - 翠花：「昨晚红顶屋刺眼黄灯在干嘛？」  
  - 信鸽：「日光美黑舱，城里人高强度黄光烤自己，主人半夜局部人工美黑。」

### 囤积癖乌鸦

- **视觉**：贼眉鼠眼黑鸦；谷仓顶巨巢堆「财宝」（瓶盖碎玻璃糖纸）；身下压 **画爱心白蛋（实为石）**。
- **性格**：贪婪、废品收藏家、领地意识强、嚣张。
- **关系**：与张阿菊互骂；鄙视大黄跳跃。
- **故事线**：T-2 清晨俯冲叼草丛假蛋，大黄拦失败；翻面见小鸡鬼脸涂鸦惊为「部落图腾宝石」。T0 真蛋失踪当天仍炫耀收藏 → 被黛西恶意攀咬 + 大黄断片证词捧为 **第一章 Boss**。

**交互**

- 屋顶对峙；获得【画鬼脸沉重石头】；乌鸦：「别想抹去神秘图腾！」

### 看门狗 · 大黄

- **视觉**：大体型金毛或田园黄狗；保安金属牌；开局头顶湿树叶宿醉，苹果渣散落。
- **性格**：忠诚、责任心爆棚、不会撒谎、伤心借酒、酒品差。
- **故事线**：T-2 见乌鸦掠走「白球」自责 → 连醉两天 → 防空洞开给信鸽；T-1 深夜酒嗝被小鸡误读水怪。

**交互**

- 玩家听黛西后找大黄核实：酒嗝哭诉时间错乱，但咬定黑恶魔掠走白色宝贝 → 强化锁定乌鸦。  
- 出示假蛋石头后清醒：「原来是石头？我没丢淑芬的孩子！」→ 提供鸡舍外灰色影子/脚印羽毛调查方向。

---

### 芭蕾绵羊 · 绵棉

- **视觉**：毛茸茸绵羊；屁股 **斑秃**（淑芬拔毛）；破布条当芭蕾裙；常闭眼笨拙踮脚转圈。
- **性格**：沉浸艺术、笨拙、内疚爆表。
- **故事线**：T-3 被拔毛；T-3 傍晚踢假蛋到院子中央；T-1 夜踩白粉 SPA 的张阿菊；对玩家「自首」踩碎球体 → 真相后指路 **雨靴声去红顶屋** 并提示求 **黑猫**。

**交互（节选）**

- 开局询问丢蛋：抱怨被拔毛 → 意识到「白色圆球」联想 → 心虚赶走玩家。  
- 白粉现场羊蹄印 → 对质崩溃大哭：「闭眼大回旋踩中白粉惨叫球体，把蛋踩成粉末！」  
- 乌龙澄清后：感恩 + 低声提供人类雨靴与黑猫线索。

---

### 失眠暴躁黑猫

- **视觉**：乌黑优雅体态；严重黑眼圈；开局躲 **大橡树** 树冠生闷气。
- **性格**：傲慢、领地强、洁癖起床气；与老鼠敌对；嘲大黄跳不高。
- **故事线**：T-2 清晨冷眼乌鸦事件；T-3 晚拍碎水中月加重青蛙 emo；老鼠偷 **薄荷鱼玩具**；T-1 被主人温柔抱起以为按摩 → 被放地板 + **抽走天鹅绒垫** 给蛋与纸箱保温箱 + 黄灯 → 感情受骗出走树上。

**交互**

- 绵羊线前第一章 **不可交互**（仅口碑铺垫）。  
- 绵羊线后大橡树下：控诉感情欺骗；要玩家从老鼠处夺回玩具。  
- 薄荷鱼交还：吸high，钻猫门开窗。  
- 卧室物证：保温箱底 **天鹅绒垫** + 爱心 **破碎蛋壳** 呼应控诉。

### 黑市老鼠兄弟

- **视觉**：胖灰鼠×2；谷仓墙缝；橡子帽、微算盘。
- **性格**：市侩、唯利是图、胆大包天、乐子人。
- **故事线**：编造水怪吓小鸡；偷猫玩具逃跑掉 **池塘边** 被青蛙当浮木；墙缝 T0 听见卧室 **咔嚓** 碎裂声。

**交互**

- 要回玩具：「玩具丢了，想知道下落？**三块奶酪碎**，少一块不行。」  
- 盲盒情报：前期 bullshit；上交增多后吐露「水怪是编的」「墙缝听到咔嚓碎裂声」等。

### 悲伤蛙

- **视觉**：微胖青蛙坐「绿垫子」（实为薄荷鱼）；空洞眼神。
- **性格**：重度抑郁、存在主义、谜语人；不关心丢蛋案本身。
- **故事线**：T-3 傍晚误吓小鸡；T-2 夜捡薄荷鱼垫屁股；T-1 见主人舀水 →「生命之源枯竭」；见黑猫碎月 →「白圆被黑色恶魔拍碎」。

**交互**

- 开局池塘问圆圆白球：长篇误读，加深对黑色动物怀疑。  
- 终局跑腿：缓慢挪开屁股交出【沾蛙味极品猫薄荷鱼】。

- **视觉**：极小；农场中央宽叶上；全程移动极慢。
